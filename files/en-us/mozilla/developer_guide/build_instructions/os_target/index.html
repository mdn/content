OS_TARGET is typically the output of the 'uname -s' command on the target platform. The following list is not complete as any platform could have its own OS_TARGET.

  • WINNT
  • Linux
  • Darwin
  • android
  • SunOS
  • FreeBSD
  • OpenBSD
  • NetBSD
  • OS2
  • BeOS
  • IRIX64
  • AIX
  • HP-UX
  • DragonFly
  • skyos
  • riscos
  • NTO
  • OSF1
