Alias for {{WebExtAPIRef("runtime.getBackgroundPage()")}}.

Note: This method cannot be used in Private Browsing mode â€” it always returns an empty array. Consider using {{WebExtAPIRef("runtime.sendMessage","runtime.sendMessage()")}} or {{WebExtAPIRef("runtime.connect","runtime.connect()")}}. For more info see {{bug(1329304)}}.

Syntax

var page = browser.extension.getBackgroundPage()

Parameters

None.

Return value

object. Window of the background page.

Examples

Suppose a background script defines a function foo():

// background.js

function foo() {
  console.log("I'm defined in background.js");
}

A script running in a popup can call this function directly like this:

// popup.js

var page = browser.extension.getBackgroundPage();
page.foo(); // -> "I'm defined in background.js"
