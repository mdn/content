Gets the tab that is selected in the specified window.

This is an asynchronous function that returns a Promise.

Syntax

var gettingSelected = browser.tabs.getSelected(
  windowId           // optional integer
)

Parameters

windowId{{optional_inline}}
integer. Defaults to the current window.

Return value

A Promise that will be fulfilled with a tabs.Tab object containing information about the selected tab. If the tab could not be found or some other error occurs, the promise will be rejected with an error message.
