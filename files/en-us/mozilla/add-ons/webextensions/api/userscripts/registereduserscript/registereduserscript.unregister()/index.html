The unregister() method of {{WebExtAPIRef("userScripts.RegisteredUserScript","RegisteredUserScript")}} interface unregisters the User Script represented by that interface instance and previously registered via {{WebExtAPIRef("userScripts.register","userScripts.register()")}}.

Note: User Scripts are automatically unregistered when the related extension page (from which the user scripts were registered) is unloaded, so you should register a user script from an extension page that persists at least as long as you want the user scripts to stay registered.

Syntax

const registeredUserScript = await browser.userScripts.register(
  userScriptOptions       // object
);
â€¦
await registeredUserScript.unregister()

Parameters

None.

Return value

A {{JSxRef("Promise")}} that will be resolved once the User Script is unregistered. The promise does not return any value.
