{{AddonSidebar()}}

Remove any highlighting of a previous search that was applied by a previous call to {{WebExtAPIRef("find.highlightResults()")}}, or by the browser's native UI.

Syntax

browser.find.removeHighlighting()

Parameters

None.

Return value

None.
