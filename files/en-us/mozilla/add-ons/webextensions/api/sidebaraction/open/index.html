{{AddonSidebar()}}
Open the sidebar in the active window.

You can only call this function from inside the handler for a user action.

This is an asynchronous function that returns a Promise.

Syntax

browser.sidebarAction.open()

Parameters

None.

Return value

A Promise that is resolved with no arguments.

Browser compatibility

{{Compat}}

Examples

Open the sidebar when the user selects a context menu item:

browser.menus.create({
  id: "open-sidebar",
  title: "open sidebar",
  contexts: ["all"]
});

browser.menus.onClicked.addListener(() => {
  browser.sidebarAction.open();
});
