Removes all items from the storage area.

This is an asynchronous function that returns a Promise.

Syntax

var clearing = browser.storage.<storageType>.clear()

<storageType> will be one of the writable storage types â€” {{WebExtAPIRef("storage.sync")}} or {{WebExtAPIRef("storage.local")}}.

Parameters

None.

Return value

A Promise that will be fulfilled with no arguments if the operation succeeded. If the operation failed, the promise will be rejected with an error message.

Browser compatibility

{{Compat}}

Examples

function onCleared() {
  console.log("OK");
}

function onError(e) {
  console.log(e);
}

var clearStorage = browser.storage.local.clear();
clearStorage.then(onCleared, onError);
