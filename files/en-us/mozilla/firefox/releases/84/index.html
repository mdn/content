This article provides information about the changes in Firefox 84 that will affect developers. Firefox 84 is the current Nightly version of Firefox, and will ship on December 15, 2020.

Changes for web developers

Developer Tools

HTML

Removals

CSS

  • We've added support for complex selectors to the {{cssxref(":not()")}} pseudo-class ({{bug(933562)}}). 

Removals

  • We've removed the proprietary -moz-default-appearance property values scrollbar-small (scrollbar-width: thin is used instead) and scrollbar (macOS only; scrollbar-horizontal and scrollbar-vertical are used instead) ({{bug(1673132)}}).

JavaScript

No changes

Removals

HTTP

Removals

Security

  • Firefox now ensures that localhost URLs, such as http://localhost/ and http://dev.localhost/, refer to the local host's loopback interface (e.g. http://127.0.0.1). As a result, resources loaded from localhost are now assumed to have been delivered securely (see Secure contexts), and also will not be treated as mixed content. ({{bug(1220810)}}, {{bug(1488740)}})

APIs

No changes

DOM

Media, WebRTC, and Web Audio

Removals

WebAssembly

No changes

Removals

Changes for add-on developers

  • The browsingData.remove API now supports removing a subset of data types by cookieStoreId.
