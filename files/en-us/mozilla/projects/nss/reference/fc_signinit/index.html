Name

FC_SignInit - initialize a signing operation.

Syntax 

CK_RV FC_SignInit(
  CK_SESSION_HANDLE hSession,
  CK_MECHANISM_PTR pMechanism,
  CK_OBJECT_HANDLE hKey
);

Parameters

hSession
{{ mediawiki.external('in') }} session handle.
pMechanism
{{ mediawiki.external('in') }} mechanism to be used for the subsequent signing operation.
hKey
{{ mediawiki.external('in') }} handle of the key to be used .

Description

FC_SignInit initializes a signature operation.

A user must log into the token (to assume the NSS User role) before calling FC_SignInit.

Return value
