Name

FC_GetSlotList - Obtain a list of slots in the system.

Syntax 

CK_RV FC_GetSlotList(
  CK_BBOOL tokenPresent,
  CK_SLOT_ID_PTR pSlotList,
  CK_ULONG_PTR pulCount
);

Parameters

tokenPresent
{{ mediawiki.external('in') }} If true only slots with a token present are included in the list, otherwise all slots are included.
pSlotList
{{ mediawiki.external('out') }} Either null or a pointer to an existing array of CK_SLOT_ID objects.
pulCount
{{ mediawiki.external('out') }} Pointer to a CK_ULONG variable which receives the slot count.;

Description

FC_GetSlotList obtains a list of slots in the system.

A user may call FC_GetSlotList without logging into the token (to assume the NSS User role).

Return value

CKR_OK
