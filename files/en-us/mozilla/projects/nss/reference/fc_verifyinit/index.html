Name

FC_VerifyInit - initialize a verification operation.

Syntax 

CK_RV FC_VerifyInit(
  CK_SESSION_HANDLE hSession,
  CK_MECHANISM_PTR pMechanism,
  CK_OBJECT_HANDLE hKey
);

Parameters

hSession
{{ mediawiki.external('in') }} session handle.
pMechanism
{{ mediawiki.external('in') }} mechanism to be used for the verification operation.
hKey
{{ mediawiki.external('in') }} handle of the key to be used.

Description

FC_VerifyInit initializes a verification operation where the signature is an appendix to the data.

A user must log into the token (to assume the NSS User role) before calling FC_VerifyInit.
