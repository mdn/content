Name

FC_CopyObject - create a copy of an object.

Syntax 

CK_RV FC_CopyObject(
  CK_SESSION_HANDLE hSession,
  CK_OBJECT_HANDLE hObject,
  CK_ATTRIBUTE_PTR pTemplate,
  CK_ULONG usCount,
  CK_OBJECT_HANDLE_PTR phNewObject
);

Parameters

hSession
{{ mediawiki.external('in') }} session handle.
hObject
{{ mediawiki.external('in') }} object handle.
pTemplate
{{ mediawiki.external('in') }} object template.
usCount
{{ mediawiki.external('in') }} number of attributes in the template.
phnewObject
{{ mediawiki.external('out') }} pointer to location to receive the new object's handle.

Description

FC_CopyObject creates a copy of an object using the attributes specified in the template.

A user must log into the token (to assume the NSS User role) before copying a secret or private key object.
