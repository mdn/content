Name

FC_DigestKey - add the digest of a key to a multi-part digest operation.

Syntax 

CK_RV FC_DigestKey(
  CK_SESSION_HANDLE hSession,
  CK_OBJECT_HANDLE hKey
);

Parameters

hSession
{{ mediawiki.external('in') }} session handle.
hKey
{{ mediawiki.external('in') }} handle of the key to be digested.

Description

FC_DigestKey continues a multi-part digest operation by digesting the value of a secret key. The digest for the entire message is returned by a call to FC_DigestFinal.

A user must log into the token (to assume the NSS User role) before calling FC_DigestKey.
