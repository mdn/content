Name

FC_Decrypt - Decrypt a block of data.

Syntax 

CK_RV FC_Decrypt(
  CK_SESSION_HANDLE hSession,
  CK_BYTE_PTR pEncryptedData,
  CK_ULONG usEncryptedDataLen,
  CK_BYTE_PTR pData,
  CK_ULONG_PTR pusDataLen
);

Parameters

hSession
{{ mediawiki.external('in') }} session handle.
pEncryptedData
{{ mediawiki.external('in') }} pointer to encrypted data block.
usEncryptedDataLen
{{ mediawiki.external('in') }} length of the data in bytes.
pData
{{ mediawiki.external('out') }} pointer to location where recovered data is to be stored.
pusDataLen
{{ mediawiki.external('in,out') }} pointer to location where the length of recovered data is to be stored.

Description

FC_Decrypt decrypts a block of data according to the attributes of the previous call to FC_DecryptInit.

A user must log into the token (to assume the NSS User role) before calling FC_Decrypt.
