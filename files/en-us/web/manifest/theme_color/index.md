<p>The <code>theme_color</code> member is a string that defines the default theme color for the application. This sometimes affects how the OS displays the site (e.g., on Android's task switcher, the theme color surrounds the site).</p>

<h2 id="Examples">Examples</h2>

<pre class="brush: json">"theme_color": "red"
</pre>
