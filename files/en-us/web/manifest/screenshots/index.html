The screenshots member defines an array of screenshots intended to showcase the application. These images are intended to be used by progressive web app stores.

Examples

"screenshots" : [
  {
    "src": "screenshot1.webp",
    "sizes": "1280x720",
    "type": "image/webp"
  },
  {
    "src": "screenshot2.webp",
    "sizes": "1280x720",
    "type": "image/webp"
  }
]
