The Age header contains the time in seconds the object has been in a proxy cache.

The Age header is usually close to zero. If it is Age: 0, it was probably just fetched from the origin server; otherwise It is usually calculated as a difference between the proxy's current date and the {{HTTPHeader("Date")}} general header included in the HTTP response.

Header type {{Glossary("Response header")}}
{{Glossary("Forbidden header name")}} no

Syntax

Age: <delta-seconds>

Directives

<delta-seconds>

A non-negative integer, representing time in seconds the object has been in a proxy cache.

Examples

Age: 24
