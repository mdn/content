The Permissions-Policy header was previously named Feature-Policy. As of June 2021, the {{domxref('FeaturePolicy')}} API as well as some linked-to articles and demos still use the older name.

The HTTP {{HTTPHeader("Permissions-Policy")}} header picture-in-picture directive controls whether the current document is allowed to play a video in a Picture-in-Picture mode via the corresponding API.

Syntax

Permissions-Policy: picture-in-picture <allowlist>;
<allowlist>
A list of origins for which the feature is allowed. See Permissions-Policy.

Default policy

As of June 2019, the spec draft and Google Chrome set default allow list to *.
