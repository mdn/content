The Permissions-Policy header was previously named Feature-Policy. As of June 2021, the {{domxref('FeaturePolicy')}} API as well as some linked-to articles and demos still use the older name.

This Permissions Policy directive was at one point defined as xr (but implemented in Chrome as {{httpheader("Permissions-Policy/vr", "vr")}}), use {{httpheader("Permissions-Policy/xr-spatial-tracking", "xr-spatial-tracking")}} instead.
