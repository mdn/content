The HTTP {{HTTPHeader("Feature-Policy")}} header screen-wake-lockÂ directive controls whether the current document is allowed to use Screen Wake Lock API to indicate that device should not dim or turn off the screen.

Note: In earlier specification drafts this directive was called {{HTTPHeader("Feature-Policy/wake-lock","wake-lock")}}.

Syntax

Feature-Policy: screen-wake-lock <allowlist>;
