The HTTP {{HTTPHeader("Feature-Policy")}} header `sync-xhr`Â directive controls whether the current document is allowed to make synchronous {{domxref("XMLHttpRequest")}} requests.

## Syntax

```
Feature-Policy: sync-xhr <allowlist>;
```

- \<allowlist>
  - : A list of origins for which the feature is allowed. See [`Feature-Policy`](/en-US/docs/Web/HTTP/Headers/Feature-Policy#syntax).

## Default policy

By default the policy is set to `*`, which means synchronous requests are allowed in all frames.
