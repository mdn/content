The HTTP {{HTTPHeader("Feature-Policy")}} header document-domain directive controls whether the current document is allowed to set {{domxref("document.domain")}}. When this policy is enabled, attempting to set {{domxref("document.domain")}} will fail and cause a SecurityError {{domxref("DOMException")}} to be thrown.

Syntax

Feature-Policy: document-domain <allowlist>;
<allowlist>
{{page("Web/HTTP/Feature_Policy/Using_Feature_Policy", "allowlist")}}

Default policy

The default value is *.
