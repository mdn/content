The HTTPÂ 101 Switching Protocols response code indicates the protocol the server is switching to as requested by a client which sent the message including the {{HTTPHeader("Upgrade")}} request header.

The server includes in this response an {{HTTPHeader("Upgrade")}} response header to indicate the protocol it switched to. The process is described in detail in the article Protocol upgrade mechanism.

Status

101 Switching Protocols

Examples

Switching protocols might be used with WebSockets.

HTTP/1.1 101 Switching Protocols
Upgrade: websocket
Connection: Upgrade
