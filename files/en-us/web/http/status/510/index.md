<p>The HyperText Transfer Protocol (HTTP)Â  <code><strong>510 Not Extended</strong></code>
  response status code is sent in the context of the HTTP Extension Framework, defined in
  <a href="https://datatracker.ietf.org/doc/html/rfc2774">RFC 2774</a>.</p>

<p>In that specification a client may send a request that contains an extension
  declaration, that describes the extension to be used. If the server receives such a
  request, but any described extensions are not supported for the request, then the server
  responds with the 510 status code.</p>
