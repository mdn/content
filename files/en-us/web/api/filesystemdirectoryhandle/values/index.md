The **`values()`** method of the
{{domxref("FileSystemDirectoryHandle")}} interface returns a new _array iterator_
containing the values for each index in the `FileSystemDirectoryHandle`
object.

## Syntax

```js
FileSystemDirectoryHandle.values();
```

### Parameters

None.

### Return value

A new {{jsxref('Array')}}
