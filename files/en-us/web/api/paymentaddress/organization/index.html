The organization read-only property of the {{domxref('PaymentAddress')}} interface returns a string containing the name of the organization, firm, company, or institution at the address.

Syntax

var paymentOrganization = PaymentAddress.organization;

Value

A {{domxref("DOMString")}} whose value is the name of the organization or company located at the address described by the PaymentAddress object. This should be the name of the organization that is to receive the shipment for shipping addresses, or which is repsonsible for payment for payment addresses.
