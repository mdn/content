{{APIRef("Payment Request API")}}

The city read-only property of the {{domxref('PaymentAddress')}} interface returns a string containing the city or town portion of the address.

Syntax

var paymentCity = PaymentAddress.city;

Value

A {{domxref("DOMString")}} indicating the city or town portion of the address described by the {{domxref("PaymentAddress")}} object.
