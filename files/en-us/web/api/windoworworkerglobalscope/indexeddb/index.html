The indexedDB read-only property of the {{domxref("WindowOrWorkerGlobalScope")}} mixin provides a mechanism for applications to asynchronously access the capabilities of indexed databases.

Syntax

var IDBFactory = self.indexedDB;

Value

An {{domxref("IDBFactory")}} object.

Example

The following code creates a request for a database to be opened asynchronously, after which the database is opened when the request's onsuccess handler is fired: 

var db;
function openDB() {
 var DBOpenRequest = window.indexedDB.open('toDoList');
 DBOpenRequest.onsuccess = function(e) {
   db = DBOpenRequest.result;
 }
}
