<p>The <code><strong>mark</strong></code> event of the <a href="/en-US/docs/Web/API/Web_Speech_API">Web Speech API</a> {{domxref("SpeechSynthesisUtterance")}} object is fired when the spoken utterance reaches a named SSML "mark" tag.</p>

<h2 id="Examples">Examples</h2>

<p>You can use the <code>mark</code> event in an <code><a href="/en-US/docs/Web/API/EventTarget/addEventListener">addEventListener</a></code> method:</p>

<pre class="brush: js">utterThis.addEventListener('mark', function(event) {
  console.log('A mark was reached: ' + event.name);
});</pre>

<p>Or use the <code><a href="/en-US/docs/Web/API/SpeechSynthesisUtterance/onmark">onmark</a></code> event handler property:</p>

<pre class="brush: js">utterThis.onmark = function(event) {
  console.log('A mark was reached: ' + event.name);
}</pre>
