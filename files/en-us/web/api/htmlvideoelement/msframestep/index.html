The HTMLVideoElement.msFrameStep() method steps the video by one frame forward or one frame backward.

This proprietary method is specific to Internet Explorer and Microsoft Edge.

Syntax

HTMLVideoElement.msFrameStep(forward);

Parameters

forward
A Boolean which if set to true steps the video forward by one frame, if false steps the video backwards by one frame.

Return value

This method does not return a value.
