Note: This method is deprecated in favor of {{DOMxRef("FileReaderSync.readAsArrayBuffer","readAsArrayBuffer()")}}.

The readAsBinaryString() method of the {{DOMxRef("FileReaderSync")}} interface allows to read {{DOMxRef("File")}} or {{DOMxRef("Blob")}} objects in a synchronous way into an {{DOMxRef("DOMString")}}. This interface is only available in workers as it enables synchronous I/O that could potentially block.

Syntax

readAsBinaryString(File);
readAsBinaryString(Blob);

Parameters

blob
The DOM {{DOMxRef("File")}} or {{DOMxRef("Blob")}} to read.

Return value

An {{DOMxRef("DOMString")}} representing the input data.
