{{APIRef("File API")}}
The readAsDataURL() method of the {{DOMxRef("FileReaderSync")}} interface allows to read {{DOMxRef("File")}} or {{DOMxRef("Blob")}} objects in a synchronous way into an {{DOMxRef("DOMString")}} representing a data URL. This interface is only available in workers as it enables synchronous I/O that could potentially block.

Syntax

readAsDataURL(File);
readAsDataURL(Blob);

Parameters

blob
The DOM {{DOMxRef("File")}} or {{DOMxRef("Blob")}} to read.

Return value

An {{DOMxRef("DOMString")}} representing the input data as a data URL.
