The ondevicechange property of the {{domxref("XRSystem")}} interface is passed a {{Event("devicechange")}} event whenever availability of an immersive device changes. The event that is fired is a "simple event" that implements the Event interface. 

Syntax

navigator.xr.ondevicechange = function(event) { ... };

Value

{{jsxref('undefined')}}

Example

navigator.xr.ondevicechange = function(ev) {
  console.log("The availability of immersive XR devices has changed.")
};
