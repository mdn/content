{{ APIRef("DOM") }}

The TreeWalker.previousNode() method moves the current {{domxref("Node")}} to the previous visible node in the document order, and returns the found node. It also moves the current node to this one. If no such node exists,or if it is before that the root node defined at the object construction, returns null and the current node is not changed.

Syntax

node = treeWalker.previousNode();

Example

var treeWalker = document.createTreeWalker(
    document.body,
    NodeFilter.SHOW_ELEMENT,
    { acceptNode: function(node) { return NodeFilter.FILTER_ACCEPT; } },
    false
);
var node = treeWalker.previousNode(); // returns null as there is no parent
