{{ APIRef("DOM") }}

The TreeWalker.nextNode() method moves the current {{domxref("Node")}} to the next visible node in the document order, and returns the found node. It also moves the current node to this one. If no such node exists, returns null and the current node is not changed.

Syntax

node = treeWalker.nextNode();

Example

var treeWalker = document.createTreeWalker(
    document.body,
    NodeFilter.SHOW_ELEMENT,
    { acceptNode: function(node) { return NodeFilter.FILTER_ACCEPT; } },
    false
);
var node = treeWalker.nextNode(); // returns the first child of root, as it is the next node in document order
