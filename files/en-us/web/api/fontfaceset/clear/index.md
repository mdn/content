{{APIRef("CSS Font Loading API")}}

The **`clear()`** method of the {{domxref("FontFaceSet")}} interface removes all fonts added via this interface. Fonts added with the {{cssxref("@font-face")}} rule are not removed.

## Syntax

```js
clear()
```

### Parameters

None.

### Return value

{{jsxref("Undefined")}}.
