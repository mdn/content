{{APIRef("HTML DOM")}}

The read-only track property of the {{domxref("TrackEvent")}} interface specifies the media track object to which the event applies. This will be an {{domxref("AudioTrack")}}, {{domxref("VideoTrack")}}, or {{domxref("TextTrack")}} object.

Syntax

track = TrackEvent.track;

Value

An object which is one of the types {{domxref("AudioTrack")}}, {{domxref("VideoTrack")}}, or {{domxref("TextTrack")}}, depending on the type of media represented by the track. This identifies the track to which the event applies.
