{{APIRef("File API")}}

The **`abort`** method aborts the read operation. Upon return,
the {{domxref("FileReader.readyState","readyState")}} will be `DONE`.

## Syntax

```js
abort()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).
