The paused read-only property of the {{domxref("SpeechSynthesis")}} interface is a {{domxref("Boolean")}} that returns true if the SpeechSynthesis object is in a paused state, or false if not.

It can be set to {{domxref("SpeechSynthesis.pause()", "paused")}} even if nothing is currently being spoken through it. If {{domxref("SpeechSynthesisUtterance","utterances")}} are then added to the utterance queue, they will not be spoken until the SpeechSynthesis object is unpaused, using {{domxref("SpeechSynthesis.resume()")}}.

Syntax

var amIPaused = speechSynthesisInstance.paused;

Value

A {{domxref("Boolean")}}.

Examples

var synth = window.speechSynthesis;

synth.pause();

var amIPaused = synth.paused; // will return true
