The resume() method of the {{domxref("SpeechSynthesis")}} interface puts the SpeechSynthesis object into a non-paused state: resumes it if it was already paused.

Syntax

speechSynthesisInstance.resume();

Returns

Void.

Parameters

None.

Examples

var synth = window.speechSynthesis;

var utterance1 = new SpeechSynthesisUtterance('How about we say this now? This is quite a long sentence to say.');
var utterance2 = new SpeechSynthesisUtterance('We should say another sentence too, just to be on the safe side.');

synth.speak(utterance1);
synth.speak(utterance2);

synth.pause(); // pauses utterances being spoken

synth.resume() // resumes speaking
