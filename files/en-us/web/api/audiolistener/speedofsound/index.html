The speedOfSound property of the {{ domxref("AudioListener") }} interface is a double value representing the speed of sound, in meters per second.

The speedOfSound property's default value is 343.3 m/s and is used to calculate the doppler shift appropriate for the speed the panner is travelling at (as defined by {{domxref("PannerNode.setVelocity")}}.)

Note: Bear in mind that no propagation delay is automatically applied to a sound far from the listener.

Syntax

var audioCtx = new AudioContext();
var myListener = audioCtx.listener;
myListener.speedOfSound = 343.3;

Value

A double.
