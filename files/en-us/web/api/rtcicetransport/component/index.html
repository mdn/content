{{APIRef("WebRTC")}}

The read-only {{domxref("RTCIceTransport")}} property component specifies whether the object is serving to transport RTP or RTCP. The value is one of the strings in {{domxref("RTCIceComponent")}}.

Syntax

iceComponent = RTCIceTransport.component;

Value

A {{domxref("DOMString")}} whose value is specified in RTCIceCandidate.component.
