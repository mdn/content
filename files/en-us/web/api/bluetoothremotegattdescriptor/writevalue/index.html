The BluetoothRemoteGATTDescriptor.writeValue() method sets the value property to the bytes contained in an {{jsxref("ArrayBuffer")}} and returns a {{jsxref("Promise")}}.

Syntax

BluetoothRemoteGATTDescriptor.writeValue(array[]).then(function() { ... })

Parameters

array
Sets the value with the bytes contained in the array.

Returns

A {{jsxref("Promise")}}.
