The BluetoothRemoteGATTDescriptor.value read-only property returns an {{jsxref("ArrayBuffer")}} containing the currently cached descriptor value. This value gets updated when the value of the descriptor is read.

Syntax

var characteristic = BluetoothRemoteGATTDescriptor.characteristic

Returns

An {{jsxref("ArrayBuffer")}}.
