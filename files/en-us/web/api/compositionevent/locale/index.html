{{deprecated_header}}{{APIRef("DOM Events")}}

The locale read-only property of the {{domxref("CompositionEvent")}} interface returns the locale of current input method (for example, the keyboard layout locale if the composition is associated with IME).

Syntax

myLocale = CompositionEvent.locale

Value

A {{domxref("DOMString")}} representing the locale of current input method.
