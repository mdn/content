{{deprecated_header}}{{APIRef("DOM Events")}}

The initCompositionEvent() method of the {{domxref("CompositionEvent")}} interface initializes the attributes of a CompositionEvent object instance.

Syntax

 compositionEventInstance.initCompositionEvent(typeArg, canBubbleArg, cancelableArg, viewArg, dataArg, localeArg)

Parameters

typeArg
A {{domxref("DOMString")}} representing the type of composition event; this will be one of compositionstart, compositionupdate, or compositionend.
canBubbleArg
A {{domxref("Boolean")}} specifying whether or not the event can bubble.
cancelableArg
A {{domxref("Boolean")}} whether or not the event can be canceled.
viewArg
The {{domxref("Window")}} object from which the event was generated.
dataArg
A {{domxref("DOMString")}} representing the value of the data attribute.
localeArg
A {{domxref("DOMString")}} representing the value of the locale attribute.

Return value

Void.
