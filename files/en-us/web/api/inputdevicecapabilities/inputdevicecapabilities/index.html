The InputDeviceCapabilities() constructor creates a new {{domxref("InputDeviceCapabilities")}} object provides information about the physical device responsible for generating a touch event.

Syntax

var inputDeviceCapabilities = new InputDeviceCapabilities([InputDeviceCapabilitiesInit])

Returns

An instance of the {{domxref("InputDeviceCapabilities")}} interface.

Parameters

InputDeviceCapabilitiesInit {{optional_inline}}
A dictionary object containing a set of device capabilities. It contains the following property.
  • fireTouchEvents: A {{jsxref("Boolean")}} that indicates whether the device dispatches touch events.
