The DocumentTouch interface used to provide convenience methods for creating {{DOMxRef("Touch")}} and {{DOMxRef("TouchList")}} objects, but DocumentTouch been removed from the standards. These two methods now live on the {{DOMxRef("Document")}} interface.

Methods

{{DOMxRef("Document.createTouch")}}
Creates a new {{DOMxRef("Touch")}} object.
{{DOMxRef("Document.createTouchList")}}
Creates a new {{DOMxRef("TouchList")}} object.
