{{APIRef("Web Audio API")}}

The numberOfInputs property of the {{domxref("AudioNode")}} interface returns the number of inputs feeding the node. Source nodes are defined as nodes having a numberOfInputs property with a value of 0.

Syntax

var numInputs = audioNode.numberOfInputs;

Value

An integer â‰¥ 0.

Example

const audioCtx = new AudioContext();

const oscillator = audioCtx.createOscillator();
const gainNode = audioCtx.createGain();

oscillator.connect(gainNode).connect(audioCtx.destination);

console.log(oscillator.numberOfInputs); // 0
console.log(gainNode.numberOfInputs); // 1
console.log(audioCtx.destination.numberOfInputs); // 1
