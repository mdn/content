The BlobEvent() constructor returns a newly created {{domxref("BlobEvent")}} object with an associated {{domxref("Blob")}}.

Syntax

blobEvent = new BlobEvent({data: aSpecificBlob}[, timecode]);

Arguments

The BlobEvent() constructor also inherits arguments from {{domxref("Event.Event", "Event()")}}.

data
Is a {{domxref("Blob")}} associated with the event.

A {{domxref("DOMHighResTimeStamp")}} to be used in initializing the blob event.
