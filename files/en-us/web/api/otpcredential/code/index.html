The code property of the {{domxref("OTPCredential")}} interface returns the one-time password.

Syntax

let code = OTPCredential.code;

Value

A {{domxref("DOMString","string")}} containing the one-time password.

Examples

The below code uses the value of code to complete an input form element. See this code as part of a simple demo.

navigator.credentials.get({
  otp: { transport:['sms'] },
  signal: ac.signal
}).then(otp => {
  input.value = otp.code;
  if (form) form.submit();
}).catch(err => {
  console.log(err);
});
