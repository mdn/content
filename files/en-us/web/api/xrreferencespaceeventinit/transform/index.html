The {{domxref("XRReferenceSpaceEventInit")}} property transform indicates the position and orientation of the affected reference space's native origin after the changes the event represents are applied. The transform is defined using the old coordinate system, which allows it to be used to convert coordinates from the pre-event coordinate system to the post-event coordiante system.

Syntax

let eventInitDict = {
  referenceSpace: xrReferenceSpace,
  transform: xrRigidTransform
});

Value

An {{domxref("XRRigidTransform")}} object providing a transform that can be used to convert coordinates from the pre-event coordinate system to the post-event coordinate system.
