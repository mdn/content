{{APIRef("WebXR Device API")}}{{SecureContext_header}}

The {{domxref("XRReferenceSpaceEventInit")}} property referenceSpace is used to establish the value of a newly-constructed {{domxref("XRReferenceSpaceEvent")}} object when calling the {{domxref("XRReferenceSpaceEvent.XRReferenceSpaceEvent", "XRReferenceSpaceEvent()")}} constructor.

Syntax

let eventInitDict = {
  referenceSpace: xrReferenceSpace,
  transform: xrRigidTransform
});

Value

An {{domxref("XRReferenceSpace")}} indicating the source of the event.
