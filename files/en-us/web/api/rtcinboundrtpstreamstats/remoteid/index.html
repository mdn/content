{{APIRef("WebRTC")}}

The remoteId property of the {{domxref("RTCInboundRtpStreamStats")}} dictionary specifies the {{domxref("RTCStats.id", "id")}} of the {{domxref("RTCRemoteOutboundRtpStreamStats")}} object representing the remote peer's {{domxref("RTCRtpSender")}} which is sending the media to the local peer.

Syntax

var remoteStatsId = rtcInboundRtpStreamStats.remoteId;

Value

A {{domxref("DOMString")}} containing the ID of the {{domxref("RTCRemoteOutboundRtpStreamStats")}} object that represents the remote peer's {{domxref("RTCRtpSender")}} for the synchronization source represented by this stats object.
