{{APIRef("Fetch")}}

The referrer read-only property of the {{domxref("Request")}} interface is set by the user agent to be the referrer of the Request. (e.g., client, no-referrer, or a URL.)

Note: If referrer's value is no-referrer, it returns an empty string.

Syntax

var myReferrer = request.referrer;

Value

A {{domxref("DOMString")}} representing the request's referrer.

Example

In the following snippet, we create a new request using the {{domxref("Request.Request()")}} constructor (for an image file in the same directory as the script), then save the request referrer in a variable:

var myRequest = new Request('flowers.jpg');
var myReferrer = myRequest.referrer; // returns "about:client" by default
