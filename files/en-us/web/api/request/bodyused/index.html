{{APIRef("Fetch")}}

The read-only bodyUsed property of the {{domxref("Request")}} interface is a {{jsxref("Boolean")}} that indicates whether the request body has been read yet.

Syntax

request.bodyUsed;

Value

A {{jsxref("Boolean")}}.

Examples

const request = new Request('/myEndpoint', {
  method: 'POST',
  body: 'Hello world'
});

request.bodyUsed; // false

request.text().then(function(bodyAsText) {
  console.log(request.bodyUsed); // true
});
