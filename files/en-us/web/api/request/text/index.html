{{APIRef("Fetch")}}

The text() method of the {{domxref("Request")}} interface reads the request body and returns it as a promise that resolves with a {{jsxref("String")}}. The response is always decoded using UTF-8.

Syntax

request.text().then(function (text) {
  // do something with the text sent in the request
});

Parameters

None.

Return value

A Promise that resolves with a {{jsxref("String")}}.

Examples

const text = "Hello world";

const request = new Request('/myEndpoint', {
  method: 'POST',
  body: text
});

request.text().then(function(text) {
  // do something with the text sent in the request
});
