The referrerPolicy read-only property of the {{domxref("Request")}} interface returns the referrer policy, which governs what referrer information, sent in the {{HTTPHeader("Referer")}} header, should be included with the request.

Syntax

var myReferrerPolicy = request.referrerPolicy;

Value

A {{domxref("DOMString")}} representing the request's referrerPolicy. For more information and possible values, see theÂ {{HTTPHeader("Referrer-Policy")}} HTTP header page.

Example

In the following snippet, we create a new request using the {{domxref("Request.Request()")}} constructor (for an image file in the same directory as the script), then save the request referrer policy in a variable:

var myRequest = new Request('flowers.jpg');
var myReferrer = myRequest.referrerPolicy; // returns "" by default
