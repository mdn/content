{{APIRef("DOM")}}{{Non-standard_header}}

The Element.scrollTopMax read-only property returns a {{jsxref("Number")}} representing the maximum top scroll offset possible for the element.

Syntax

var pxl = elt.scrollTopMax;

Specifications

This property is not part of any specification.
