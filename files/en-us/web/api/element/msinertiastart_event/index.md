<div>{{Non-standard_header}}</div>

<p>The <strong><code>MSInertiaStart</code></strong> event is fired when contact with the touch surface stops when a scroll has enough inertia to continue scrolling. This event may not be fired if the scroll is sufficiently slow.</p>

<p>It is a proprietary event specific to Microsoft Edge and Internet Explorer.</p>
