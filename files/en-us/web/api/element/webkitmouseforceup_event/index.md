<p>{{APIRef}}{{Non-standard_header()}}</p>

<p>The non-standard <code><strong>webkitmouseforceup</strong></code> event is fired by Safari at an {{domxref("Element")}} some time after the {{event("webkitmouseforcedown")}} event, when pressure on the button has been reduced sufficiently to end the "force click".</p>

 </tbody>

<p><strong><code>webkitmouseforceup</code></strong> is a proprietary, WebKit-specific event. It is part of the <a href="/en-US/docs/Web/API/Force_Touch_events">Force Touch events</a> feature.</p>

<h2 id="Specifications">Specifications</h2>

<p><em>Not part of any specification.</em> Apple has <a href="https://developer.apple.com/library/prerelease/mac/documentation/AppleApplications/Conceptual/SafariJSProgTopics/RespondingtoForceTouchEventsfromJavaScript.html">a description at the Mac Developer Library</a>.</p>
