The Element.requestPointerLock() method lets you asynchronously ask for the pointer to be locked on the given element.

To track the success or failure of the request, it is necessary to listen for the {{event("pointerlockchange")}} and {{event("pointerlockerror")}} events at the {{domxref("Document")}} level.

Syntax

instanceOfElement.requestPointerLock();
