{{APIRef('Shadow DOM')}}{{non-standard_header}}{{deprecated_header}}{{draft}}

Use Element.createShadowRoot to create an instance of shadow DOM. When shadow DOM is created, it is always attached to an existing element. After the shadow DOM is created, the element that it is attached to is called the {{glossary("shadow root")}}.

This method has been deprecated in favor of {{DOMxRef("Element.attachShadow()","attachShadow()")}}.

Syntax

var shadowroot = element.createShadowRoot();

Parameters

No parameters.

Result value

Returns a {{DOMxRef("ShadowRoot")}}.

Specifications

This feature is no longer defined by any specifications.
