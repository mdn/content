{{APIRef}}

The mouseup event is fired at an {{domxref("Element")}} when a button on a pointing device (such as a mouse or trackpad) is released while the pointer is located inside it. mouseup events are the counterpoint to {{domxref("Element.mousedown_event", "mousedown")}} events.

Bubbles Yes
Cancelable Yes
Interface {{domxref("MouseEvent")}}
Event handler property {{domxref("GlobalEventHandlers.onmouseup", "onmouseup")}}
