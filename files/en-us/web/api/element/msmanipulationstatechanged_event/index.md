<p><code><strong>MSManipulationStateChanged</strong></code> fires when the state of an element being manipulated has changed (ie. whenever you start or finish panning or zooming an element).</p>

<p>It is a proprietary event specific to Microsoft Edge and Internet Explorer.</p>

<p>Get manipulation states using the <code>lastState </code>and <code>currentState </code>properties.</p>

<h2 id="Examples">Examples</h2>

<pre class="brush: js">// Listen for panning state change events
outerScroller.addEventListener("MSManipulationStateChanged", function(e) {
    // Check to see if they lifted while pulled to the top
    if (e.currentState == MS_MANIPULATION_STATE_INERTIA &amp;&amp;
        outerScroller.scrollTop === 0) {
        refreshItemsAsync();
    }
});
</pre>
