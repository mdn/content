Summary

Element.runtimeStyle is a proprietary property similar to {{domxref("ElementCSSInlineStyle.style")}}, except its styles, that have higher precedence and modification. It does not modify the style content attribute. It is available in old versions of Microsoft Internet Explorer.

Specifications

Not a part of any specification.

Microsoft had a description on MSDN.
