{{APIRef("DOM Events")}}

The WheelEvent.deltaY read-only property is a double representing the vertical scroll amount in the {{domxref("WheelEvent.deltaMode")}} unit.

Syntax

var dY = event.deltaY;

Example

var syntheticEvent = new WheelEvent("syntheticWheel", {"deltaY": 4, "deltaMode": 0});

console.log(syntheticEvent.deltaY);
