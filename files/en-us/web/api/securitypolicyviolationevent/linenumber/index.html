The lineNumber read-only property of the {{domxref("SecurityPolicyViolationEvent")}} interface is the line number in the document or worker at which the violation occurred.

Syntax

let lineNumber = violationEventInstance.lineNumber;

Value

A number representing the line number at which the violation occurred.

Example

document.addEventListener("securitypolicyviolation", (e) => {
  console.log(e.lineNumber);
});
