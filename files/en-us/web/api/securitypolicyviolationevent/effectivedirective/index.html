The effectiveDirective read-only property of the {{domxref("SecurityPolicyViolationEvent")}} interface is a {{domxref("DOMString")}} representing the directive whose enforcement uncovered the violation.

Syntax

let effDir = violationEventInstance.effectiveDirective;

Value

A {{domxref("DOMString")}} representing the directive whose enforcement uncovered the violation.

Example

document.addEventListener("securitypolicyviolation", (e) => {
  console.log(e.effectiveDirective);
});
