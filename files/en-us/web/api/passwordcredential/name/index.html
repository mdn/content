The name read-only property of the {{domxref("PasswordCredential")}} interface returns a {{domxref("USVSTring")}} containing a human-readable public name for display in a credential chooser.

Syntax

name =passwordCredential.name

Value

A {{domxref("USVString")}} containing a name.
