The **`x`** property of the
{{domxref("CSSScale")}} interface gets and sets the abscissa or x-axis of the
translating vector.

## Syntax

```js
var scaleX = CSSScale.x;
```

### Value

A double integer or a {{domxref("CSSNumericValue")}}
