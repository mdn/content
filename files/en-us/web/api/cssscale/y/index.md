{{APIRef("CSS Typed OM")}}

The **`y`** property of the
{{domxref("CSSScale")}} interface gets and sets the ordinate or y-axis of the
translating vector.

## Value

A {{domxref("CSSNumericValue")}}. Can be set to a number, which is wrapped in a {{domxref("CSSUnitValue")}} of `unit: "number"`.
