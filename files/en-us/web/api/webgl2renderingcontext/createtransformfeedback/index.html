{{APIRef("WebGL")}}

The WebGL2RenderingContext.createTransformFeedback() method of the WebGL 2 API creates and initializes {{domxref("WebGLTransformFeedback")}} objects.

Syntax

WebGLTransformFeedback gl.createTransformFeedback();

Parameters

None.

Return value

A {{domxref("WebGLTransformFeedback")}} object.

Examples

gl must be a {{domxref("WebGL2RenderingContext")}}. WebGLTransformFeedback objects are not available in WebGL 1.

var transformFeedback = gl.createTransformFeedback();
