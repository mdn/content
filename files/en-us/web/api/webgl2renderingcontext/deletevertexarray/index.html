{{APIRef("WebGL")}}

The WebGL2RenderingContext.deleteVertexArray() method of the WebGL 2 API deletes a given {{domxref("WebGLVertexArrayObject")}} object.

Syntax

void gl.deleteVertexArray(vertexArray);

Parameters

vertexArray
A {{domxref("WebGLVertexArrayObject")}} (VAO) object to delete.

Return value

None.

Examples

var vao = gl.createVertexArray();
gl.bindVertexArray(vao);

// ...

gl.deleteVertexArray(vao);
