{{APIRef("WebGL")}}

The WebGL2RenderingContext.readBuffer() method of the WebGL 2 API selects a color buffer as the source for pixels for subsequent calls to {{domxref("WebGLRenderingContext.copyTexImage2D", "copyTexImage2D")}}, {{domxref("WebGLRenderingContext.copyTexSubImage2D", "copyTexSubImage2D")}}, {{domxref("WebGL2RenderingContext.copyTexSubImage3D", "copyTexSubImage3D")}} or {{domxref("WebGLRenderingContext.readPixels", "readPixels")}}.

Syntax

void gl.readBuffer(src);

Parameters

src
A {{domxref("GLenum")}} specifying a color buffer. Possible values:
  • gl.BACK: Reads from the back color buffer.
  • gl.NONE: Reads from no color buffer.
  • gl.COLOR_ATTACHMENT{0-15}: Reads from one of the 16 color attachment buffers.

Return value

None.

Examples

gl.readBuffer(gl.COLOR_ATTACHMENT0);
