Syntax

Parameters

target
A {{domxref("GLenum")}} specifying the target (binding point). Must be gl.TRANSFORM_FEEDBACK.
transformFeedback
A {{domxref("WebGLTransformFeedback")}} object to bind.

None.

Examples

gl.bindTransformFeedback(gl.TRANSFORM_FEEDBACK, transformFeedback);
