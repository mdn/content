{{APIRef("WebGL")}}

The WebGL2RenderingContext.isTransformFeedback() method of the WebGL 2 API returns true if the passed object is a valid {{domxref("WebGLTransformFeedback")}} object.

Syntax

GLboolean gl.isTransformFeedback(transformFeedback);

Parameters

transformFeedback
A {{domxref("WebGLTransformFeedback")}} object to test.

Return value

A {{domxref("GLboolean")}} indicating whether the given object is a valid {{domxref("WebGLTransformFeedback")}} object (true) or not (false).

Examples

gl must be a {{domxref("WebGL2RenderingContext")}}. WebGLTransformFeedback objects are not available in WebGL 1.

var transformFeedback = gl.createTransformFeedback();

// ...

gl.isTransformFeedback(transformFeedback);
