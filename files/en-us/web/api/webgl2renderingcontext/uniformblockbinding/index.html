{{APIRef("WebGL")}}

The WebGL2RenderingContext.uniformBlockBinding() method of the WebGL 2 API assigns binding points for active uniform blocks.

Syntax

void gl.uniformBlockBinding(program, uniformBlockIndex, uniformBlockBinding);

Parameters

program
A {{domxref("WebGLProgram")}} containing the active uniform block whose binding to assign.
uniformBlockIndex
A {{domxref("GLuint")}} specifying the index of the active uniform block within the program.
uniformBlockBinding
A {{domxref("GLuint")}} specifying the binding point to which to bind the uniform block.

Return value

None.

Examples

gl.uniformBlockBinding(program, 0, 1);
