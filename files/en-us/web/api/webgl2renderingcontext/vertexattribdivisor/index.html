{{APIRef("WebGL")}}

The WebGL2RenderingContext.vertexAttribDivisor() method of the WebGL 2 API modifies the rate at which generic vertex attributes advance when rendering multiple instances of primitives with {{domxref("WebGL2RenderingContext.drawArraysInstanced()", "gl.drawArraysInstanced()")}} and {{domxref("WebGL2RenderingContext.drawElementsInstanced()", "gl.drawElementsInstanced()")}}.

Note: When using {{domxref("WebGLRenderingContext", "WebGL 1", "", 1)}}, the {{domxref("ANGLE_instanced_arrays")}} extension can provide this method, too.

Syntax

void gl.vertexAttribDivisor(index, divisor);

Parameters

index
A {{domxref("GLuint")}} specifying the index of the generic vertex attributes.
divisor
A {{domxref("GLuint")}} specifying the number of instances that will pass between updates of the generic attribute.

Return value

None.

Examples

gl.vertexAttribDivisor(0, 2);
