Syntax

void gl.bindSampler(unit, sampler);

Parameters

unit
A {{domxref("GLuint")}} specifying the index of the texture unit to which to bind the sampler to.
sampler
A {{domxref("WebGLSampler")}} object to bind.

Return value

None.

Examples

var sampler = gl.createSampler();
gl.bindSampler(0, sampler);
