The WebGL2RenderingContext.getQueryParameter() method of the WebGL 2 API returns parameter information of a {{domxref("WebGLQuery")}} object.

Syntax

any gl.getQueryParameter(query, pname);

Parameters

query
A {{domxref("WebGLQuery")}} object.
pname
A {{domxref("GLenum")}} specifying which information to return. Possible values:
  • gl.QUERY_RESULT: Returns a {{domxref("GLuint")}} containing the query result.
  • gl.QUERY_RESULT_AVAILABLE: Returns a {{domxref("GLboolean")}} indicating whether or not a query result is available.

Return value

Depends on the pname parameter, either a {{domxref("GLuint")}} or a {{domxref("GLboolean")}}.

Examples

var query = gl.createQuery();
gl.beginQuery(gl.ANY_SAMPLES_PASSED, query);

var result = gl.getQueryParameter(query, gl.QUERY_RESULT);
