The {{domxref("RTCTrackEventInit")}} dictionary's transceiver property specifies the {{domxref("RTCRtpTransceiver")}} associated with the {{event("track")}} event. The transceiver pairs the track's {{domxref("RTCTrackEvent.receiver", "receiver")}} with an {{domxref("RTCRtpSender")}} to allow bidirectional communication.

Syntax

var trackEventInit = {
  receiver: rtpReceiver,
  track: mediaStreamTrack,
  streams: [videoStream],
  transceiver: rtpTransceiver
};

var rtpTransceiver = trackEventInit.transceiver;
