The {{domxref("RTCTrackEventInit")}} dictionary's receiver property specifies the {{domxref("RTCRtpReceiver")}} associated with the event.

Syntax

var trackEventInit = {
  receiver: rtpReceiver,
  track: mediaStreamTrack,
  streams: [videoStream],
  transceiver: rtpTransceiver
};

var rtpReceiver = trackEventInit.receiver;
