{{APIRef("URL API")}}

The keys() method of the {{domxref("URLSearchParams")}} interface returns an {{jsxref("Iteration_protocols",'iterator')}} allowing iteration through all keys contained in this object. The keys are {{domxref("USVString")}} objects.

Note: This method is available in Web Workers.

Syntax

searchParams.keys();

Parameters

None.

Return value

Returns an {{jsxref("Iteration_protocols","iterator")}}.

Examples

// Create a test URLSearchParams object
var searchParams = new URLSearchParams("key1=value1&key2=value2");

// Display the keys
for(var key of searchParams.keys()) {
  console.log(key);
}

The result is:

key1
key2
