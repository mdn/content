{{ApiRef("URL API")}}

The has() method of the {{domxref("URLSearchParams")}} interface returns a {{jsxref("Boolean")}} that indicates whether a parameter with the specified name exists.

{{availableinworkers}}

Syntax

var hasName = URLSearchParams.has(name)

Parameters

name
The name of the parameter to find.

Return value

A {{jsxref("Boolean")}}.

Examples

let url = new URL('https://example.com?foo=1&bar=2');
let params = new URLSearchParams(url.search.slice(1));

params.has('bar') === true; //true
