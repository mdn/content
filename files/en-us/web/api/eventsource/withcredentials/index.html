The withCredentials read-only property of the {{domxref("EventSource")}} interface returns a {{domxref("Boolean")}} indicating whether the EventSource object was instantiated with CORS credentials set.

Syntax

var myWithCredentials = eventSource.withCredentials;

Value

A {{domxref("Boolean")}} indicating whether the EventSource object was instantiated with CORS credentials set (true), or not (false, the default).

Examples

var evtSource = new EventSource('sse.php');
console.log(evtSource.withCredentials);
