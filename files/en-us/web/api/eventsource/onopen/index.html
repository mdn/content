{{APIRef('Server Sent Events')}}

The onopen property of the {{domxref("EventSource")}} interface is an {{event("Event_handlers", "event handler")}} called when an {{event("open")}} event is received, that is when the connection was just opened.

Syntax

eventSource.onopen = function

Examples

evtSource.onopen = function() {
  console.log("Connection to server opened.");
};

Note: You can find a full example on GitHub — see Simple SSE demo using PHP.
