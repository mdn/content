The onerror property of the {{domxref("EventSource")}} interface is an {{domxref("EventHandler")}} called when an error occurs and the {{event("error")}} event is dispatched on an EventSource object.

Syntax

eventSource.onerror = function

Examples

evtSource.onerror = function() {
  console.log("EventSource failed.");
};

Note: You can find a full example on GitHub — see Simple SSE demo using PHP.
