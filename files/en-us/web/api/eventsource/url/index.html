The url read-only property of the {{domxref("EventSource")}} interface returns a {{domxref("DOMString")}} representing the URL of the source.

Syntax

var myUrl = eventSource.url;

Value

A {{domxref("DOMString")}} representing the URL of the source.

Examples

var evtSource = new EventSource('sse.php');
console.log(evtSource.url);
