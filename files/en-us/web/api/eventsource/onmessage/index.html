The onmessage property of the {{domxref("EventSource")}} interface is an {{domxref("EventHandler")}} called when a {{event("message")}} event is received, that is when a message is coming from the source.

Event objects of onmessage event handlers are of type {{domxref("MessageEvent")}}.

Syntax

eventSource.onmessage = function

Examples

evtSource.onmessage = function(e) {
  var newElement = document.createElement("li");

  newElement.textContent = "message: " + e.data;
  eventList.appendChild(newElement);
}
