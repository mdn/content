{{ApiRef("DOM")}}

The Range.getClientRects() method returns a list of {{ domxref("DOMRect") }} objects representing the area of the screen occupied by the range. This is created by aggregating the results of calls to {{ domxref("Element.getClientRects()") }} for all the elements in the range.

Syntax

rectList = range.getClientRects()

Example

range = document.createRange();
range.selectNode(document.getElementsByTagName("div").item(0));
rectList = range.getClientRects();
