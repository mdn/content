<div>{{ApiRef("DOM")}}</div>

<p>The <strong><code>Range.setStartAfter()</code></strong> method sets the start position
  of a {{domxref("Range")}} relative to a {{domxref("Node")}}. The parent
  {{domxref("Node")}} of the start of the {{domxref("Range")}} will be the same as that
  for the <code>referenceNode</code>.</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js"><em>range</em>.setStartAfter(<em>referenceNode</em>);</pre>

<h3 id="Parameters">Parameters</h3>

<dl>
  <dt><em>referenceNode</em></dt>
  <dd>The {{domxref("Node")}} to start the {{domxref("Range")}} after.</dd>
</dl>

<h2 id="Example">Example</h2>

<pre class="brush:js">var range = document.createRange();
var referenceNode = document.getElementsByTagName("div").item(0);

range.setStartAfter(referenceNode);
</pre>
