{{APIRef("DOM")}}

The Range.collapse() method collapses the {{domxref("Range")}} to one of its boundary points.

A collapsed {{domxref("Range")}} is empty, containing no content, specifying a single-point in a DOM tree. To determine if a {{domxref("Range")}} is already collapsed, see the {{domxref("Range.collapsed")}} property.

Syntax

range.collapse(toStart);

Parameters

toStart {{optional_inline}}
A {{jsxref("Boolean")}} value: true collapses the {{domxref("Range")}} to its start, false to its end. If omitted, it defaults to false {{experimental_inline}}.

Example

var range = document.createRange();

referenceNode = document.getElementsByTagName("div").item(0);
range.selectNode(referenceNode);
range.collapse(true);
