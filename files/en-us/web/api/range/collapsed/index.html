{{ APIRef("DOM") }}

The Range.collapsed read-only property returns a {{domxref("Boolean")}} flag indicating whether the start and end points of the {{domxref("Range")}} are at the same position. It returns true if the start and end boundary points of the {{domxref("Range")}} are the same point in the DOM, false if not.

A collapsed {{domxref("Range")}} is empty (containing no content), and specifies a single point in a DOM tree. To collapse a range, see the {{domxref("Range.collapse()")}} method.

Syntax

isCollapsed = range.collapsed;

Example

let range = document.createRange();

range.setStart(startNode, startOffset);
range.setEnd(endNode, endOffset);
isCollapsed = range.collapsed;
