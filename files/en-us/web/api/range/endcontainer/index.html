{{ApiRef("DOM")}}

The Range.endContainer read-only property returns the {{domxref("Node")}} within which the {{domxref("Range")}} ends. To change the end position of a node, use the {{domxref("Range.setEnd()")}} method or a similar one.

Syntax

endRangeNode = range.endContainer;

Example

var range = document.createRange();

range.setStart(startNode,startOffset);
range.setEnd(endNode,endOffset);
endRangeNode = range.endContainer;
