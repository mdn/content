The **`close()`** method of the {{domxref("VideoDecoder")}} interface ends all pending work and releases system resources.

## Syntax

```js
close()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).

## Examples

The following example closes the `VideoDecoder`.

```js
VideoDecoder.close();
```
