{{APIRef("CSS Typed OM")}}

The **`CSSUnparsedValue.values()`** method
returns a new _array iterator_ object that contains the
values for each index in the CSSUnparsedValue object.

## Syntax

```js-nolint
values()
```

### Parameters

None.

### Return value

A new {{jsxref("Array")}}.
