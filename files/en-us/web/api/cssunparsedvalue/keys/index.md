{{APIRef("CSS Typed OM")}}{{SeeCompatTable}}

The **`CSSUnparsedValue.keys()`** method
returns a new _array iterator_ object that contains the keys
for each index in the array.

## Syntax

```js
keys()
```

### Parameters

None.

### Return value

A new {{jsxref("Array")}}.
