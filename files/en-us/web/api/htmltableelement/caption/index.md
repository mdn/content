{{APIRef("HTML DOM")}}

The **`HTMLTableElement.caption`** property represents the
table caption. If no caption element is associated with the table, this property is
`null`.

## Value

A string.

## Examples

```js
if (table.caption) {
  // Do something with the caption
}
```
