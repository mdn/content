{{APIRef("HTML DOM")}}

The **`HTMLTableElement.tHead`** represents the
{{HTMLElement("thead")}} element of a {{HTMLElement("table")}} . Its value will be
`null` if there is no such element.

## Value

A {{domxref("HTMLTableSectionElement")}}.
