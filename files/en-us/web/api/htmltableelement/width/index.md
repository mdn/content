{{APIRef("HTML DOM")}} {{Deprecated_Header}}

The **`HTMLTableElement.width`** property represents the
desired width of the table.

## Value

A string representing the width in number of pixels or as a percentage value.

## Examples

```js
mytable.width = "75%";
```
