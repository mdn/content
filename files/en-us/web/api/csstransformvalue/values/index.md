The **`CSSTransformValue.values()`**  returns a
new *array iterator* object that contains the values for
each index in the CSSTransformValue object.

## Syntax

```js
CSSTransformValue.values();
```

### Parameters

None

### Return value

A new {{jsxref("Array")}}.
