The CSSTransformValue interface of the {{domxref('CSS Typed Object Model API','','',' ')}} represents transform-list values as used by the transform property.

Properties

{{domxref("CSSTransformValue.is2D")}}
Returns a boolean as to whether the transform is 2D or 3D. In the case of the CSSTransformValue returns true unless and of the individual functions return false for Is2D, in which case it returns false. {{ReadOnlyInline}}
