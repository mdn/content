{{APIRef("WebXR Device API")}}{{SecureContext_Header}}

The inputsourceschange event is sent to an {{domxref("XRSession")}} when the set of available WebXR input devices changes. The received event, of type {{domxref("XRInputSourcesChangeEvent")}}, contains a list of any newly {{domxref("XRInputSourcesChangeEvent.added", "added")}} and/or {{domxref("XRInputSourcesChangeEvent.removed", "removed")}} input devices.

Bubbles Yes
Cancelable No
Interface {{domxref("XRInputSourcesChangeEvent")}}
Event handler property {{domxref("XRSession.oninputsourceschange", "oninputsourceschange")}}

The event object contains lists of the newly-added and/or removed input devices in its {{domxref("XRInputSourcesChangeEvent.added", "added")}} and {{domxref("XRInputSourcesChangeEvent.removed", "removed")}} properties.
