The {{domxref("XRSession")}} interface's read-only environmentBlendMode property identifies if—and to  what degree—the computer-generated imagery is overlaid atop the real world. This is used to differentiate between fully-immersive VR sessions and AR sessions which render over a pass-through image of the real world, possibly partially transparently.

The value is a {{domxref("DOMString")}} which contains one of the values defined by the {{domxref("XREnvironmentBlendMode")}} enumerated type.

Important: environmentBlendMode is part of the WebXR Augmented Reality Module, which has not yet reached a stable state. You should use caution when using its features.

Syntax

blendMode = xrSession.environmentBlendMode;

Value

A {{domxref("DOMString")}} whose value is one of the strings found in the enumerated type {{domxref("XREnvironmentBlendMode")}}, defining if—and if so, how—virtual, rendered content is overlaid atop the image of the real world.
