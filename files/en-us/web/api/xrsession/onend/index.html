The onend attribute of the {{DOMxRef("XRSession")}} object is the event handler for the {{DOMxRef("XRSession.end_event", "end")}} event, which is dispatched after the XR session ends and all related hardware-specific routines are completed.

Syntax

XRSession.onend = function(event) { ... };

Example

XRSession.onend = function(event) {
  console.log("The XR session has ended")
}
