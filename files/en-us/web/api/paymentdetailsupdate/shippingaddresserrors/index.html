Syntax

var addressErrors = PaymentDetailsUpdate.shippingAddressErrors;

Value

An {{domxref("AddressErrors")}} object, which contains {{domxref("DOMString")}}s describing errors in the properties of a {{domxref("PaymentAddress")}}. For each property in PaymentAddress, a property by the same name is found in shippingAddressErrors if and only if a validation error occurred for that property. In that case, the property in shippingAddressErrors is a string describing the validation error, ideally including suggestions about fixing the error.
