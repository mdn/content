{{APIRef("HTML DOM")}}

The read-only {{domxref("VideoTrack")}} property sourceBuffer returns the {{domxref("SourceBuffer")}} that created the track, or null if the track was not created by a {{domxref("SourceBuffer")}} or the {{domxref("SourceBuffer")}} has been removed from the {{domxref("MediaSource.sourceBuffers")}} attribute of its parent media source.

Syntax

var sourceBuffer = VideoTrack.sourceBuffer;

Value

A {{domxref("SourceBuffer")}} or null.
