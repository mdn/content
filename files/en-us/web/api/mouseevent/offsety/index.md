<p>The <strong><code>offsetY</code></strong> read-only property of the
  {{domxref("MouseEvent")}} interface provides the offset in the Y coordinate of the mouse
  pointer between that event and the padding edge of the target node.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js">var <em>yOffset</em> = <em>instanceOfMouseEvent</em>.offsetY;
</pre>

<h3 id="Return_value">Return value</h3>

<p>A <code>double</code> floating point value. Early versions of the spec defined this as
  an integer. See the Browser compatibility section for details.</p>
