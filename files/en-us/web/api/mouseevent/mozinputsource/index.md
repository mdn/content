<p>The <strong><code>MouseEvent.mozInputSource</code></strong> read-only property on
  {{domxref("MouseEvent")}} provides information indicating the type of device that
  generated the event. This lets you, for example, determine whether a mouse event was
  generated by an actual mouse or by a touch event (which might affect the degree of
  accuracy with which you interpret the coordinates associated with the event).</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js">var <em>source</em> = <em>instanceOfMouseEvent</em>.mozInputSource;</pre>
