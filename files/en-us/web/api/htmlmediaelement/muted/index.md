{{APIRef("HTML DOM")}}

The **`HTMLMediaElement.muted`** indicates whether the media
element muted.

## Value

A boolean value. `true` means muted and `false` means
not muted.

## Examples

```js
const obj = document.createElement('video');
console.log(obj.muted); // false
```
