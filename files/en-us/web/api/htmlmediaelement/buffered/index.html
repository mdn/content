{{APIRef("HTML DOM")}}

The buffered read-only property of {{domxref("HTMLMediaElement")}} objects returns a new static normalized TimeRanges object that represents the ranges of the media resource, if any, that the user agent has buffered at the moment the buffered property is accessed.

Note: This feature is not available in Web Workers.

Syntax

var timeRange = audioObject.buffered

Value

A new static normalized TimeRanges object that represents the ranges of the media resource, if any, that the user agent has buffered at the moment the buffered property is accessed.

Example

var obj = document.createElement('video');
console.log(obj.buffered); // TimeRanges { length: 0 }
