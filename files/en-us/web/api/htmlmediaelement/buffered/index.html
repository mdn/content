{{APIRef("HTML DOM")}}

The HTMLMediaElement.buffered read-only property returns a new {{domxref("TimeRanges")}} object that indicates the ranges of the media source that the browser has buffered (if any) at the moment the buffered property is accessed.

Note: This feature is not available in Web Workers.

Syntax

var timeRange = audioObject.buffered

Value

A {{domxref("TimeRanges")}} object. This object is normalized, which means that ranges are ordered, don't overlap, aren't empty, and don't touch (adjacent ranges are folded into one bigger range).

Example

var obj = document.createElement('video');
console.log(obj.buffered); // TimeRanges { length: 0 }
