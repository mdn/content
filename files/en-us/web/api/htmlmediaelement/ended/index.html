{{APIRef("HTML DOM")}}

The HTMLMediaElement.ended indicates whether the media element has ended playback.

Syntax

var isEnded = HTMLMediaElement.ended

Value

A {{domxref("Boolean")}} which is true if the media contained in the element has finished playing.

If the source of the media is a {{domxref("MediaStream")}}, this value is true if the value of the stream's {{domxref("MediaStream.active", "active")}} property is false.

Example

var obj = document.createElement('video');
console.log(obj.ended); // false
