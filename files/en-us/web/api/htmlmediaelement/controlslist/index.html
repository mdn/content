The controlsList property of the {{domxref("HTMLMediaElement")}} interface returns a DOMTokenList that helps the user agent select what controls to show on the media element whenever the user agent shows its own set of controls. The DOMTokenList takes one or more of three possible values: nodownload, nofullscreen, and noremoteplayback.

Syntax

var domTokenList = HTMLMediaElement.controlsList;

Value

A {{domxref("DOMTokenList")}}.
