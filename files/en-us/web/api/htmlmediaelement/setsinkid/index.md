<p>{{APIRef("HTML DOM")}} {{SeeCompatTable}}</p>

<p>The <strong><code>HTMLMediaElement.setSinkId()</code></strong> method sets the ID of
  the audio device to use for output and returns a <a
    href="/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise"
    title="The Promise object is used for deferred and asynchronous computations. A Promise represents an operation that hasn't completed yet, but is expected in the future."><code>Promise</code></a>.
  This only works when the application is authorized to use the specified device.</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js"><em>HTMLMediaElement</em>.setSinkId(<em>sinkId</em>).then(function() { ... })</pre>

<h3 id="Returns">Returns</h3>

<p>A {{jsxref("Promise")}} that resolves to {{jsxref("undefined")}}.</p>

<h3 id="Parameters">Parameters</h3>

<dl>
  <dt>sinkId</dt>
  <dd>The {{domxref("MediaDeviceInfo.deviceId")}} of the audio output device.</dd>
</dl>

<h2 id="Examples">Examples</h2>

<pre class="brush:js">const devices = await navigator.mediaDevices.enumerateDevices();
const audioDevices = devices.filter(device =&gt; device.kind === 'audiooutput');
const audio = document.createElement('audio');
await audio.setSinkId(audioDevices[0].deviceId);
console.log('Audio is being played on ' + audio.sinkId);
</pre>
