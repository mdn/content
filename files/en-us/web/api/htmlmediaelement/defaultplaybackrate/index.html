The HTMLMediaElement.defaultPlaybackRate property indicates the default playback rate for the media.

Syntax

var dSpeed = video.defaultPlaybackRate;
audio.defaultPlaybackRate = 1.0;

Value

A double.Â 1.0 is "normal speed," values lower than 1.0 make the media play slower than normal, higher values make it play faster. The value 0.0 is invalid and throws a NOT_SUPPORTED_ERR exception.

Example

var obj = document.createElement('video');
console.log(obj.defaultPlaybackRate); // 1
