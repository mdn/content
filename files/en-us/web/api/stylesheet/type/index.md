{{APIRef("CSSOM")}}

The **`type`** property of the {{domxref("StyleSheet")}}
interface specifies the style sheet language for the given style sheet.

## Syntax

```js
string = stylesheet.type
```

## Example

```js
 myStyleSheet.type = 'text/css';
```
