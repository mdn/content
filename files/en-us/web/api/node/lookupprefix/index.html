{{APIRef("DOM")}}

The Node.lookupPrefix() method returns a {{domxref("DOMString")}} containing the prefix for a given namespace URI, if present, and null if not. When multiple prefixes are possible, the result is implementation-dependent.

Due to bug 312019, this method does not work with dynamically assigned namespaces, that is, those set with the {{domxref("Node.prefix")}} property.
