The onloadeddata property of the {{domxref("GlobalEventHandlers")}} mixin is the {{event("Event_handlers", "event handler")}} for processing {{event("loadeddata")}} events.

The loadeddata event is fired when the first frame of the media has finished loading.

Syntax

element.onloadeddata = handlerFunction;
var handlerFunction = element.onloadeddata;

handlerFunction is either null or a JavaScript function specifying the handler for the event.
