The onloadstart property sets and returns the event handler for the {{event("loadstart")}} event.

Syntax

element.onloadstart = handlerFunction;
var handlerFunction = element.onloadstart;

handlerFunction should be either null or a JavaScript function specifying the handler for the event.

Notes

See the DOM event handlers page for information on working with on... handlers.

The loadstart event is fired when progress has begun.

See the {{event("loadstart")}} event documentation for more information about the event.
