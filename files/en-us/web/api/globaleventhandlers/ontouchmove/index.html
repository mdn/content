{{ApiRef("HTML DOM")}}

A {{domxref("GlobalEventHandlers","global event handler")}} for the {{event("touchmove")}} event.

{{SeeCompatTable}}

Note: This attribute has not been formally standardized. It is specified in the {{SpecName('Touch Events 2')}} {{Spec2('Touch Events 2')}} specification and not in {{SpecName('Touch Events')}} {{Spec2('Touch Events')}}. This attribute is not widely implemented.

Syntax

var moveHandler = someElement.ontouchmove;

Return value

moveHandler
The touchmove event handler for element someElement.

Example

This example shows two ways to use ontouchmove to set an element's touchmove event handler.

<html>
<head>
<script>

function moveTouch(ev) {
 // Process the event
}

function init() {
 var el=document.getElementById("target1");
 el.ontouchmove = moveTouch;
}

</script>
</head>

<body onload="init();">
<div id="target1"> Touch me ... </div>
<div id="target2" ontouchmove="moveTouch(event)"> Touch me ... </div>
</body>
</html>
