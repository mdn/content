The onselectstart property of the {{domxref("GlobalEventHandlers")}} mixin is an {{domxref("EventHandler")}} that processes {{event("selectstart")}} events.

The selectstart event fires when the user starts to make a new text selection on a webpage.

Syntax

object.onselectstart = functionRef;

Value

functionRef is a function name or a function expression. The function receives a {{domxref("FocusEvent")}} object as its sole argument.

Examples

document.onselectstart = function() {
  console.log('Selection started!');
};

For a full example, see our Key quote generator demo.

document.onselectstart = () => {
  return false; // cancel selection
};
