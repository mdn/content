{{SeeCompatTable}}{{APIRef("Media Session API")}}

The album property of the {{domxref("MediaMetaData")}} interface returns or sets the name of the album or collection containing the media to be played.

Syntax

var album = mediaMetaData.album
mediaMetaData.album = album

Value

A {{jsxref("String")}} containing the name of the album.
