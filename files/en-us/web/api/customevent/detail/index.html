{{APIRef("DOM")}}

The detail readonly property of the {{domxref("CustomEvent")}} interface returns any data passed when initializing the event.

{{AvailableInWorkers}}

Syntax

 let myDetail = customEventInstance.detail;

Return value

Whatever data the event was initialized with.

Example

// add an appropriate event listener
obj.addEventListener("cat", function(e) { process(e.detail) });

// create and dispatch the event
let event = new CustomEvent("cat", {
  detail: {
    hazcheeseburger: true
  }
});
obj.dispatchEvent(event);

// Will return an object containing the hazcheeseburger property
let myDetail = event.detail;
