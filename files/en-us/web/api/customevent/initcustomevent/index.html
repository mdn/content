{{APIRef("DOM")}}

The CustomEvent.initCustomEvent() method initializes a CustomEvent object. If the event has already been dispatched, this method does nothing.

Events initialized in this way must have been created with the {{ domxref("Document.createEvent()") }} method. This method must be called to set the event before it is dispatched, using {{ domxref("EventTarget.dispatchEvent()") }}. Once dispatched, it doesn't do anything anymore.

Do not use this method anymore, as it is deprecated.

Note: Rather than using the feature, instead use specific event constructors, like {{domxref("CustomEvent.CustomEvent", "CustomEvent()")}}. The page on Creating and triggering events gives more information about the way to use those.

Syntax

event.initCustomEvent(type, canBubble, cancelable, detail);

Parameters

type
Is a {{domxref("DOMString")}} containing the name of the event.
canBubble
Is a {{jsxref("Boolean")}} indicating whether the event bubbles up through the DOM or not.
cancelable
Is a {{jsxref("Boolean")}} indicating whether the event is cancelable.
detail
The data passed when initializing the event.
