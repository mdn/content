The paymentRequestOrigin read-only property of the {{domxref("PaymentRequestEvent")}} interface returns the origin where the {{domxref("PaymentRequest")}} object was initialized.

Syntax

var aUsvString = paymentRequestEvent.paymentRequestOrigin

Value

A USVString.
