{{SeeCompatTable}}{{APIRef("Payment Request API")}}

The instrumentKey read-only property of the {{domxref("PaymentRequestEvent")}} interface returns a {{domxref("PaymentInstrument")}} object reflecting the payment instrument selected by the user or an empty string if the user has not registered or chosen a payment instrument.

Syntax

var instrumentKey = paymentRequestEvent.instrumentKey

Value

A {{domxref("PaymentInstrument")}} object.
