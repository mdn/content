The topOrigin read-only property of the {{domxref("PaymentRequestEvent")}} interface returns the top level payee origin where the {{domxref("PaymentRequest")}} object was initialized.

Syntax

var aUsvString = paymentRequestEvent.topOrigin

Value

A USVString
