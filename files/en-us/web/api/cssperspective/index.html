The CSSPerspective interface of the {{domxref('CSS_Object_Model#css_typed_object_model','','',' ')}} represents the perspective() value of the individual {{CSSXRef('transform')}} property in CSS.  It inherits properties and methods from its parent {{domxref('CSSTransformValue')}}.
