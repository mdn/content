{{APIRef("WebRTC")}}
The {{domxref("RTCIceCandidateStats")}} dictionary's transportId property is a string that uniquely identifies the transport that produced the {{domxref("RTCTransportStats")}} from which information about this candidate was taken.

Syntax

transportID = rtcIceCandidateStats.transportId;

Value

A {{domxref("DOMString")}} whose value uniquely identifies the transport from which any transport-related information accumulated in the {{domxref("RTCIceCandidateStats")}} was taken. This can be used to compare candidates that would use the same transport, for example.
