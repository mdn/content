The {{domxref("RTCIceCandidateStats")}} interface's candidateType property is a string which indicates the type of {{Glossary("ICE")}} candidate the object represents.

Syntax

candidateType = rtcIceCandidateStats.candidateType;

Value

A {{domxref("DOMString")}} whose value is one of the strings found in RTCIceCandidate.type.
