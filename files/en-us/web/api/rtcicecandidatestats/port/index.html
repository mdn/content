The {{domxref("RTCIceCandidateStats")}} dictionary's port property specifies the network port used by the candidate.

Syntax

candidatePort = rtcIceCandidateStats.port;

Value

An integer value indicating the network port used by the {{domxref("RTCIceCandidate")}} described by the RTCIceCandidateStats object.
