The UIEvent.isChar read-only property returns a {{jsxref("Boolean")}} indicating whether the event produced a key character or not.

Syntax

var isChar = UIEvent.isChar;

Value

A Boolean which is true if the event produces a character; otherwise false. Some keystroke combinations may raise events but not produce any character (example: ctrl-alt-?). When this is the case, isChar returns false. It is used when event handlers need to do something like echo the input on the screen.

Example

In this snippet, which is part of an event handler, the event is checked to see if it generates a character; if it does, the value of {{domxref("UIEvent.which")}} is appended to a string which buffers the typed characters.

if (event.isChar) {
  characterBuffer += e.which;
}

Specifications

This property is not part of any specification.
