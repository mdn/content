The **`CSSMathNegate()`** constructor creates a
new {{domxref("CSSMathNegate")}} object which negates the value passed into it.

## Syntax

```js
var CSSMathNegate = new CSSMathNegate(arg);
```

### Parameters

- arg
  - : A {{domxref('CSSNumericValue')}}.
