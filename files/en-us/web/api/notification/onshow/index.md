The **`onshow`** property of the {{domxref("Notification")}}
interface specifies an event listener to receive {{domxref("Element/show_event",
  "show")}} events. These events occur when a {{domxref("Notification")}} is displayed.

## Syntax

```js
Notification.onshow = function() { /* ... */ };
```
