The onclose property of the {{domxref("Notification")}} interface specifies an event listener to receive {{domxref("HTMLDialogElement/close_event", "close")}} events. These events occur when a {{domxref("Notification")}} is closed.

Syntax

Notification.onclose = function() { ... };
