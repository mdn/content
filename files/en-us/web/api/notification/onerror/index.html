The onerror property of the {{domxref("Notification")}} interface specifies an event listener to receive {{domxref("HTMLElement/error_event", "error")}} events. These events occur when something goes wrong with a {{domxref("Notification")}} (in many cases an error preventing the notification from being displayed.)

Syntax

Notification.onerror = function() { ... };
