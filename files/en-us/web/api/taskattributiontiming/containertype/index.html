The containerType readonly property of the {{domxref("TaskAttributionTiming")}} interface returns the type of frame container, one of iframe, embed, or object.

Syntax

var containerType = TaskAttributionTiming.containerType;

Value

A {{domxref("DOMString")}} containing the container's type, one of iframe, embed, or object.
