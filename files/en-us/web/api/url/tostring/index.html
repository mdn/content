{{ApiRef("URL API")}}

The URL.toString() stringifier method returns a {{domxref("USVString")}} containing the whole URL. It is effectively a read-only version of {{domxref("URL.href")}}.

{{AvailableInWorkers}}

Syntax

const href = url.toString()

Return value

A {{domxref("USVString")}}.

Examples

const url = new URL("https://developer.mozilla.org/en-US/docs/Web/API/URL/toString");
url.toString(); // should return the URL as a string
