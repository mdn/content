{{APIRef("URL API")}}

The searchParams readonly property of the {{domxref("URL")}} interface returns a {{domxref("URLSearchParams")}} object allowing access to the {{httpmethod("GET")}} decoded query arguments contained in the URL.

{{AvailableInWorkers}}

Syntax

const urlSearchParams = url.searchParams

Value

A {{domxref("URLSearchParams")}} object.

Examples

If the URL of your page is https://example.com/?name=Jonathan%20Smith&age=18 you could parse out the name and age parameters using:

let params = (new URL(document.location)).searchParams;
let name = params.get('name'); // is the string "Jonathan Smith".
let age = parseInt(params.get('age')); // is the number 18
