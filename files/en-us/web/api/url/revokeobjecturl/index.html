{{ApiRef("URL API")}}

The URL.revokeObjectURL() static method releases an existing object URL which was previously created by calling {{domxref("URL.createObjectURL()") }}. Call this method when you've finished using an object URLÂ to let the browser know not to keep the reference to the file any longer.

{{AvailableInWorkers}}

Note: This method is not available from service workers, due to issues with the {{domxref("Blob")}} interface's life cycle and the potential for leaks.

Syntax

URL.revokeObjectURL(objectURL)

Parameters

objectURL
A {{domxref("DOMString")}} representing a object URL that was previously created by calling {{domxref("URL.createObjectURL", "createObjectURL()") }}.
