The NDEFWriter interface of the Web NFC API is an abstract interface used to write data to compatible NFC devices, e.g. NFC tags supporting NDEF, when these devices are within the reader's magnetic induction field.

Constructor

{{DOMxRef("NDEFWriter.NDEFWriter", "NDEFWriter.NDEFWriter()")}} {{Experimental_Inline}}
Returns an NDEFWriter.

Methods

{{DOMxRef("NDEFWriter.write", "NDEFWriter.write()")}} {{Experimental_Inline}}
Called to write NDEF message to a tag (after ensuring hardware and UA compatibility and obtaining permission from the user) or get an error explaining why feature is not available.
