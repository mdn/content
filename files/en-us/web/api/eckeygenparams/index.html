{{ APIRef("Web Crypto API") }}

The EcKeyGenParams dictionary of the Web Crypto API represents the object that should be passed as the algorithm parameter into {{domxref("SubtleCrypto.generateKey()")}}, when generating any elliptic-curve-based key pair: that is, when the algorithm is identified as either of ECDSA or ECDH.

Properties

name
A {{domxref("DOMString")}}. This should be set to ECDSA or ECDH, depending on the algorithm you want to use.
namedCurve

A {{domxref("DOMString")}} representing the name of the elliptic curve to use. This may be any of the following names for NIST-approved curves:

  • P-256
  • P-384
  • P-521

Examples

See the examples for {{domxref("SubtleCrypto.generateKey()")}}.

Browser compatibility

Browsers that support the "ECDH" or "ECDSA" algorithms for the {{domxref("SubtleCrypto.generateKey()")}} method will support this type.
