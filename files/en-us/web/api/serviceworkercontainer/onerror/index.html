{{APIRef("Service Workers API")}}{{ SeeCompatTable() }}

The onerror property of the {{domxref("ServiceWorkerContainer")}} interface is an event handler fired whenever an {{Event("error")}} event occurs in the associated service workers.

Syntax

serviceWorkerContainer.onerror = function(errorevent) { ... }

Example

navigator.serviceWorker.onerror = function(errorevent) {
  console.log(`received error message: ${errorevent.message}`);
}
