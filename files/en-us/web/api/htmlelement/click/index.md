<p>The <code><strong>HTMLElement.click()</strong></code> method simulates a mouse clickÂ on
  an element.</p>

<p>When <code>click()</code> is used with supported elements (such as an
  {{HTMLElement("input")}}), it fires the element's click event. This event then bubbles
  up to elements higher in the document tree (or event chain) and fires their click
  events.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js"><em>element</em>.click()</pre>

<h2 id="Example">Example</h2>

<p>Simulate a mouse-click when moving the mouse pointer over a checkbox:</p>

<h3 id="HTML">HTML</h3>

<pre class="brush: html">&lt;form&gt;
  &lt;input type="checkbox" id="myCheck" onmouseover="myFunction()" onclick="alert('click event occurred')"&gt;
&lt;/form&gt;</pre>

<h3 id="JavaScript">JavaScript</h3>

<pre class="brush: js">// On mouse-over, execute myFunction
function myFunction() {
  document.getElementById("myCheck").click();
}
</pre>
