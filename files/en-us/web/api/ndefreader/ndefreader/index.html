The NDEFReader() constructor of the Web NFC API returns a newly constructed {{DOMxRef("NDEFReader")}} object used to read NDEF messages from compatiable NFC devices, e.g. NDEF tags, within the reader's magnetic induction field.

Syntax

writer = new NDEFReader();

Parameters

None.

Return value

A new {{DOMxRef("NDEFReader")}}.
