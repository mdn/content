The label read-only property of the {{domxref("TrackDefault")}} interface returns the default label for an associated {{domxref("SourceBuffer")}} to use when an initialization segment does not contain label information for a new track.

Syntax

var mylabel = TrackDefault.label;

Value

A {{domxref("DOMString")}}.
