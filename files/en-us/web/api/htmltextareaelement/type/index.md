{{ApiRef("HTML DOM")}}

The **`type`** read-only property of the {{domxref("HTMLTextAreaElement")}} interface returns the string `"textarea"`.

## Value

The string `"textarea"`.

## Example

```js
const textArea = document.querySelector("textarea");
console.log(textArea.type); // "textarea"
```
