{{APIRef("DOM Events")}}

The KeyboardEvent() constructor creates a new {{domxref("KeyboardEvent")}}.

Syntax

 event = new KeyboardEvent(typeArg, KeyboardEventInit);

Values

typeArg
Is a {{domxref("DOMString")}} representing the name of the event.
KeyboardEventInit{{optional_inline}}
Is a KeyboardEventInit dictionary, having the following fields:
  • "key", optional and defaulting to "", of type {{domxref("DOMString")}}, that sets the value of {{domxref("KeyboardEvent.key")}}.
  • "code", optional and defaulting to "", of type {{domxref("DOMString")}}, that sets the value of {{domxref("KeyboardEvent.code")}}.
  • "location", optional and defaulting to 0, of type unsigned long, that sets the value of {{domxref("KeyboardEvent.location")}}.
  • "ctrlKey", optional and defaulting to false, of type {{jsxref("Boolean")}}, that sets the value of {{domxref("KeyboardEvent.ctrlKey")}}.
  • "shiftKey", optional and defaulting to false, of type {{jsxref("Boolean")}}, that sets the value of {{domxref("KeyboardEvent.shiftKey")}}.
  • "altKey", optional and defaulting to false, of type {{jsxref("Boolean")}}, that sets the value of {{domxref("KeyboardEvent.altKey")}}.
  • "metaKey", optional and defaulting to false, of type {{jsxref("Boolean")}}, that sets the value of {{domxref("KeyboardEvent.metaKey")}}.
  • "repeat", optional and defaulting to false, of type {{jsxref("Boolean")}}, that sets the value of {{domxref("KeyboardEvent.repeat")}}.
  • "isComposing", optional and defaulting to false, of type {{jsxref("Boolean")}}, that sets the value of {{domxref("KeyboardEvent.isComposing")}}.
  • "charCode", optional and defaulting to 0, of type unsigned long, that sets the value of the deprecated {{domxref("KeyboardEvent.charCode")}}.
  • "keyCode", optional and defaulting to 0, of type unsigned long, that sets the value of the deprecated {{domxref("KeyboardEvent.keyCode")}}.
  • "which", optional and defaulting to 0, of type unsigned long, that sets the value of the deprecated {{domxref("KeyboardEvent.which")}}.

The KeyboardEventInit dictionary also accepts fields from the {{domxref("UIEvent.UIEvent", "UIEventInit")}} and {{domxref("Event.Event", "EventInit")}} dictionaries.
