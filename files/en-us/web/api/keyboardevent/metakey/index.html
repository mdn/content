{{APIRef("DOM Events")}}

The KeyboardEvent.metaKey read-only property returning a {{jsxref("Boolean")}} that indicates if the Meta key was pressed (true) or not (false) when the event occurred. Some operating systems may intercept the key so it is never detected.

Note: On Macintosh keyboards, this is the ⌘ Command key.

At least as of Firefox 48, the ⊞ Windows key is no longer considered the "Meta" key. KeyboardEvent.metaKey is false when the ⊞ Windows key is pressed.

Syntax

var metaKeyPressed = instanceOfKeyboardEvent.metaKey

Return value

A {{jsxref("Boolean")}}

Example

function ismetaKey(e) {
  alert("metaKey = " + e.metaKey);
}

<button onclick="ismetaKey(event)">Click me with the meta key</button>
