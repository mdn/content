{{APIRef("DOM Events")}}

The **`repeat`** read-only property of the
{{domxref("KeyboardEvent")}} interface returns a boolean value that is
`true` if the given key is being held down such that it is automatically
repeating.

## Syntax

```js
var repeat = event.repeat;
```

### Return value

A boolean value
