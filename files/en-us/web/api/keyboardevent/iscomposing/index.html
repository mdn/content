{{APIRef("DOM Events")}}

The KeyboardEvent.isComposing read-only property returns a {{jsxref("Boolean")}} value indicating if the event is fired within a composition session, i.e. after {{domxref("Element/compositionstart_event", "compositionstart")}} and before {{domxref("Element/compositionend_event", "compositionend")}}.

Syntax

var bool = event.isComposing;

Example

var kbdEvent = new KeyboardEvent("syntheticKey", false);
console.log(kbdEvent.isComposing); // return false
