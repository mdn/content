{{APIRef("DOM Events")}}

The KeyboardEvent.ctrlKey read-only property returns a {{jsxref("Boolean")}} that indicates if the control key was pressed (true) or not (false) when the event occurred.

Syntax

var ctrlKeyPressed = instanceOfKeyboardEvent.ctrlKey

Return value

A {{jsxref("Boolean")}}

Example

<html>
<head>
<title>ctrlKey example</title>

<script type="text/javascript">

function showChar(e){
  alert(
    "Key Pressed: " + e.key + "\n"
    + "CTRL key pressed: " + e.ctrlKey + "\n"
  );
}

</script>
</head>

<body onkeypress="showChar(event);">
<p>Press any character key, with or without holding down the CTRL key.<br />
You can also use the SHIFT key together with the CTRL key.</p>
</body>
</html>
