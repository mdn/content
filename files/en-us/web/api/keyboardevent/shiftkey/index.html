{{APIRef("DOM Events")}}

The KeyboardEvent.shiftKey read-only property is a {{jsxref("Boolean")}} that indicates if the shift key was pressed (true) or not (false) when the event occurred.

Syntax

var shiftKeyPressed = instanceOfKeyboardEvent.shiftKey

Return value

A {{jsxref("Boolean")}}

Example

<html>
<head>
<title>shiftKey example</title>

<script type="text/javascript">

function showChar(e){
  alert(
    "Key Pressed: " + String.fromCharCode(e.charCode) + "\n"
    + "charCode: " + e.charCode + "\n"
    + "SHIFT key pressed: " + e.shiftKey + "\n"
    + "ALT key pressed: " + e.altKey + "\n"
  );
}

</script>
</head>

<body onkeypress="showChar(event);">
<p>Press any character key, with or without holding down
 the SHIFT key.<br />
You can also use the SHIFT key together with the ALT key.</p>
</body>
</html>
