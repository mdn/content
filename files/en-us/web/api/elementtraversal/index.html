The ElementTraversal interface used to define methods that allowed access from one {{domxref("Node")}} to another in the document tree.

It proved useless, as very few types of {{domxref("Node")}} were able to implement all its methods and properties. It has been split into two interfaces,Â containing the useful methods and properties for each kind of nodes:

  • {{domxref("ChildNode")}}
  • {{domxref("ParentNode")}}

As it was a pure interface, with no object of this type, this change has no effect on the Web.
