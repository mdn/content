The MediaRecorderErrorEvent interface represents errors returned by the MediaStream Recording API. It is an {{domxref("Event")}} object that encapsulates a reference to a {{domxref("DOMException")}} describing the error that occurred.

Properties

Inherits properties from its parent interface, {{domxref("Event")}}.

{{domxref("MediaRecorderErrorEvent.error", "error")}} {{ReadOnlyInline}}
A {{domxref("DOMException")}} containing information about the error that occurred. Read only.
