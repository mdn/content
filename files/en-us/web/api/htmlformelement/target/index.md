{{APIRef("HTML DOM")}}

The **`target`** property of the {{domxref("HTMLFormElement")}}
interface represents the target of the form's action (i.e., the frame in which to render
its output).

## Value

A string.

## Examples

```js
myForm.target = document.frames[1].name;
```
