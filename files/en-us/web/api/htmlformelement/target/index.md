<p>The <strong><code>target</code></strong> property of the {{domxref("HTMLFormElement")}}
  interface represents the target of the form's action (i.e., the frame in which to render
  its output).</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js"><em>string</em> = <em>HTMLFormElement</em>.target
<em>HTMLFormElement</em>.target = <em>string</em>
</pre>

<h2 id="Example">Example</h2>

<pre class="brush: js">myForm.target = document.frames[1].name;
</pre>
