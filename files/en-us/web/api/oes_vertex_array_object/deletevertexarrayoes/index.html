{{APIRef("WebGL")}}

The OES_vertex_array_object.deleteVertexArrayOES() method of the WebGL API deletes a given {{domxref("WebGLVertexArrayObject")}} object.

Syntax

void ext.deleteVertexArrayOES(arrayObject);

Parameters

arrayObject
A {{domxref("WebGLVertexArrayObject")}} (VAO) object to delete.

Return value

None.

Examples

var ext = gl.getExtension('OES_vertex_array_object');
var vao = ext.createVertexArrayOES();
ext.bindVertexArrayOES(vao);

// ...

ext.deleteVertexArrayOES(vao);
