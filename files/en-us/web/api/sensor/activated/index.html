{{APIRef("Generic Sensor API")}}

The activated read-only property of the {{domxref("Sensor")}} interface returns a {{jsxref("Boolean")}} indicating whether the sensor is active.

Syntax

var boolean = sensorInstance.activated

Because {{domxref('Sensor')}} is a base class, activated may only be read from one of its derived classes.

Value

A {{jsxref("Boolean")}}.
