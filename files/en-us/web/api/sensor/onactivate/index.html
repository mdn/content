{{APIRef("Generic Sensor API")}}

The onactivate {{event("Event_handlers", "event handler")}} is called when one of the Sensor interface's child interfaces becomes active.

Syntax

sensorInstance.onactivate = function
sensorInstance.addEventListener('activate', function() { ... })

Because {{domxref('Sensor')}} is a base class, onactivate may only be used on one of its derived classes.
