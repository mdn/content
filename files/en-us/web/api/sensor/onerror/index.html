{{APIRef("Sensor API")}}

The onerror event handler is called when an error occurs on one of the child interfaces of the {{domxref("Sensor")}} interface.

Syntax

sensorInstance.onerror = function
  sensorInstance.addEventListener('error', function() { ... })

Because {{domxref('Sensor')}} is a base class, onerror may only be used on one of its derived classes.
