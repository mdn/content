{{APIRef("Generic Sensor API")}}

The hasReading read-only property of the {{domxref("Sensor")}} interface returns a {{jsxref("Boolean")}} indicating whether the sensor has a reading.

Syntax

var boolean = sensorInstance.hasReading

Because {{domxref('Sensor')}} is a base class, hasReading may only be read from one of its derived classes.

Value

A {{jsxref("Boolean")}}.
