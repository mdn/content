{{APIRef("HTML DOM")}}

The HTMLPortalElement interface of the {{domxref('HTML DOM API','','',' ')}} represents the {{htmlelement("portal")}} HTML element, providing the properties and methods used to manipulate a portal.

{{InheritanceDiagram(600,120)}}

Properties

This interface also inherits properties from {{domxref("HTMLElement")}}.

{{domxref('HTMLPortalElement.referrerPolicy')}}
A {{domxref("DOMString")}} that reflects the {{htmlattrxref("referrerPolicy", "portal")}} HTML attribute indicating which referrer policy to use when fetching the linked resource.
{{domxref('HTMLPortalElement.src')}}
A {{domxref("USVString")}} that reflects the {{htmlattrxref("src", "portal")}} HTMLÂ attribute, containing the address of the content to be embedded.

Event handlers

{{domxref('HTMLPortalElement.onmessage')}}
Fired when a message is received by the object, and no exception is thrown.
{{domxref('HTMLPortalElement.onmessageerror')}}
Fired when a message is received by the object, but an exception is thrown.

Methods

This interface also inherits methods from {{domxref("HTMLElement")}}.

{{domxref('HTMLPortalElement.activate()')}}
Returns a {{jsxref('Promise')}} that resolves when the activation completes.
{{domxref('HTMLPortalElement.postMessage()')}}
Sends a message to the portaled page, which can consist of any JavaScript object. Returns {{jsxref("undefined")}}.
