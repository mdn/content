{{APIRef("HTML DOM")}}

The activate() method of the {{domxref("HTMLPortalElement")}} interface returns a {{jsxref('Promise')}} that resolves when the activation completes.

Calling activate() causes the embedding window to navigate to the content already loaded into the portal.

Syntax

var promise = HTMLPortalElement.activate(portalActivateOptions);

Parameters

portalActivateOptions {{Optional_Inline}}
An optional {{jsxref("array")}} of {{domxref("Transferable")}} objects.

Return value

A {{jsxref('Promise')}}.

Exceptions

InvalidStateError
Thrown if the {{Glossary("browsing context")}} of the document hosting the portal is null.
InvalidStateError
Thrown if the document hosting the portal has a portal state other than 'none'. This prevents portals being embedded in portals.

Examples

The promise returned by activate() will not be resolved until navigation to the URL example.com will be successful. Once the response arrives then the promise will fulfill and activation has completed.

const portal = document.createElement("portal");
portal.src = "https://example.com";
document.body.append(portal);
portal.activate();
