{{APIRef("HTML DOM")}}

The src property of the {{domxref("HTMLPortalElement")}} interface reflects the {{htmlattrxref("src", "portal")}} HTML attribute. This contains the address of the content to be embedded.

Syntax

var src = HTMLPortalElement.src;
HTMLPortalElement.src = src

Value

A {{domxref("USVString")}} which is a valid, non-empty URL.

Examples

const portal = document.createElement('portal');
portal.src = 'https://example.com' // The page to navigate to
