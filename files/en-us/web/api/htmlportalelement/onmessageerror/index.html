The onmessageerror EventHandler of the {{domxref("HTMLPortalElement")}} interface is anEvent Handler, that is a function to be called when a message is received.

The event fires when the embedding page sends a message to the portaled content, and an exception is thrown.

Syntax

HTMLPortalElement.onmessageerror = function() { ... };
