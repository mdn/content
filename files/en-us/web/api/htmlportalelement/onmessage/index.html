The onmessage EventHandler of the {{domxref("HTMLPortalElement")}} interface is an Event Handler, that is a function to be called when a message is received.

The event fires when the embedding page sends a message to the portaled content, and no exception is thrown.

Syntax

HTMLPortalElement.onmessage = function() { ... };
