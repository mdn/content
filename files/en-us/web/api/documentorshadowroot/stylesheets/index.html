The styleSheets read-only property of the {{domxref("DocumentOrShadowRoot")}} interface returns a {{domxref('StyleSheetList')}} of {{domxref('CSSStyleSheet')}} objects, for stylesheets explicitly linked into or embedded in a document.

Examples

function getStyleSheet(unique_title) {
  for (var i=0; i<document.styleSheets.length; i++) {
    var sheet = document.styleSheets[i];
    if (sheet.title == unique_title) {
      return sheet;
    }
  }
}

Notes

The returned list is ordered as follows:

  • StyleSheets retrieved from {{htmlelement("link")}}Â headers are placed first, sorted in header order.
  • StyleSheets retrieved from the DOM are placed after, sorted in tree order.
