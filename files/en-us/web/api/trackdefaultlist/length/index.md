The **`length`** read-only property of the {{domxref("TrackDefaultList")}} interface returns the number of {{domxref("TrackDefault")}} objects in the list.

## Syntax

```js
var myListLength = TrackDefaultList.length;
```

### Value

An unsigned long number.
