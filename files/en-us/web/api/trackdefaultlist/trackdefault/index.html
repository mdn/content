The TrackDefault() getter method of the {{domxref("TrackDefaultList")}} interface allows the TrackDefault objects in the list to be accessed with an array operator (i.e. [].)

Syntax

var myTrackDefault = TrackDefaultList[index];

Parameters

index
The index position of the {{domxref("TrackDefault")}} object you want to return.

Returns

A {{domxref("TrackDefault")}} object.

Errors

No specific errors are returned, but if the supplied index is great than or equal to {{domxref("TrackDefaultList.length")}}, the operation will return undefined.
