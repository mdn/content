{{APIRef("Web Audio API")}}

The StereoPannerNode() constructor of the Web Audio API creates a new {{domxref("StereoPannerNode")}} object which is an {{domxref("AudioNode")}} that represents a simple stereo panner node that can be used to pan an audio stream left or right.

Syntax

var stereoPannerNode = StereoPannerNode(context, options)

Parameters

Inherits parameters from the {{domxref("AudioNodeOptions")}} dictionary.

context
A reference to an {{domxref("AudioContext")}}.
options {{optional_inline}}
Options are as follows:
  • pan: A floating point number in the range [-1,1] indicating the position of an {{domxref("AudioNode")}} in an output image. The value -1 represents full left and 1 represents full right. The default value is 0.

Return value

A new {{domxref("StereoPannerNode")}} object instance.
