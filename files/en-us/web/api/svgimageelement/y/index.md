{{APIRef("SVG")}}

The **`y`** read-only property of the
{{domxref("SVGImageElement")}} interface returns an {{domxref("SVGAnimatedLength")}}
corresponding to the {{SVGAttr("y")}} attribute of the given {{SVGElement("image")}}
element.

## Value

An {{domxref("SVGAnimatedLength")}}.
