{{APIRef("Encrypted Media Extensions")}}

The onmessage property of the {{domxref("MediaKeySession")}} is an event handler, fired whenever a {{domxref("MediaKeyMessageEvent")}} occurs, denoting a message is generated by the content decryption module.

This interface inherits from the {{domxref("ExtendableEvent")}} interface.

{{InheritanceDiagram(700, 60, 20)}}

Syntax

MediaKeySession.onmessage = function(MediaKeyMessageEvent) { ... }
