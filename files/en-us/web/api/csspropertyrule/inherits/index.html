The read-only inherits property of the {{domxref("CSSPropertyRule")}} interface returns the inherit flag of the custom property registration represented by the {{domxref("@property")}} rule, a boolean describing whether or not the property inherits by default.

Syntax

const inherits = CSSPropertyRule.inherits;

Value

A boolean.
