{{APIRef("SVG")}}

The **`SVGGeometryElement.pathLength`** property reflects the
{{SVGAttr("pathLength")}} attribute and returns the total length of the path, in user
units.

## Syntax

```js
var pathLength = someElement.pathLength;
```
