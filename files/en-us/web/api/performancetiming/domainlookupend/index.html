{{APIRef("Navigation Timing")}}

This interface of this property is deprecated in the Navigation Timing Level 2 specification. Please use the {{domxref("PerformanceNavigationTiming")}} interface instead.

The legacy PerformanceTiming.domainLookupEnd read-only property returns an unsigned long long representing the moment, in milliseconds since the UNIX epoch, where the domain lookup is finished. If a persistent connection is used, or the information is stored in a cache or a local resource, the value will be the same as {{domxref("PerformanceTiming.fetchStart")}}.

Syntax

time = performanceTiming.domainLookupEnd;
