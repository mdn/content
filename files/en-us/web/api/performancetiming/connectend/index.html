{{APIRef("Navigation Timing")}}{{Deprecated_Header}}

Warning

This interface of this property is deprecated in the Navigation Timing Level 2 specification. Please use the {{domxref("PerformanceNavigationTiming")}} interface instead.

The legacy PerformanceTiming.connectEnd read-only property returns an unsigned long long representing the moment, in milliseconds since the UNIX epoch, where the connection is opened network. If the transport layer reports an error and the connection establishment is started again, the last connection establisment end time is given. If a persistent connection is used, the value will be the same as {{domxref("PerformanceTiming.fetchStart")}}. A connection is considered as opened when all secure connection handshake, or SOCKS authentication, is terminated.

Syntax

time = performanceTiming.connectEnd;
