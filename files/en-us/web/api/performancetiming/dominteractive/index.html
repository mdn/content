{{APIRef("Navigation Timing")}}{{Deprecated_Header}}

Warning

This interface of this property is deprecated in the Navigation Timing Level 2 specification. Please use the {{domxref("PerformanceNavigationTiming")}} interface instead.

The legacy PerformanceTiming.domInteractive read-only property returns an unsigned long long representing the moment, in milliseconds since the UNIX epoch, when the parser finished its work on the main document, that is when its {{domxref("Document.readyState")}} changes to 'interactive' and the corresponding {{event("readystatechange")}} event is thrown.

This property can be used to measure the speed of loading Web sites that users feels. Nevertheless there are a few caveats that happens if scripts are blocking rendering and not loaded asynchronously or with custom Web fonts. Check if you are in one of these cases before using this property as a proxy for the user experience of a Web site's speed of loading.

Syntax

time = performanceTiming.domInteractive;
