{{APIRef("Navigation Timing")}}

This interface of this property is deprecated in the Navigation Timing Level 2 specification. Please use the {{domxref("PerformanceNavigationTiming")}} interface instead.

The legacy PerformanceTiming.navigationStart read-only property returns an unsigned long long representing the moment, in milliseconds since the UNIX epoch, right after the prompt for unload terminates on the previous document in the same browsing context. If there is no previous document, this value will be the same as {{domxref("PerformanceTiming.fetchStart")}}.

Syntax

time = performanceTiming.navigationStart;
