This interface of this property is deprecated in the Navigation Timing Level 2 specification. Please use the {{domxref("PerformanceNavigationTiming")}} interface'sÂ {{domxref("PerformanceNavigationTiming.loadEventStart")}} read-only property instead.

The legacy PerformanceTiming.loadEventStart read-only property returns an unsigned long long representing the moment, in milliseconds since the UNIX epoch, when the {{event("load")}} event was sent for the current document. If this event has not yet been sent, it returns 0.

Syntax

time = performanceTiming.loadEventStart;

Specifications

This feature is no longer on track to become a standard, as the Navigation Timing specification has marked it as deprecated. Use the {{domxref("PerformanceNavigationTiming")}} interface instead.
