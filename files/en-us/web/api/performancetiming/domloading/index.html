{{APIRef("Navigation Timing")}}

This interface of this property is deprecated in the Navigation Timing Level 2 specification. Please use the {{domxref("PerformanceNavigationTiming")}} interface instead.

The legacy PerformanceTiming.domLoading read-only property returns an unsigned long long representing the moment, in milliseconds since the UNIX epoch, when the parser started its work, that is when its {{domxref("Document.readyState")}} changes to 'loading' and the corresponding {{event("readystatechange")}} event is thrown.

Syntax

time = performanceTiming.domLoading;
