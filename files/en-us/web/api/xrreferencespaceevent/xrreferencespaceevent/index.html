The XRReferenceSpaceEvent() constructor is used to create a new {{domxref("XRReferenceSpaceEvent")}} object, which represents an event regarding the state of a WebXR reference space object, {{domxref("XRReferenceSpace")}}.

Currently, only the {{domxref("XRReferenceSpace.reset_event", "reset")}} event is defined using this type.

Syntax

let refSpaceEvent = new XRReferenceSpaceEvent(type, eventInitDict);

Parameters

type
A {{domxref("DOMString")}} indicating the event type which has occurred. Currently, this is always reset.
eventInitDict

An object based on the {{domxref("XRReferenceSpaceEventInit")}} dictionary, containing data used to initialize the new event object. The properties of this object are:

{{page("/en-US/docs/Web/API/XRReferenceSpaceEventInit", "Properties")}}

Return value

A new XRReferenceSpaceEvent object, initialized as defined by the input parameters.
