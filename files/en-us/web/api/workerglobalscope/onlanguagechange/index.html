{{APIRef("Web Workers API")}}

The onlanguagechange property of the {{domxref("WorkerGlobalScope")}} interface represents an {{domxref("EventHandler")}} to be called when the {{event("languagechange")}} event occurs and bubbles through the {{domxref("Worker")}}.

Syntax

self.onlanguagechange = function() { ... };

Example

The following code snippet shows an onlanguagechange handler set inside a worker:

self.onlanguagechange = function() {
  console.log('Your preferred language settings have been changed');
}
