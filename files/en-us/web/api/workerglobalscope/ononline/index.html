{{APIRef("Web Workers API")}}

The ononline property of the {{domxref("WorkerGlobalScope")}} interface represents an {{event("Event_handlers", "event handler")}} to be called when the {{event("online")}} event occurs.

Syntax

self.ononline = function() { ... };

Example

The following code snippet shows an ononline handler set inside a worker:

self.ononline = function() {
  console.log('Your worker is now online');
}
