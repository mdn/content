{{APIRef("Web Workers API")}}

The onoffline property of the {{domxref("WorkerGlobalScope")}} interface represents an {{domxref("EventHandler")}} to be called when the {{event("offline")}} event occurs and bubbles through the {{domxref("Worker")}}.

Syntax

self.onoffline = function() { ... };

Example

The following code snippet shows an onoffline handler set inside a worker:

self.onoffline = function() {
  console.log('Your worker is now offline');
}
