{{APIRef("Service Workers API")}}{{Deprecated_Header}}

The FetchEvent.client read-only property returns the {{domxref("Client")}} that the current service worker is controlling.

Note: This feature has been deprecated, with its functionality replaced by {{domxref("FetchEvent.clientId")}} and {{domxref("Clients.get()")}}. See this Github issue for discussion/background.

Syntax

var myClient = FetchEvent.client;

Value

A {{domxref("Client")}} object.

Example

self.addEventListener('fetch', function(event) {
  console.log(event.client);
});

Browser compatibility
