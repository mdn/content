{{APIRef("HTML Drag and Drop API")}}

{{ Non-standard_header() }}

The DataTransfer.mozUserCancelled property is used in the {{event("dragend")}} event handler to determine if the user canceled the drag or not. If the user canceled the event, the property returns true and returns false otherwise. This property only applies to the {{event("dragend")}} event.

Note: This property is Gecko-specific.

This property is {{readonlyInline}}.

Syntax

dataTransfer.mozUserCancelled;

Return value

A {{jsxref("Boolean")}} representing true if the user canceled the drag event and returns false otherwise.

Example

This example shows the use of the mozUserCancelled property in the {{event("dragend")}} event handler.

function dragend_handler(event)
{
  var dragData = event.dataTransfer;
  console.log("mozUserCancelled = " + dragData.mozUserCancelled);
}

Specifications

This property is not defined in any Web standard.
