<p>The <strong><code>objectStoreNames</code></strong> read-only property of the
  {{domxref("IDBTransaction")}} interface returns a {{domxref("DOMStringList")}} of names
  of {{domxref("IDBObjectStore")}} objects.</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js">var <em>myDatabase</em> = <em>transactionObj</em>.objectStoreNames;</pre>

<h3 id="Returns">Returns</h3>

<p>A {{domxref("DOMStringList")}} of names of
  {{domxref("IDBObjectStore")}} objects.</p>
