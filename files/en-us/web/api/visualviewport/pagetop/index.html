{{SeeCompatTable}}{{APIRef("Visual Viewport")}}

The pageTop read-only property of the {{domxref("VisualViewport")}} interface returns the y coordinate of the top edge of the visual viewport relative to the initial containing block origin, in CSS pixels.

Syntax

var pageTop = VisualViewport.pageTop

Value

A double.
