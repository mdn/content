{{APIRef("Visual Viewport")}}

The **`height`** read-only property of the
{{domxref("VisualViewport")}} interface returns the height of the visual viewport, in
CSS pixels.

## Syntax

```js
var height = VisualViewport.height
```

### Value

A double.
