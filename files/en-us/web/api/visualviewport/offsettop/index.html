{{SeeCompatTable}}{{APIRef("Visual Viewport")}}

The offsetTop read-only property of the {{domxref("VisualViewport")}} interface returns the offset of the top edge of the visual viewport from the top edge of the layout viewport in CSS pixels.

Syntax

var offsetTop = VisualViewport.offsetTop

Value

A double.
