The **`onresize`** event handler of
the {{domxref("VisualViewport")}} interface is called when a viewport is resized, i.e.
when the
[`resize`](/en-US/docs/Web/API/VisualViewport/resize_event) event
is fired.

## Syntax

```js
VisualViewport.onresize = function(e) { ... }
```

## Examples

    VisualViewport.onresize = function(e) {
      ...
    }
