<p>The <strong><code>onresize</code></strong> event handler of
  the {{domxref("VisualViewport")}} interface is called when a viewport is resized, i.e.
  when the
  <code><a href="/en-US/docs/Web/API/VisualViewport/resize_event">resize</a></code> event
  is fired.</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js">VisualViewport.onresize = function(e) { ... }</pre>

<h2 id="Examples">Examples</h2>

<pre>VisualViewport.onresize = function(e) {
  ...
}</pre>
