The onresize event handler of the {{domxref("VisualViewport")}} interface is called when a viewport is resized, i.e. when the resize event is fired.

Syntax

VisualViewport.onresize = function(e) { ... }

Examples

VisualViewport.onresize = function(e) {
  ...
}
