{{APIRef("WebRTC")}}

The WebRTC API's RTCIceComponent enumerated type contains {{domxref("DOMString")}} values that each identify a specific ICE component; these are "rtp" and "rtcp". These strings are mapped to corresponding numeric values as they appear in the candidate a-line in SDP.

Values

"rtp"
Identifies an ICE transport which is being used for the Real-time Transport Protocol (RTP), or for RTP multiplexed with the RTP Control Protocol (RTCP). RTP is defined in {{RFC(3550)}}. An RTCIceComponent of this value corresponds to the component ID field in the candidate a-line with the value 1.
"rtcp"
Identifies an ICE transport being used for RTCP, which is defined in {{RFC(3550, "", 6)}}. This value of RTCIceComponent corresponds to the component ID 2.

Usage notes

The RTCIceComponent type is used by the {{domxref("RTCIceCandidate.component", "component")}} property of {{domxref("RTCIceCandidate")}} and {{domxref("RTCIceTransport")}} objects.
