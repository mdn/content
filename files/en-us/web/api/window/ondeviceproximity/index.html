The ondeviceproximity property of the {{domxref("Window")}} interface specifies an {{domxref("EventHandler")}} to receive {{event("deviceproximity")}} events. These events occur when the device sensor detects that an object becomes closer to or farther from the device.

Syntax

window.onuserproximity = funcRef

Where funcRef is a function to be called when the {{event("deviceproximity")}} event occurs. These events are of type {{domxref("DeviceProximityEvent")}}.
