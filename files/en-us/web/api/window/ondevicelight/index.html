{{deprecated_header}}

Specifies an event listener to receive {{event("devicelight")}} events. These events occur when the device's light level sensor detects a change in the intensity of the ambient light level.

Syntax

window.ondevicelight = funcRef

Where funcRef is a function to be called when the {{event("devicelight")}} event occurs. These events are of type {{domxref("DeviceLightEvent")}}.
