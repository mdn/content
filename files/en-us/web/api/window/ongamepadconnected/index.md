<p>The <strong><code>ongamepadconnected</code></strong> property of the
  {{domxref("Window")}} interface represents an event handler that will run when a gamepad
  is connected (when the {{event('gamepadconnected')}} event fires).</p>

<p>The event object is of type {{domxref("GamepadEvent")}}.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js">window.ongamepadconnected = function() { ... };
</pre>

<h2 id="Examples">Examples</h2>

<pre
  class="brush:js; ">window.ongamepadconnected = function(event) {
  // All buttons and axes values can be accessed through
  event.gamepad;
};</pre>
