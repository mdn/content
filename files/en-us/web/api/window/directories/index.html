Summary

Returned the window personalbar toolbar object. Use the {{ domxref("window.personalbar") }} instead.

Syntax

var dirBar = window.directories;

Parameters

dirBar is an object of the type BarProp.

Example

<script>
 function dirs() {
  alert(window.directories);
 }
</script>

Specifications

Not part of specification.
