The onvrdisplayfocus property of the {{domxref("Window")}} interface represents an event handler that will run when presentation to a display has resumed after being blurred (when the {{event("vrdisplayfocus")}} event fires).

The event object is of type {{domxref("VRDisplayEvent")}}.

Syntax

window.onvrdisplayfocus = function() { ... };

Examples

window.onvrdisplayfocus = function() {
  info.textContent = 'Display focused.';
};
