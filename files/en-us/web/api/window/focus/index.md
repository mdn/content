<p>{{APIRef}}</p>

<p>Makes a request to bring the window to the front. It may fail due to user settings and the window isn't guaranteed to be frontmost before this method returns.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="eval">window.focus()
</pre>

<h2 id="Example">Example</h2>

<pre class="eval">if (clicked) { window.focus(); }
</pre>
