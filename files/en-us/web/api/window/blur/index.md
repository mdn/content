{{APIRef}}

Shifts focus away from the window.

## Syntax

```js-nolint
blur()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).

## Examples

```js
window.blur();
```

## Notes

The window\.blur() method is the programmatic equivalent of the user shifting focus away
from the current window.
