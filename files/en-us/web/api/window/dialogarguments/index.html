{{Deprecated_Header}}{{APIRef}}

The dialogArguments property returns the parameters that were passed into the {{domxref("window.showModalDialog()")}} method. This lets you determine what parameters were specified when the modal dialog was created.

Syntax

value = window.dialogArguments;
