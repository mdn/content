The onvrdisplayactivate property of the {{domxref("Window")}} interface represents an event handler that will run when a display is able to be presented to (when the {{event("vrdisplayactivate")}} event fires), for example if an HMD has been moved to bring it out of standby, or woken up by being put on.

The event object is of type {{domxref("VRDisplayEvent")}}.

Syntax

window.onvrdisplayactivate = function() { ... };

Examples

window.onvrdisplayactivate = function() {
  info.textContent = 'Display activated.';
  reportDisplays();
};
