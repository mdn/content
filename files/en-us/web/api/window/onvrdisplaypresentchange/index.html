The onvrdisplaypresentchange property of the {{domxref("Window")}} interface represents an event handler that will run when the presenting state of a VR display changes — i.e. goes from presenting to not presenting, or vice versa (when the {{event("vrdisplaypresentchange")}} event fires).

The event object is of type {{domxref("VRDisplayEvent")}}.

Syntax

window.onvrdisplaypresentchange = functionRef;

Examples

The presenting state of a {{domxref("VRDisplay")}} can be checked using the {{domxref("VRDisplay.isPresenting")}} property.

window.onvrdisplaypresentchange = function() {
  if(vrDisplay.isPresenting) {
    info.textContent = 'Display has started presenting.';
  } else {
    info.textContent = 'Display has stopped presenting.';
  }
};
