{{APIRef}}{{Non-standard_header}}{{deprecated_header}}

The **`Window.home()`** method returns the window to the home
page.

## Syntax

```js
window.home();
```

## Example

```js
function goHome() {
  window.home();
}
```

## Specifications

DOM Level 0. Not part of specification.
