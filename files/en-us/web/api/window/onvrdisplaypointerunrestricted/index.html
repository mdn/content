The onvrdisplaypointerunrestricted property of the {{domxref("Window")}} interface represents an event handler that will run when the VR display's pointer input is no longer restricted to consumption via a pointerlocked element.

The event object is of type {{domxref("VRDisplayEvent")}}.

Syntax

window.onvrdisplaypointerunrestricted = functionRef;

Examples

window.onvrdisplaypointerunrestricted = function() {
  // Run code to handle freeing app from pointer lock
};
