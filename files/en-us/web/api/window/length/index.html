Returns the number of frames (either {{HTMLElement("frame")}} or {{HTMLElement("iframe")}} elements) in the window.

Syntax

framesCount = window.length;
  • framesCount is the number of frames.

Example

if (window.length) {
  // this is a document with subframes
}
