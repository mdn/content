{{APIRef("DOM")}}

The **`isSecureContext`** read-only property of the {{domxref("Window")}} interface returns a boolean indicating whether the current [context is secure](/en-US/docs/Web/Security/Secure_Contexts) (`true`) or not (`false`).

## Value

A boolean.
