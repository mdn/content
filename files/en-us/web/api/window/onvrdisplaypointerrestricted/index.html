The onvrdisplaypointerrestricted property of the {{domxref("Window")}} interface represents an event handler that will run when the VR display's pointer input is restricted to consumption via a pointerlocked element.

The event object is of type {{domxref("VRDisplayEvent")}}.

Syntax

window.onvrdisplaypointerrestricted = functionRef;

Examples

window.onvrdisplaypointerrestricted = function() {
  // Run code to handle pointer lock
};
