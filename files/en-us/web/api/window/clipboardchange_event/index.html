{{APIRef}}{{seecompattable}}

The clipboardchange event fires when the system clipboard content changes.

Bubbles No
Cancelable No
Interface {{DOMxRef("ClipboardEvent")}}
Event handler property None

Examples

JavaScript

window.addEventListener('clipboardchange', () => {
  console.log('Clipboard contents changed');
});
