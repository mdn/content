The Window.sizeToContent() method sizes the window according to its content. In order for it to work, the DOM content should be loaded when this function is calledâ€”for example, once the {{event("DOMContentLoaded")}} event has been thrown.

Since Firefox 20, the mimimal size of the window is clamped to prevent the window from being too small for the user to interact with.

Syntax

window.sizeToContent()

Example

window.sizeToContent();

This feature is not part of any specification.
