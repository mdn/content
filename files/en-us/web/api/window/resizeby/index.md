<p>The <code><strong>Window.resizeBy()</strong></code> method resizes the current window
  by a specified amount.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js">window.resizeBy(<em>xDelta</em>, <em>yDelta</em>)
</pre>

<h3 id="Parameters">Parameters</h3>

<ul>
  <li><code>xDelta</code> is the number of pixels to grow the window horizontally.</li>
  <li><code>yDelta</code> is the number of pixels to grow the window vertically.</li>
</ul>

<h2 id="Example">Example</h2>

<pre class="brush:js">// Shrink the window
window.resizeBy(-200, -200);
</pre>

<h2 id="Notes">Notes</h2>

<p>This method resizes the window relative to its current size. To resize the window in
  absolute terms, use {{domxref("window.resizeTo()")}}.</p>

<h3 id="Creating_and_resizing_an_external_window">Creating and resizing an external window
</h3>


<p>For security reasons, it's no longer possible in Firefox for a website to change the
  default size of a window in a browser if the window wasnâ€™t created by
  <code>window.open()</code>, or contains more than one tab. See the compatibility table
  for details on the change.</p>

<p>Even if you create window by <code>window.open()</code> <strong>it is not resizable by
    default.</strong> To make the window resizable, you must open it with the
  <code>"resizable"</code> feature.</p>


<pre class="brush: js">// Create resizable window
myExternalWindow = window.open("http://myurl.domain", "myWindowName", "resizable");

// Resize window to 500x500
myExternalWindow.resizeTo(500, 500);

// Make window relatively smaller to 400x400
myExternalWindow.resizeBy(-100, -100);</pre>

<p>The window you create must respect the Same Origin Policy. If the window you open is
  not in the same orgin as the current window, you will not be able to resize, or access
  any information on, that window/tab.</p>
