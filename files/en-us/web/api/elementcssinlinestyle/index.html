{{APIRef("CSSOM")}}

The ElementCSSInlineStyle mixin describes CSSOM-specific features common to the {{DOMxRef("HTMLElement")}}, {{DOMxRef("SVGElement")}} and {{DOMxRef("MathMLElement")}} interfaces. Each of these interfaces can, of course, add more features in addition to the ones listed below.

Note: ElementCSSInlineStyle is a mixin and not an interface; you can't actually create an object of type ElementCSSInlineStyle.
