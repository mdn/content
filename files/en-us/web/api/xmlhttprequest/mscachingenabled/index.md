The **`msCachingEnabled()`** method gets the current caching state for an [XMLHttpRequest](/en-US/docs/Web/API/XMLHttpRequest).

## Syntax

```js
msCachingEnabled()
```

### Parameters

None.

### Return value

A boolean value. If `true`, the `XMLHttpRequest` is cached to disk. If `false`, it is not written to disk.

## Specifications

This feature is not part of any current specification. It is no longer on track to become a standard.

## Browser compatibility

This proprietary method is specific to Internet Explorer.
