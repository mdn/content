The onstatechange EventHandler of the {{domxref("MIDIPort")}} interface is an {{domxref("EventHandler")}} that processes state change events.

The null event fires when a port changes from open to closed, or closed to open.

Syntax

MIDIPort.onstatechange = function;
MIDIPort.addEventListener('statechange', function);

Example

In the following example the current {{domxref("MIDIPort.state")}} is logged each time it changes.

port.onstatechange = function() {
  console.log(port.state);
}
