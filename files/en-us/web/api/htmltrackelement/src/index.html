{{APIRef("HTML DOM")}}

The HTMLTrackElement.src property reflects the value of the {{HTMLElement("track")}} element's {{htmlattrxref("src", "track")}} attribute, which indicates the URL of the text track's data.

Syntax

var textTrackURL = HTMLTrackElement.src;

Value

A {{domxref("DOMString")}} object containing the URL of the text track data.
