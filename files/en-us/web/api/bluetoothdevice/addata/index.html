The BluetoothDevice.adData read-only property returns instance of {{DOMxRef("BluetoothAdvertisingData")}} containing the most recent advertising data received for the device.

Syntax

var instanceOfBluetoothAdData = instanceOfBluetoothDevice.adData

Returns

An instance of {{DOMxRef("BluetoothAdvertisingData")}}.
