The BluetoothDevice.name read-only property returns a {{DOMxRef("DOMString")}} that provides a human-readable name for the device.

This page describes the W3C Community Group BluetoothDevice.name property. For the Firefox OS property of the same name, see {{DOMxRef("BluetoothDevice_%28Firefox_OS%29/name", "BluetoothDevice.name (Firefox OS)")}}.

Syntax

var name = instanceOfBluetoothDevice.name

Returns

A {{DOMxRef("DOMString")}}.
