{{APIRef("XDomain")}}{{deprecated_header}}{{non-standard_header}}

Returns the response body of an {{domxref("XDomainRequest")}} as a string.

Note: This property is valid during the {{domxref("XDomainRequest.onprogress")}} and {{domxref("XDomainRequest.onload")}} events.

Syntax

var response = xdr.responseText;

This sets response to contain the response body of the request, as a string.

Example

var response = xdr.responseText;

Specifications

Not part of any specification.
