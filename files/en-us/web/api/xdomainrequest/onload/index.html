An event handler for when an {{domxref("XDomainRequest")}} has finished receiving the response from the server. You can find the entire server response in the {{domxref("XDomainRequest.responseText")}} property.

Syntax

xdr.onload = funcRef;

Example

var xdr = new XDomainRequest();
xdr.open("POST", "http://example.com/api/method");

xdr.onload = function(){
  //Handle Response with xdr.responseText
}

xdr.send("param1=value1&param2=value2");

Specifications

Not part of any specification.
