An event handler which is called when an {{domxref("XDomainRequest")}} encounters an error.

Note: There is no way to determine the cause of the error from the {{domxref("XDomainRequest")}} interface.

Syntax

xdr.onerror = funcRef;

Parameters

funcRef
A function to call when an error occurs.

Example

var xdr = new XDomainRequest();
xdr.open("POST", "http://example.com/api/method");

xdr.onerror = function(){
  //Handle Error
}

xdr.onload = function(){
  //Handle Response with xdr.responseText
}

xdr.send("param1=value1&param2=value2");

Specifications

Not part of any specification.
