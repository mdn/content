This method is called periodically as an event handler for progress events on {{domxref("XDomainRequest")}}s, so that code can monitor progress while loading content.

Note: While handling this event, you can look at {{domxref("XDomainRequest.responseText")}} to get the response so far.

Once loading is complete, the {{domxref("XDomainRequest.onload")}} event handler gets called.

Syntax

xdr.onprogress = funcRef;

Parameters

funcRef
A function to call when progress events occur.

Example

var xdr = new XDomainRequest();
xdr.open("POST", "http://example.com/api/method");

xdr.onprogress = function(){
  //Handle Partial Response with xdr.responseText
}

xdr.onload = function(){
  //Handle Response with xdr.responseText
}

xdr.send("param1=value1&param2=value2");
