<p>The <strong><code>HTMLTableColElement</code></strong> interface provides special properties (beyond the {{domxref("HTMLElement")}} interface it also has available to it inheritance) for manipulating single or grouped table column elements.</p>

<p>{{InheritanceDiagram(600, 120)}}</p>

<h2 id="Properties">Properties</h2>

<p><em>Inherits properties from its parent, {{domxref("HTMLElement")}}</em>.</p>

<dl>
 <dt>{{domxref("HTMLTableColElement.align")}} {{deprecated_inline}}</dt>
 <dd>Is a {{domxref("DOMString")}} that indicates the horizontal alignment of the cell data in the column.</dd>

 <dt>{{domxref("HTMLTableColElement.ch")}} {{deprecated_inline}}</dt>
 <dd>Is a {{domxref("DOMString")}} representing the alignment character for cell data.</dd>

 <dt>{{domxref("HTMLTableColElement.chOff")}} {{deprecated_inline}}</dt>
 <dd>Is a {{domxref("DOMString")}} representing the offset for the alignment character.</dd>

 <dt>{{domxref("HTMLTableColElement.span")}}</dt>
 <dd>Is an <code>unsigned long</code> that reflects the {{htmlattrxref("span", "col")}} HTMLÂ attribute, indicating the number of columns to apply this object's attributes to. Must be a positive integer.</dd>

 <dt>{{domxref("HTMLTableColElement.vAlign")}} {{deprecated_inline}}</dt>
 <dd>Is a {{domxref("DOMString")}} that indicates the vertical alignment of the cell data in the column.</dd>

 <dt>{{domxref("HTMLTableColElement.width")}} {{deprecated_inline}}</dt>
 <dd>Is a {{domxref("DOMString")}} representing the default column width.</dd>
</dl>

<h2 id="Methods">Methods</h2>

<p><em>No specific method; inherits methods from its parent, {{domxref("HTMLElement")}}</em>.</p>
