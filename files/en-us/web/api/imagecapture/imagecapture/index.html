{{APIRef("MediaStream Image")}}

The ImageCapture() constructor creates a new {{domxref("ImageCapture")}} object.

Syntax

const imageCapture = new ImageCapture(videoTrack)

Parameters

videoTrack
A {{domxref("MediaStreamTrack")}} from which the still images will be taken. This can be any source, such as an incoming stream of a video conference, a playing movie, or the stream from a webcam.

Return value

A new ImageCapture object which can be used to capture still frames from the specified video track.

Example

The following example shows how to use a call to {{domxref("MediaDevices.getUserMedia()")}} to retrieve the {{domxref("MediaStreamTrack")}} needed by the ImageCapture() constructor. 

 navigator.mediaDevices.getUserMedia({video: true})
  .then(mediaStream => {
    document.querySelector('video').srcObject = mediaStream
    const track = mediaStream.getVideoTracks()[0];
    imageCapture = new ImageCapture(track);
  })
  .catch(error => console.log(error));
