The hasPosition read-only property of the {{domxref("VRPositionState")}} interface returns a boolean indicating whether the {{domxref("VRPositionState.position")}} property is valid (i.e. if the hardware is currently registering a valid position). If it is false, the position property will return null.

Syntax

var myHasPosition = VRPositionState.hasPosition;

Value

A {{domxref("Boolean")}}.
