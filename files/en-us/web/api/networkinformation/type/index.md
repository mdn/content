The **`NetworkInformation.type`** read-only property returns
the type of connection a device is using to communicate with the network.

## Value

An enumerated value that is one of the following values:

- `"bluetooth"`
- `"cellular"`
- `"ethernet"`
- `"none"`
- `"wifi"`
- `"wimax"`
- `"other"`
- `"unknown"`
