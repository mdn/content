{{apiref("Network Information API")}}{{SeeCompatTable}}

The NetworkInformation.onchange event handler contains the code that is fired when connection information changes, and the {{event("change")}} is received by the {{domxref("NetworkInformation")}} object.

Syntax

netInfo.onchange = function() { ... }

Examples

// Get the connection type.
var type = navigator.connection.type;

function changeHandler(e) {
   // Handle change of connection type here.
}

// Register for event changes:
navigator.connection.onchange = changeHandler;

// Another way: navigator.connection.addEventListener('change', changeHandler);
