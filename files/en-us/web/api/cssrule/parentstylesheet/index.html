{{ APIRef("CSSOM") }}

The parentStyleSheet property of the {{domxref("CSSRule")}} interface returns the {{domxref("StyleSheet")}} object in which the current rule is defined.

Syntax

var stylesheet = cssRule.parentStyleSheet

Values

stylesheet
A {{domxref("StyleSheet")}} object.

Example

let myRules = document.styleSheets[0].cssRules;
console.log(myRules.parentStyleSheet);
