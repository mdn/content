<h2 id="Summary">Summary</h2>

<p>The <code>truncate</code> method is used to remove some data within the file.</p>

<p>If the method is called with no argument, the operation removes all the bytes starting
  at the index set in {{domxref("LockedFile.location")}}.</p>

<p>If the method is called with an argument, the operation removes all the bytes starting
  at the index corresponding to the parameter and regardless of the value of
  {{domxref("LockedFile.location")}}.</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js">var request = <em>instanceOfLockedFile</em>.truncate(start);</pre>

<h3 id="Parameters">Parameters</h3>

<dl>
  <dt><code>start</code> {{optional_inline}}</dt>
  <dd>A number representing the index where to start the operation.</dd>
</dl>

<h3 id="Return">Return</h3>

<p>A {{domxref("FileRequest")}} object to handle the success or failure of the operation.
</p>
