{{APIRef("File System API")}}{{non-standard_header}}

Summary

The write method is used to write some data within the file.

The write starts at the position set in {{domxref("LockedFile.location")}} and moves that position by the number of written bytes.

Syntax

var request = instanceOfLockedFile.write(data);

Parameters

data
The data to write into the file. It can be a string or an {{domxref("ArrayBuffer")}}.

Return

A {{domxref("FileRequest")}} object to handle the success or failure of the operation.
