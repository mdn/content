{{SeeCompatTable}}{{APIRef("Push API")}}

The supportedContentEncodings read-only property of the {{domxref("PushManager")}} interface returns an array of supported content codings that can be used to encrypt the payload of a push message.

Syntax

var encodings[] = PushManager.supportedContentEncodings

Value

An array of strings.
