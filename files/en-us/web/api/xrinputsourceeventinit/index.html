The XRInputSourceEventInit dictionary is used when calling the {{domxref("XRInputSourceEvent.XRInputSourceEvent", "XRInputSourceEvent()")}} constructor to provide configuration options for the newly-created {{domxref("XRInputSourceEvent")}} object to take on.

Properties

The XRInputSourceEventInit dictionary inherits properties from the {{domxref("EventInit")}} dictionary. It also offers the following:

{{domxref("XRInputSourceEventInit.frame", "frame")}}
An {{domxref("XRFrame")}} object representing the event frame during which the event took place. This event is not associated with the animation process, and has no viewer information contained within it.
{{domxref("XRInputSourceEventInit.inputSource", "inputSource")}}
An {{domxref("XRInputSource")}} object representing the input device from which the event is being sent.
