{{APIRef("DOM")}}

The onabort read-only property of the {{domxref("AbortSignal")}} interface is an event handler Invoked when an {{event("abort_(cancellable_fetch)", "abort")}} event fires, i.e. when the fetch request(s) the signal is communicating with is/are aborted.

Syntax

abortSignal.onabort = function() { ... };

Examples

In the following snippet, we create a new AbortController object, and get its {{domxref("AbortSignal")}} (available in the signal property). Later onÂ we check whether or not it the signal has been aborted using the onabort property, and send an appropriate log to the console.

var controller = new AbortController();
var signal = controller.signal;

signal.onabort = function() {
  console.log('Request aborted');
};
