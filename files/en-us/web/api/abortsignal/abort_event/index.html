The abort event of the Fetch API is fired when a fetch request is aborted, i.e. using {{domxref("AbortController.abort()")}}.

Bubbles No
Cancelable No
Interface {{domxref("Event")}}
Event handler onabort

Examples

In the following snippets, we create a new AbortController object, and get its {{domxref("AbortSignal")}} (available in the signal property). Later onÂ we check whether or not it the signal has been aborted using the onabort property, and send an appropriate log to the console.

You can use the abort event in an addEventListener method:

var controller = new AbortController();
var signal = controller.signal;

signal.addEventListener('abort', function() {
 console.log('Request aborted');
};

Or use the onabort event handler property:

var controller = new AbortController();
var signal = controller.signal;

signal.onabort = function() {
  console.log('Request aborted');
};
