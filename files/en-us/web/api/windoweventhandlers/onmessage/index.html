The onmessage property of the {{domxref("WindowEventHandlers")}} mixin is the {{event("Event_handlers", "event handler")}} called whenever an object receives a {{event("message")}} event.

Syntax

window.addEventListener('message', function(event) { ... })
window.onmessage = function(event) { ... }
