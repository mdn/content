{{SeeCompatTable}}{{APIRef("MediaStream Recording")}}

The videoBitsPerSecond read-only property of the {{domxref("MediaRecorder")}} interface returns the video encoding bit rate in use. This may differ from the bit rate specified in the constructor, if it was provided.

Syntax

var videoBitsPerSecond = MediaRecorder.videoBitsPerSecond

Value

A {{jsxref("Number")}} (unsigned long).
