{{APIRef("Web Sockets API")}}

The WebSocket.onclose property is an event handler that is called when the WebSocket connection's {{domxref("WebSocket.readyState","readyState")}} changes to {{domxref("WebSocket.readyState","CLOSED")}}. It is called with a {{domxref("CloseEvent")}}.

Syntax

aWebSocket.onclose = function(event) {
  console.log("WebSocket is closed now.");
};

Value

An {{domxref("EventListener")}}.
