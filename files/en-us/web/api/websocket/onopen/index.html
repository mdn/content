{{APIRef("Web Sockets API")}}

The WebSocket.onopen property is an event handler that is called when the {{domxref("WebSocket")}} connection's {{domxref("WebSocket.readyState","readyState")}} changes to {{domxref("WebSocket.readyState","1")}}; this indicates that the connection is ready to send and receive data. It is called with an {{domxref("Event")}}.

Syntax

aWebSocket.onopen = function(event) {
  console.log("WebSocket is open now.");
};

Value

An {{domxref("EventListener")}}.
