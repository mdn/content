The CSSMathProduct() constructor creates a new {{domxref("CSSMathProduct")}} object which creates a new {{domxref('CSSMathProduct')}} object which multiplies the arguments passed into it. 

Syntax

var CSSMathProduct = new CSSMathProduct(args)

Parameters

args
A list of values for the {{domxref('CSSMathProduct')}} object to be either a double integer or a {{domxref('CSSNumericValue')}}.
