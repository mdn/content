The **`toJSON()`** method of the
{{domxref("PerformanceServerTiming")}} interface returns a string that
is the JSON representation of the {{domxref('PerformanceServerTiming')}} object.

## Syntax

```js
toJSON()
```

### Parameters

None.

### Return value

A {{domxref('DomString')}} containing JSON.
