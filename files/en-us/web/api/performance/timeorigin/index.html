{{SeeCompatTable}}{{APIRef("High Resolution Time")}}

The timeOrigin read-only property of the {{domxref("Performance")}} interface returns the high resolution timestamp of the start time of the performance measurement.

{{AvailableInWorkers}}

Syntax

var timeOrigin = performance.timeOrigin

Value

A high resolution timestamp.
