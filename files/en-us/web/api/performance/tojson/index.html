The toJSON() method of the {{domxref("Performance")}} interface is a standard serializer: it returns a JSON representation of the performance object's properties.

{{availableinworkers}}

Syntax

myPerf = performance.toJSON()

Arguments

None
Â 

Return value

myPerf
A JSON object that is the serialization of the {{domxref("Performance")}} object.

Example

var js;
js = window.performance.toJSON();
console.log("json = " + JSON.stringify(js));
