{{APIRef("WebGL")}}

The WebGLRenderingContext.isBuffer() method of the WebGL API returns true if the passed {{domxref("WebGLBuffer")}} is valid and false otherwise.

Syntax

GLboolean gl.isBuffer(buffer);

Parameters

buffer
A {{domxref("WebGLBuffer")}} to check.

Return value

A {{domxref("GLboolean")}} indicating whether or not the buffer is valid.

Examples

Creating a buffer

var canvas = document.getElementById('canvas');
var gl = canvas.getContext('webgl');
var buffer = gl.createBuffer();

gl.isBuffer(buffer);
