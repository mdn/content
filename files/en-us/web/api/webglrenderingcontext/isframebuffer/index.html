{{APIRef("WebGL")}}

The WebGLRenderingContext.isFramebuffer() method of the WebGL API returns true if the passed {{domxref("WebGLFramebuffer")}} is valid and false otherwise.

Syntax

GLboolean gl.isFramebuffer(framebuffer);

Parameters

framebuffer
A {{domxref("WebGLFramebuffer")}} to check.

Return value

A {{domxref("GLboolean")}} indicating whether or not the frame buffer is valid.

Examples

Checking a frame buffer

var canvas = document.getElementById('canvas');
var gl = canvas.getContext('webgl');
var framebuffer = gl.createFramebuffer();

gl.isFramebuffer(framebuffer);
