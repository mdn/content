{{APIRef("WebGL")}}

The WebGLRenderingContext.isRenderbuffer() method of the WebGL API returns true if the passed {{domxref("WebGLRenderbuffer")}} is valid and false otherwise.

Syntax

GLboolean gl.isRenderbuffer(renderbuffer);

Parameters

renderbuffer
A {{domxref("WebGLRenderbuffer")}} to check.

Return value

A {{domxref("GLboolean")}} indicating whether or not the renderbuffer is valid.

Examples

Checking a renderbuffer

var canvas = document.getElementById('canvas');
var gl = canvas.getContext('webgl');
var renderbuffer = gl.createRenderbuffer();

gl.isRenderbuffer(renderbuffer);
