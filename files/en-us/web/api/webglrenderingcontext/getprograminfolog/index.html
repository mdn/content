{{APIRef("WebGL")}}

The WebGLRenderingContext.getProgramInfoLog returns the information log for the specified {{domxref("WebGLProgram")}} object. It contains errors that occurred during failed linking or validation of WebGLProgram objects.

Syntax

gl.getProgramInfoLog(program);

Parameters

program
The {{domxref("WebGLProgram")}} to query.

Return value

A {{domxref("DOMString")}} that contains diagnostic messages, warning messages, and other information about the last linking or validation operation. When a {{domxref("WebGLProgram")}} object is initially created, its information log will be a string of length 0.

Examples

Checking program errors

var program = gl.createProgram();

// Attach pre-existing shaders
gl.attachShader(program, vertexShader);
gl.attachShader(program, fragmentShader);

gl.linkProgram(program);

gl.getProgramInfoLog(program);
