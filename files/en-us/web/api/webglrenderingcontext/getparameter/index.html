{{APIRef("WebGL")}}

The WebGLRenderingContext.getParameter() method of the WebGL API returns a value for the passed parameter name.

Syntax

any gl.getParameter(pname);

Parameters

pname
A {{domxref("GLenum")}} specifying which parameter value to return. See below for possible values.

Return value

Depends on the parameter.
