{{APIRef("WebGL")}}

The WebGLRenderingContext.isTexture() method of the WebGL API returns true if the passed {{domxref("WebGLTexture")}} is valid and false otherwise.

Syntax

GLboolean gl.isTexture(texture);

Parameters

texture
A {{domxref("WebGLTexture")}} to check.

Return value

A {{domxref("GLboolean")}} indicating whether or not the texture is valid.

Examples

Checking a texture

var canvas = document.getElementById('canvas');
var gl = canvas.getContext('webgl');
var texture = gl.createTexture();

gl.isTexture(texture);
