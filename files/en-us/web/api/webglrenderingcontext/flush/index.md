{{APIRef("WebGL")}}

The **`WebGLRenderingContext.flush()`** method of the [WebGL API](/en-US/docs/Web/API/WebGL_API) empties different buffer commands,
causing all commands to be executed as quickly as possible.

## Syntax

```js-nolint
flush()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).
