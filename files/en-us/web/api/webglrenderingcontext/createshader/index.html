{{APIRef("WebGL")}}

The {{domxref("WebGLRenderingContext")}} method createShader() of the WebGL API creates a {{domxref("WebGLShader")}} that can then be configured further using {{domxref("WebGLRenderingContext.shaderSource()")}} and {{domxref("WebGLRenderingContext.compileShader()")}}.

Syntax

WebGLShader gl.createShader(type);

Parameters

type
Either gl.VERTEX_SHADER or gl.FRAGMENT_SHADER

Examples

See {{domxref("WebGLShader")}} for usage and examples.
