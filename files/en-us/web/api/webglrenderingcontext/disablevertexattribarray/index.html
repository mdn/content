{{APIRef("WebGL")}}

The WebGLRenderingContext.disableVertexAttribArray() method of the WebGL API turns the generic vertex attribute array off at a given index position.

Syntax

void gl.disableVertexAttribArray(index);

Parameters

index
A {{domxref("GLuint")}} specifying the index of the vertex attribute to disable.

Return value

None.

Examples

gl.disableVertexAttribArray(0);
