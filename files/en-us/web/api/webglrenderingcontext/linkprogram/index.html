{{APIRef("WebGL")}}

The {{domxref("WebGLRenderingContext")}} interface's linkProgram() method links a given {{domxref("WebGLProgram")}}, completing the process of preparing the GPU code for the program's fragment and vertex shaders.

Syntax

void gl.linkProgram(program);

Parameters

program
The {{domxref("WebGLProgram")}} to link.

Return value

None.

Examples

var program = gl.createProgram();

// Attach pre-existing shaders
gl.attachShader(program, vertexShader);
gl.attachShader(program, fragmentShader);

gl.linkProgram(program);

if ( !gl.getProgramParameter( program, gl.LINK_STATUS) ) {
  var info = gl.getProgramInfoLog(program);
  throw new Error('Could not compile WebGL program. \n\n' + info);
}
