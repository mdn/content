{{APIRef("WebGL")}}

The WebGLRenderingContext.deleteBuffer() method of the WebGL API deletes a given {{domxref("WebGLBuffer")}}. This method has no effect if the buffer has already been deleted.

Syntax

void gl.deleteBuffer(buffer);

Parameters

buffer
A {{domxref("WebGLBuffer")}} object to delete.

Return value

None.

Examples

Deleting a buffer

var canvas = document.getElementById('canvas');
var gl = canvas.getContext('webgl');
var buffer = gl.createBuffer();

// ...

gl.deleteBuffer(buffer);
