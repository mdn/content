{{APIRef("WebGL")}}

The WebGLRenderingContext.isShader() method of the WebGL API returns true if the passed {{domxref("WebGLShader")}} is valid, false otherwise.

Syntax

GLboolean gl.isShader(shader);

Parameters

shader
A {{domxref("WebGLShader")}} to check.

Return value

A {{domxref("GLboolean")}} indicating whether or not the shader is valid.

Examples

Checking a shader

var canvas = document.getElementById('canvas');
var gl = canvas.getContext('webgl');
var shader = gl.createShader(gl.VERTEX_SHADER);

// ...

gl.isShader(shader);
