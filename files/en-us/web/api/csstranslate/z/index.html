The z property of the {{domxref("CSSTranslate")}} interface representing the z-component of the translating vector. A positive value moves the element towards the viewer, and a negative value farther away.

If this value is present then the transform is a 3D transform and the is2D property will be set to false.

Syntax

var translateZ = CSSTranslate.z;

Value

A {{cssxref('length')}}.
