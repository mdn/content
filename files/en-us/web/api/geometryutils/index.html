{{APIRef("CSSOM")}}

The GeometryUtils interface provides different utility function to retrieve geometry information about {{Glossary("DOM")}} nodes.

GeometryUtils is a raw interface and no object of this type can be created; it is implemented by {{domxref("Text")}}, {{domxref("Element")}}, {{domxref("CSSPseudoElement")}}, and {{domxref("Document")}} objects.

Properties

This interface does not implement any properties.

Methods

{{domxref("GeometryUtils.getBoxQuads()")}}{{experimental_inline}}
Returns a list of {{domxref("DOMQuad")}} objects representing the CSS fragments of the node.
