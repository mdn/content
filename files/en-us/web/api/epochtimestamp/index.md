{{APIRef("DOM")}}

The **`EpochTimeStamp`** type represents a timestamp value in milliseconds (excluding leap seconds), relative to 01 January, 1970 00:00:00 UTC.

This was previously known as {{domxref("DOMTimeStamp")}}.
