The appendData() method of the {{domxref("CharacterData")}} interface adds the provided data to the end of the Node's current data.

Syntax

  characterData.appendData(data)

Parameters

data
The data to append to the given CharacterData object.

Return value

{{jsxref("undefined")}}.
