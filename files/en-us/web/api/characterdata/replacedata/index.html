The CharacterData.replaceData() is similar to CharacterData.insertData(), but replaceData() allows overwriting of a certain number of bytes.

Syntax

  characterData.replaceData(offset, count, data)

Parameters

offset
The number of characters from the start of the data to insert at.
count
The number of characters from the offset to replace with the provided data.
data
The data to insert into the given CharacterData.

Return value

{{jsxref("undefined")}}.
