{{APIRef("Fullscreen API")}}

The FullscreenOptions dictionary is used to provide configuration options when calling {{DOMxRef("Element.requestFullscreen", "requestFullscreen()")}} on an element to place that element into full-screen mode.

Properties

{{DOMxRef("FullscreenOptions.navigationUI", "navigationUI")}}{{Optional_Inline}}
A string controlling whether or not to keep browser user interface elements visible while the element is in full-screen mode. The default, "auto", lets the browser make this decision.
