{{APIRef("WebXR Device API")}}

The XREye enumerated type is used to identify which eye a particular {{domxref("XRView")}} represents: left or right; or, if the view doesn't represent a single eye, the value may be none.

Values

left
The {{domxref("XRView")}} represents the point-of-view of the viewer's left eye.
right
The view represents the viewer's right eye.
none
The XRView describes a monoscopic view, or the view otherwise doesn't represent a particular eye's point-of-view.

Usage notes

These values are used by the {{domxref("XRView")}} property {{domxref("XRView.eye", "eye")}}.
