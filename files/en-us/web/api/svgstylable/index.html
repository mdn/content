{{APIRef("SVG")}}

The SVGStylable interface is implemented on all objects corresponding to SVG elements that can have {{ SVGAttr("style") }}, {{SVGAttr("class")}} and presentation attributes specified on them.

Methods

{{domxref("SVGStylable.getPresentationAttribute()")}} {{deprecated_inline}}
Returns the base (i.e., static) value of a given presentation attribute as an object of type {{ domxref("CSSValue") }}. The returned object is live; changes to the objects represent immediate changes to the objects to which the {{ domxref("CSSValue") }} is attached.
