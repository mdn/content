The read-only {{domxref("XRInputSourcesChangeEvent")}} property {{domxref("XRInputSourcesChangeEvent.removed", "removed")}} is an array of zero or more {{domxref("XRInputSource")}} objects representing the input sources which have been removed from the {{domxref("XRSession")}}.

Syntax

removedInputs = xrInputSourcesChangeEvent.removed;

Value

An {{jsxref("Array")}} of zero or more {{domxref("XRInputSource")}} objects, each representing one input device removed from the XR system.
