{{APIRef("Encoding API")}}

The **`TextEncoder()`** constructor returns a newly created
{{DOMxRef("TextEncoder")}} object that will generate a byte stream with UTF-8 encoding.

## Syntax

```js
new TextEncoder();
```

### Parameters

None.
