{{APIRef("History API")}}

The History.forward() method causes the browser to move forward one page in the session history. It has the same effect as calling {{domxref("History.go", "history.go(1)")}}.

This method is {{glossary("asynchronous")}}. Add a listener for the {{event("popstate")}} event in order to determine when the navigation has completed.

Syntax

history.forward()

Examples

The following examples createÂ a button that moves forward one step in the session history.

HTML

<button id='go-forward'>Go Forward!</button>

JavaScript

document.getElementById('go-forward').addEventListener('click', e => {
  window.history.forward();
})
