{{APIRef("History API")}}

The History.back() method causes the browser to move back one page in the session history. It has the same effect as calling {{domxref("History.go", "history.go(-1)")}}. If there is no previous page, this method call does nothing.

This method is {{glossary("asynchronous")}}. Add a listener for the {{event("popstate")}} event in order to determine when the navigation has completed.

Syntax

history.back()

Examples

The following short example causes a button on the page to navigate back one entry in the session history.

HTML

<button id="go-back">Go back!</button>

JavaScript

document.getElementById('go-back').addEventListener('click', () => {
  history.back();
});
