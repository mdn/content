The **`close()`** method of the {{domxref("AudioData")}} interface clears all states and releases the reference to the media resource.

## Syntax

```js
close()
```

### Parameters

None.

### Return value

Undefined.

## Examples

The following example shows the `AudioData` object being closed.

```js
AudioData.close();
```
