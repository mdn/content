The **`sampleRate`** read-only property of the {{domxref("AudioData")}} interface returns the sample rate in Hz.

## Value

A decimal value.

## Examples

The below example prints the value of `sampleRate` to the console.

```js
console.log(audioData.sampleRate);
```
