The PaintWorklet.devicePixelRatio read-only property of the {{domxref("PaintWorklet")}} interface returns the current device's ratio of physical pixels to logical pixels.

Syntax

var devicePixelRatio = paintWorklet.devicePixelRatio;

Value

A double-precision integer.
