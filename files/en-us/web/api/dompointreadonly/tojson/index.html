{{APIRef("DOM")}}

The {{domxref("DOMPointReadOnly")}} method toJSON() returns a {{domxref("DOMPointInit")}} object giving the {{Glossary("JSON")}} form of the point object.

Syntax

pointJSON = DOMPointReadOnly.toJSON();

Parameters

None.

Return value

A new {{domxref("DOMPointInit")}} object whose properties are set to the values in the DOMPoint or DOMPointReadOnly on which the method was called.

Example

This example creates a {{domxref("DOMPoint")}} object representing the top-left corner of the current window, in screen coordinates, then converts that to JSON.

var topLeft = new DOMPoint(window.screenX, window.screenY);

var pointJSON = topLeft.toJSON();
