{{ APIRef("DOM") }}

The DocumentFragment.lastElementChild read-only property returns the document fragment's last child {{domxref("Element")}}, or null if there are no child elements.

Syntax

// Getter
element = fragment.lastElementChild;

// No setter; read-only property

Example

let fragment = new DocumentFragment();
fragment.lastElementChild; // null

let paragraph = document.createElement('p');
fragment.appendChild(paragraph);

fragment.lastElementChild; // <p>
