{{ APIRef("DOM") }}

The DocumentFragment.firstElementChild read-only property returns the document fragment's first child {{domxref("Element")}}, or null if there are no child elements.

Syntax

// Getter
element = fragment.firstElementChild;

// No setter; read-only property

Example

let fragment = new DocumentFragment();
fragment.firstElementChild; // null

let paragraph = document.createElement('p');
fragment.appendChild(paragraph);

fragment.firstElementChild; // <p>
