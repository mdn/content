The CSSMathMin() constructor creates a new {{domxref("CSSMathMin")}} object which represents the CSS {{CSSXref('min()','min()')}} function.

Syntax

var CSSMathMin = new CSSMathMin(args);

Parameters

args
A list of values for the {{domxref('CSSMathProduct')}} object to be either a double integer or a {{domxref('CSSNumericValue')}}.

Exceptions

TypeError
Thrown if there is a failure when adding all of the values in args.
