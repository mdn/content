The removeSourceBuffer() method of the {{domxref("MediaSource")}} interface removes the given {{domxref("SourceBuffer")}} from the {{domxref("SourceBuffers")}} list associated with this MediaSource object.

Syntax

mediaSource.removeSourceBuffer(sourceBuffer);

Parameters

sourceBuffer
The {{domxref("SourceBuffer")}} object to be removed.

Return value

{{jsxref('undefined')}}

Exceptions

Exception Explanation
NotFoundError The supplied sourceBuffer doesn't exist in {{domxref("MediaSource.sourceBuffers")}}.

Examples

for (i = 0; i < 10; i++) {
  var sourceBuffer = mediaSource.addSourceBuffer(mimeCodec);
}

mediaSource.removeSourceBuffer(mediaSource.sourceBuffers[0]);
