--- title: "HTMLMapElement: name property" short-title: name slug: Web/API/HTMLMapElement/name page-type: web-api-instance-property browser-compat: api.HTMLMapElement.name --- {{ApiRef("HTML DOM")}} The **`name`** property of the {{domxref("HTMLMapElement")}} is a string gives the map unique name. Its value can be used with the `useMap` attribute of the {{HTMLElement("img")}} element to reference a `` element. If `id` is given to a {{HTMLElement("map")}} element then `name` property should be same as `id`. ## Value A non-empty string without whitespaces. ## Example ```html ``` ```js const mapElement = document.getElementsByName("image-map")[0]; console.log(mapElement.name); // output: "image-map" ``` ## Specifications {{Specifications}} ## Browser compatibility {{Compat}} ## See also - {{domxref("HTMLImageElement.useMap")}} property - {{domxref("HTMLAreaElement")}} element
