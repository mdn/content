{{SeeCompatTable}}{{APIRef("MediaStream Image")}}

The imageHeight read-only property of the {{domxref("PhotoCapabilities")}} interface returns a {{domxref("MediaSettingsRange")}} object indicating the image height range supported by the user agent.

Syntax

var mediaSettingsRange = photoCapabilities.imageHeight

Value

A {{domxref("MediaSettingsRange")}} object.
