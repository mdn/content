The **`encoding`**
property of the {{DOMxRef("NDEFRecord")}} interface returns the encoding of
a textual payload, or `null` otherwise.

## Syntax

```js
NDEFRecord.encoding
```

### Value

A {{DOMxRef("USVString")}} which can be one of the following: `"utf-8"`,
`"utf-16"`, `"utf-16le"`, or `"utf-16be"`.
