The lang property ofthe {{DOMxRef("NDEFRecord")}} interface is {{DOMxRef("USVString")}} containing the {{Glossary("language tag")}} of the record contents, if it is a vailable.

The record might be missing a language tag, for example, if the recorded information is not locale-specific.

Syntax

NDEFRecord.lang

Value

A {{DOMxRef("USVString")}}.
