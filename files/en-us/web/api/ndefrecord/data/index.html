The data property of the {{DOMxRef("NDEFRecord")}} interface is {{DOMxRef("DataView")}} providing access to the payload data of the record.

It provides access to a raw content of the record and thus data consumer might need to decode it.

Syntax

NDEFRecord.data

Value

A {{DOMxRef("DataView")}} that contains encoded payload data of the record.
