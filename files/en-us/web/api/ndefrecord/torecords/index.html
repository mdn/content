The toRecords() method of the {{DOMxRef("NDEFRecord")}} interface of Web NFC API parses record payload {{DOMxRef("NDEFRecord.data")}} besed on {{DOMxRef("NDEFRecord.recordType")}} and returns the result.

In practice, it's possible to implement the parsing mechanism manually, but this method simplifies the process.

Syntax

NDEFRecord.toRecords()

Parameters

None.

Return value

A list of {{DOMxRef("NDEFRecord")}}s.

Exceptions

NotSupported
The {{Glossary("User Agent")}} does not know how to parse this combination of {{DOMxRef("NDEFRecord.data")}} and {{DOMxRef("NDEFRecord.recordType")}}.
