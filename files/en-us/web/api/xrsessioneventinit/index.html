The XRSessionEventInit dictionary is used when calling the {{domxref("XRSessionEvent.XRSessionEvent", "XRSessionEvent()")}} constructor to provide the new event's initial values. You're unlikely to need to create these events yourself, however, as they're created by the XR system.

Properties

{{domxref("XRSessionEventInit.session", "session")}}
The {{domxref("XRSession")}} to which the event is to be delivered.
