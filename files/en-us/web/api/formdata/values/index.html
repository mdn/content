{{APIRef("XMLHttpRequest")}}

The FormData.values() method returns an {{jsxref("Iteration_protocols",'iterator')}} allowing to go through all values contained in this object. The values are {{domxref("USVString")}} or {{domxref("Blob")}} objects.

Note: This method is available in Web Workers.

Syntax

formData.values();

Return value

Returns an {{jsxref("Iteration_protocols","iterator")}}.

Example

// Create a test FormData object
var formData = new FormData();
formData.append('key1', 'value1');
formData.append('key2', 'value2');

// Display the values
for (var value of formData.values()) {
   console.log(value);
}

The result is:

value1
value2
