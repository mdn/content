The CSSSkew interface of the {{domxref('CSS_Object_Model#css_typed_object_model','','',' ')}} is part of the {{domxref('CSSTransformValue')}} interface. It represents the skew() value of the individual {{CSSXRef('transform')}} property in CSS.

Constructor

{{domxref("CSSSkew.CSSSkew()")}}
Creates a new CSSSkew object.

Properties

{{domxref('CSSSkew.ax','ax')}}
Returns or sets the x-axis value.
{{domxref('CSSSkew.ay','ay')}}
Returns or sets the y-axis value.
