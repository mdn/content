The HTMLAnchorElement.pathname property is a {{domxref("USVString")}} containing an initial '/' followed by the path of the URLÂ not including the query string or fragment (or the empty string if there is no path).

Syntax

// Getter
string = anchor.pathname;
// Setter
anchor.pathname = string;

Examples

// An <a id="myAnchor" href="/en-US/docs/HTMLAnchorElement"> element is in the document
const anchor = document.getElementById("myAnchor");
anchor.pathname; // returns '/en-US/docs/HTMLAnchorElement'
