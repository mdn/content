The blob() method of the {{domxref("FontMetadata")}} interface requests the font data.

Syntax

let null = FontMetadata.blob();

Parameters

None.

Return value

A {{jsxref("Promise")}} which resolves with a byte sequence containing a table list, which is a list of font tables.
