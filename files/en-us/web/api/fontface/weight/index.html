The weight property of the {{domxref("FontFace")}} interface retrieves or sets the weight of the font. It is equivalent to the {{cssxref("@font-face/font-weight", "font-weight")}} descriptor.

Syntax

var weightDescriptor = FontFace.weight;
FontFace.weight = weightDescriptor;

Value

A {{domxref('CSSOMString')}} containing a descriptor as it would be defined in a style sheet's @font-face rule.
