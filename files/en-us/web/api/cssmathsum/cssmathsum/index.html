The CSSMathSum() constructor creates a new {{domxref("CSSMathSum")}} object which creates a new {{domxref('CSSKeywordValue')}} object which represents the result obtained by calling {{domxref('CSSNumericValue.add','add()')}}, {{domxref('CSSNumericValue.sub','sub()')}}, or {{domxref('CSSNumericValue.toSum','toSum()')}} on {{domxref('CSSNumericValue')}}.

Syntax

var CSSMathSum = new CSSMathSum(values)

Parameters

values
One or more double integers or {{domxref('CSSNumericValue')}} objects.
