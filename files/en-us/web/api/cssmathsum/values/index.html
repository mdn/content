The CSSMathSum.values read-only property of the {{domxref("CSSMathSum")}} interface returns a {{domxref('CSSNumericArray')}} object which contains one or more {{domxref('CSSNumericValue')}} objects.

Syntax

var cssNumericArray = CSSMathSum.values;

Value

A {{domxref('CSSNumericArray')}}.
