The **`CSSMathSum.values`** read-only property
of the {{domxref("CSSMathSum")}} interface returns a {{domxref('CSSNumericArray')}}
object which contains one or more {{domxref('CSSNumericValue')}} objects.

## Value

A {{domxref('CSSNumericArray')}}.
