The onavailabilitychanged property of the {{DOMxRef("Bluetooth")}} interface is an {{event("Event_handlers", "event handler")}} that processes availabilitychanged events that fire when the Bluetooth system as a whole becomes available or unavailable to the {{Glossary("User Agent")}}.

Syntax

Bluetooth.onavailabilitychanged = functionRef;

Value

functionRef is the handler function to be called when the Bluetooth availabilitychanged event fires.
