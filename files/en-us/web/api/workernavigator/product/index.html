{{APIRef("HTML DOM")}} {{Deprecated_Header}}

The value of the WorkerNavigator.product property is always "Gecko", in any browser. This property is kept only for compatibility purposes.

Note: Do not rely on this property to return a real product name. All browsers return "Gecko" as the value of this property.

Syntax

productName = navigator.product

Value

The string "Gecko".
