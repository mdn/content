{{APIRef("HTML DOM")}} {{Deprecated_Header}}

The value of the WorkerNavigator.appName property is always "Netscape", in any browser. This property is kept only for compatibility purposes.

Note

Do not rely on this property to return a real browser name. All browsers return "Netscape" as the value of this property.

Syntax

appName = navigator.appName

Value

The string "Netscape".
