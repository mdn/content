{{APIRef("WebRTC API")}}

The getParameters() method of the {{domxref("RTCRtpSender")}} interface returns an {{domxref("RTCRtpSendParameters")}} object describing the current configuration for the encoding and transmission of media on the {{domxref("RTCRtpSender.track")}} property.

Syntax

var rtpSendParameters = rtpSender.getParameters()

Parameters

None.

Return value

An {{domxref("RTCRtpSendParameters")}} object indicating the current configuration of the sender.
