{{APIRef}}

The **`CanvasRenderingContext2D.reset()`** method of the Canvas 2D API resets the rendering context, including the backing buffer, the drawing state stack, any defined paths, and styles.

## Syntax

```js
reset()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).
