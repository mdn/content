{{APIRef("SVG")}}

The cy read-only property of the {{domxref("SVGCircleElement")}} interface reflects the {{SVGAttr("cy")}} attribute of a {{SVGElement("circle")}} element and by that defines the y-coordinate of the circle's center. If unspecified, the effect is as if theÂ value is set to 0.

Syntax

var yCoordinate = element.cy;

Value

An {{domxref("SVGAnimatedLength")}} representing the y-coordinate of the circle's center.

Example

SVG

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100" width="200" height="200">
  <circle cy="50" cy="50" r="50" fill="gold" id="circle"/>
</svg>

JavaScript

const circle = document.getElementById('circle');
console.log(circle.cy);
