The **`close()`** method of the {{domxref("AudioEncoder")}} interface ends all pending work and releases system resources.

## Syntax

```js
AudioEncoder.close()
```

### Parameters

None.

### Return Value

{{jsxref('undefined')}}.

## Examples

The following example closes the `AudioEncoder`.

```js
AudioEncoder.close();
```
