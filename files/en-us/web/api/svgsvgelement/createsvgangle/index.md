{{APIRef("SVG")}}

The `createSVGAngle()` method of the {{domxref("SVGSVGElement")}} interface creates an {{domxref("SVGAngle")}} object outside of any document trees.

## Syntax

```js-nolint
createSVGAngle()
```

### Parameters

None.

### Return value

An {{domxref("SVGAngle")}} object, initialized to a value of `0` (unitless).
