{{APIRef("SVG")}}

The `createSVGNumber()` method of the {{domxref("SVGSVGElement")}} interface creates an {{domxref("SVGNumber")}} object outside of any document trees.

The object is initialized to `0`.

## Syntax

```js-nolint
createSVGNumber()
```

### Parameters

None.

### Return value

An {{domxref("SVGNumber")}} object.
