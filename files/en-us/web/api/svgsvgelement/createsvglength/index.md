{{APIRef("SVG")}}

The `createSVGLength()` method of the {{domxref("SVGSVGElement")}} interface creates an {{domxref("SVGLength")}} object outside of any document trees.

## Syntax

```js-nolint
createSVGLength()
```

### Parameters

None.

### Return value

An {{domxref("SVGLength")}} object, initialized to a value of `0` (unitless).
