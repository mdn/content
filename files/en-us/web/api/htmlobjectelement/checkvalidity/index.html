{{APIRef("HTML DOM")}}

The checkValidity() method of the {{domxref("HTMLObjectElement")}} interface returns a {{jsxref("Boolean")}} that always is true, because object objects are never candidates for constraint validation.

Syntax

const valid = HTMLObjectElement.checkValidity();

Parameters

None.

Return value

true

Exceptions

None.
