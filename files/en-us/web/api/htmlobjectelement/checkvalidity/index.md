{{APIRef("HTML DOM")}}

The **`checkValidity()`** method of the
{{domxref("HTMLObjectElement")}} interface returns a boolean value that always
is true, because object objects are never candidates for constraint validation.

## Syntax

```js
checkValidity()
```

### Parameters

None.

### Return value

`true`

### Exceptions

None.
