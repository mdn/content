{{APIRef("HTML DOM")}}

The **`data`** property of the
{{domxref("HTMLObjectElement")}} interface returns a string that
reflects the {{htmlattrxref("data", "object")}} HTML attribute, specifying the address
of a resource's data.

## Value

A string.
