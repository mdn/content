{{APIRef("HTML DOM")}}

The **`data`** property of the
{{domxref("HTMLObjectElement")}} interface returns a string that
reflects the [`data`](/en-US/docs/Web/HTML/Element/object#data) HTML attribute, specifying the address
of a resource's data.

## Value

A string.
