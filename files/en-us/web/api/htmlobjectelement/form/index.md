{{APIRef("HTML DOM")}}

The **`form`** read-only property of the
{{domxref("HTMLObjectElement")}} interface returns a {{domxref("HTMLFormElement")}}
representing the object element's form owner, or null if there isn't one.

## Value

A {{domxref('HTMLFormElement')}}.
