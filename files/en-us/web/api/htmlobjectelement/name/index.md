{{APIRef("HTML DOM")}}

The **`name`** property of the
{{domxref("HTMLObjectElement")}} interface returns a string that
reflects the [`name`](/en-US/docs/Web/HTML/Element/object#name) HTML attribute, specifying the name of
the browsing context.

## Value

A string.
