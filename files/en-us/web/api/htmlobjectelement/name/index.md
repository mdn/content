{{APIRef("HTML DOM")}}

The **`name`** property of the
{{domxref("HTMLObjectElement")}} interface returns a string that
reflects the {{htmlattrxref("name", "object")}} HTML attribute, specifying the name of
the browsing context.

## Value

A string.
