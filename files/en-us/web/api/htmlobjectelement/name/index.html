{{APIRef("HTML DOM")}}

The name property of the {{domxref("HTMLObjectElement")}} interface returns a {{domxref("DOMString")}} that reflects the {{htmlattrxref("name", "object")}} HTML attribute, specifying the name of the browsing context.

Syntax

var String = HTMLObjectElement.name;
HTMLObjectElement.name = String;

Value

A {{domxref('DOMString')}}.
