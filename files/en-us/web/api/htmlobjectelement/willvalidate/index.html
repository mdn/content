{{APIRef("HTML DOM")}}

The willValidate read-only property of the {{domxref("HTMLObjectElement")}} interface returns a {{jsxref("Boolean")}} that indicates whether the element is a candidate for constraint validation. Always false for HTMLObjectElement objects.

Syntax

var Boolean = HTMLObjectElement.willValidate;

Value

A {{jsxref("Boolean")}}.
