{{APIRef("HTML DOM")}}

The type property of the {{domxref("HTMLObjectElement")}} interface returns a {{domxref("DOMString")}} that reflects the {{htmlattrxref("type", "object")}} HTML attribute, specifying the MIME type of the resource.

Syntax

var String = HTMLObjectElement.type
HTMLObjectElement.type = String;

Value

A {{domxref('DOMString')}}.
