{{APIRef("HTML DOM")}}

The **`type`** property of the
{{domxref("HTMLObjectElement")}} interface returns a string that
reflects the {{htmlattrxref("type", "object")}} HTML attribute, specifying the MIME type
of the resource.

## Value

A string.
