{{APIRef("HTML DOM")}}

The **`height`** property of the
{{domxref("HTMLObjectElement")}} interface Returns a string that
reflects the {{htmlattrxref("height", "object")}} HTML attribute, specifying the
displayed height of the resource in CSS pixels.

## Value

A string.
