{{APIRef("HTML DOM")}}

The useMap property of the {{domxref("HTMLObjectElement")}} interface returns a {{domxref("DOMString")}} that reflects the {{htmlattrxref("usemap", "object")}} HTML attribute, specifying a {{HTMLElement("map")}} element to use.

Syntax

var String = HTMLObjectElement.useMap;
HTMLObjectElement.useMap = String;

Value

A {{domxref('DOMString')}}.
