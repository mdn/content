{{APIRef("HTML DOM")}}

The **`useMap`** property of the
{{domxref("HTMLObjectElement")}} interface returns a string that
reflects the {{htmlattrxref("usemap", "object")}} HTML attribute, specifying a
{{HTMLElement("map")}} element to use.

## Value

A string.
