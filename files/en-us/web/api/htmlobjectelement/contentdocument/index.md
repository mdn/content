{{APIRef("HTML DOM")}}

The **`contentDocument`** read-only property of
the {{domxref("HTMLObjectElement")}} interface Returns a {{domxref("Document")}}
representing the active document of the object element's nested browsing context, if
any; otherwise null.

## Value

A {{domxref('Document')}}.
