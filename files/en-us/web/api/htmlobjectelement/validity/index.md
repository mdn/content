{{APIRef("HTML DOM")}}

The **`validity`** read-only property of the
{{domxref("HTMLObjectElement")}} interface returns a {{domxref("ValidityState")}} with
the validity states that this element is in.

## Value

A {{domxref("ValidityState")}} object.
