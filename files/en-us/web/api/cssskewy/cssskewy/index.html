The CSSSkewY() constructor creates a new {{domxref("CSSSkewY")}} object which represents the skewY() value of the individual {{CSSXRef('transform')}} property in CSS.

Syntax

var CSSSkewY = new CSSSkewY(ay);

Parameters

{{domxref('CSSSkewY.ay','ay')}}
A value for the ay angle of the {{domxref('CSSSkewY')}} object to be constructed. This must be a {{domxref('CSSNumericValue')}}.
