The CSSSkewX interface of the {{domxref('CSS Typed Object Model API','','',' ')}} represents the skewX() value of the individual {{CSSXRef('transform')}} property in CSS. It inherits properties and methods from its parent {{domxref("CSSTransformValue")}}.
