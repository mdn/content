{{APIRef("WebRTC API")}}

The read-only source property of the {{domxref("RTCRtpContributingSource")}} interface returns the source identifier of a particular stream of RTP packets. The value is the contributing source (CSRC) or synchronization source (SSRC) identifier, depending on whether the object is an RTCRtpContributingSource or {{domxref("RTCRtpSynchronizationSource")}}, which is based on the former.

Syntax

var sourceID = RTCRtpContributingSource.source

Value

An unsigned, 32-bit integer value which uniquely identifies the source of RTP packets described by this RTCRtpContributingSource (in which case the value is a CSRC identifier) or RTCRtpSynchronizationSource (the value is an SSRC identifier).
