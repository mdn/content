{{APIRef("Screen Orientation")}}{{SeeCompatTable}}

The lock() property of the {{domxref("ScreenOrientation")}} interface locks the orientation of the containing document to its default orientation.

Syntax

screen.orientation.lock(orientation)

Parameters

orientation
An orientation lock type. One of the following:
  • "any"
  • "natural"
  • "landscape"
  • "portrait"
  • "portrait-primary"
  • "portrait-secondary"
  • "landscape-primary"
  • "landscape-secondary"

Return value

A {{jsxref("Promise")}}.
