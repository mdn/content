{{APIRef("Screen Orientation")}}

The **`unlock()`** property of the
{{domxref("ScreenOrientation")}} interface unlocks the orientation of the containing
document from its default orientation.

## Syntax

```js-nolint
unlock()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).
