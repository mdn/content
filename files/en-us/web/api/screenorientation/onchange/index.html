The onchange property of the {{domxref("ScreenOrientation")}} is an event handler fired whenever is the {{domxref("EventHandler")}} called when the screen changes orientation.

Syntax

ScreenOrientation.addEventListener('change', function(e) { ... })
ScreenOrientation.onchange = function(e) { ... }
