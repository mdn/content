{{ APIRef("CSSOM") }}

The CSSNamespaceRule.namespaceURI read-only property returns a {{domxref("DOMString")}} containing the text of the URI of the given namespace.

Syntax

var namespaceURI = CSSNamespaceRule.namespaceURI

Returns

A {{domxref("DOMString")}} containing a URI.
