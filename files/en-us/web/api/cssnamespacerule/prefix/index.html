{{ APIRef("CSSOM") }}

The CSSNamespaceRule.prefix read-only property returns a {{domxref("DOMString")}} with the name of the prefix associated to this namespace. If there is no such prefix, returns  null.

Syntax

var prefix = CSSNamespaceRule.prefix

Returns

A {{domxref("DOMString")}} containing the prefix associated to this namespace.
