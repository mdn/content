{{APIRef("Geometry Interfaces")}}

The **`y`** read-only property of the **`DOMRectReadOnly`** interface represents the y coordinate of the `DOMRect`'s origin.

## Syntax

```js
var recY = DOMRect.y;
```

### Value

A double.
