<p>The <code><strong>left</strong></code> read-only property of the <strong><code>DOMRectReadOnly</code></strong> interface returns the left coordinate value of the <code>DOMRect.</code> (Has the same value as <code>x</code>, or <code>x + width</code> if <code>width</code> is negative.)</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js">var recLeft = DOMRect.left;</pre>

<h3 id="Value">Value</h3>

<p>A double.</p>
