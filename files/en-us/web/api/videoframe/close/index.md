The **`close()`** method of the {{domxref("VideoFrame")}} interface clears all states and releases the reference to the media resource.

## Syntax

```js
close()
```

### Parameters

None.

### Return value

Undefined.

## Examples

The following example shows the `VideoFrame` object being closed.

```js
VideoFrame.close();
```
