{{APIRef("SVG")}}

The SVGAnimationElement.targetElement property refers to the element which is being animated. If no target element is being animated (for example, because the {{SVGAttr("href")}} attribute specifies an unknown element), the value returned is null.

Syntax

var targetElement = someElement.targetElement;
