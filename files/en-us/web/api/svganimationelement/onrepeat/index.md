The **`SVGAnimationElement.onrepeat`** property refers to the
event handler for the
[`repeatEvent`](/en-US/docs/Web/API/SVGAnimationElement/repeatEvent_event).

## Syntax

```js
var repeatEventHandler = someElement.onrepeat;
```
