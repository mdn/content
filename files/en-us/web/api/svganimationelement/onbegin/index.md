The **`SVGAnimationElement.onbegin`** property refers to the
event handler for the
[`beginEvent`](/en-US/docs/Web/API/SVGAnimationElement/beginEvent_event).

## Syntax

```js
var beginEventHandler = someElement.onbegin;
```
