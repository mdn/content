{{ApiRef}}

The **`embeds`** read-only property of the
{{domxref("Document")}} interface returns a list of the embedded
{{htmlelement("embed")}} elements within the current document.

## Syntax

```js
nodeList = document.embeds
```

### Value

An {{domxref("HTMLCollection")}}.
