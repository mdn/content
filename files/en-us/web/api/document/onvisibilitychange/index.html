The Document.onvisibilitychange property represents the event handler that is called when a {{event("visibilitychange")}} event reaches this object.

Syntax

obj.onvisibilitychange = function;
  • function is the name of a user-defined function, without the () suffix or any parameters, or an anonymous function declaration.

Example

document.onvisibilitychange = function() {

};
