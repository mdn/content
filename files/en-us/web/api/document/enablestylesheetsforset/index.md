<p>Enables the style sheets matching the specified name in the current style sheet set,
  and disables all other style sheets (except those without a title, which are always
  enabled).</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js"><var>document</var>.enableStyleSheetsForSet(<var>name</var>);
</pre>

<h3 id="Parameters">Parameters</h3>

<dl>
  <dt><var>name</var></dt>
  <dd>The name of the style sheets to enable. All style sheets with a title that match
    this name will be enabled, while all others that have a title will be disabled.
    Specify an empty string for the <var>name</var> parameter to disable all alternate and
    preferred style sheets (but not the persistent style sheets; that is, those with no
    <code>title</code> attribute).</dd>
</dl>

<h2 id="Notes">Notes</h2>

<ul>
  <li>Title matches are case-sensitive.</li>
  <li>Calling this method with a <code>null</code> <var>name</var> has no effect; if you
    want to disable all alternate and preferred style sheets, you <strong>must</strong>
    pass "", the empty string.</li>
  <li>Stylesheets that don't have a title are never affected by this method.</li>
  <li>This method never affects the values of {{domxref("document.lastStyleSheetSet")}} or
    {{domxref("document.preferredStyleSheetSet")}}.</li>
</ul>

<h2 id="Example">Example</h2>

<pre>document.enableStyleSheetsForSet("Some style sheet set name");
</pre>
