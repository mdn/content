{{ApiRef("DOM")}}

The documentURI read-only property of the {{domxref("Document")}} interface returns the document location as a string.

Syntax

const uri = document.documentURI

Example

JavaScript

document.getElementById("url").textContent = document.documentURI;

HTML

<p id="urlText">
  URL:<br/>
  <span id="url">URL goes here</span>
</p>
