{{APIRef}}

The copy event fires when the user initiates a copy action through the browser's user interface.

Bubbles Yes
Cancelable Yes
Interface {{domxref("ClipboardEvent")}}
Event handler property {{domxref("HTMLElement/oncopy", "oncopy")}}

The original target for this event is the {{domxref("Element")}} that was the intended target of the copy action. You can listen for this event on the {{domxref("Document")}} interface to handle it in the capture or bubbling phases. For full details on this event please see the page on the Element: copy event.

Examples

document.addEventListener('copy', (event) => {
    console.log('copy action initiated')
});
