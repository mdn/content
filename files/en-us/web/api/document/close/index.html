{{APIRef("DOM")}}

The Document.close() method finishes writing to a document, opened with {{domxref("Document.open()")}}.

Syntax

document.close();

Example

// Open a document to write to it
document.open();

// Write the content of the document
document.write("<p>The one and only content.</p>");

// Close the document
document.close();
