{{APIRef}}

The dragleave event is fired when a dragged element or text selection leaves a valid drop target.

Bubbles Yes
Cancelable No
Default action None.
Interface {{domxref("DragEvent")}}
Event handler property {{domxref("GlobalEventHandlers/ondragleave", "ondragleave")}}
