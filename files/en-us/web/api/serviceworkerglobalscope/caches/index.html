{{APIRef("Service Workers API")}}

The caches read-only property of the {{domxref("ServiceWorkerGlobalScope")}} interface returns the {{domxref("CacheStorage")}} object associated with the service worker.

Syntax

var myCacheStorage = self.caches;

Value

A {{domxref("CacheStorage")}} object.
