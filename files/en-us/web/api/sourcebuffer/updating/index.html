The updating read-only property of the {{domxref("SourceBuffer")}} interface indicates whether the SourceBuffer is currently being updated — i.e. whether an {{domxref("SourceBuffer.appendBuffer()")}},  {{domxref("SourceBuffer.appendStream()")}}, or {{domxref("SourceBuffer.remove()")}} operation is currently in progress.

Syntax

var isUpdating = sourceBuffer.updating;

Value

A {{jsxref("Boolean")}}.
