The appendBuffer() method of the {{domxref("SourceBuffer")}} interface appends media segment data from an {{jsxref("ArrayBuffer")}} or ArrayBufferView object to the SourceBuffer.

Syntax

sourceBuffer.appendBuffer(source);

Parameters

source
A {{domxref("BufferSource")}} (that is, either an {{domxref("ArrayBufferView")}} or {{jsxref("ArrayBuffer")}}) which contains the media segment data you want to add to the SourceBuffer.

Return value

{{jsxref('undefined')}}.

Exceptions

None.
