The
**`BluetoothRemoteGATTServer.connect()`** method causes the
script execution environment to connect to `this.device`.

## Syntax

```js-nolint
connect()
```

### Parameters

None.

### Return value

A {{jsxref("Promise")}} that resolves to a {{domxref("BluetoothRemoteGATTServer")}}.
