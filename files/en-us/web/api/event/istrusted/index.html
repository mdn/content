{{APIRef("DOM")}}

The isTrusted read-only property of the {{domxref("Event")}} interface is a {{domxref("Boolean")}} that is true when the event was generated by a user action, and false when the event was created or modified by a script or dispatched via {{domxref("EventTarget.dispatchEvent()")}}.

Syntax

var eventIsTrusted = event.isTrusted;

Value

{{domxref("Boolean")}}

Example

if (e.isTrusted) {
  /* The event is trusted */
} else {
  /* The event is not trusted */
}
