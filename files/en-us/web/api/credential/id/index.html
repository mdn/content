{{SeeCompatTable}}{{APIRef("Credential Management API")}}

The id property of the {{domxref("Credential")}} interface returns a {{domxref("DOMString")}} containing the credential's identifier. This might be any one of a GUID, username, or email address.

Syntax

var id = Credential.id;

Value

A {{domxref("DOMString")}} containing the credential's identifier.
