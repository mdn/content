The reportId read-only property of the {{domxref("HIDInputReportEvent")}} interface returns the reportId value.

Syntax

let reportId = HIDInputReportEvent.reportId;

Value

An 8-bit report ID. If the HID device does not use report IDs, this will be 0.

Examples

In the following example the reportId of an incoming input report is logged to the console.

device.addEventListener("inputreport", event => {
  const { data, device, reportId } = event;
  console.log(reportId);
});
