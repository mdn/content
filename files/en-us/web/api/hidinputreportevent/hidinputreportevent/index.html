{{APIRef("WebHID API")}}

The HIDInputReportEvent() constructor creates a new {{domxref("HIDInputReportEvent")}} object. Typically this constructor is not used as events are created when an input report is received.

Syntax

new HIDInputReport(type, HIDInputReportEventInit)

Parameters

type
A {{domxref("DOMString")}} with the value "inputreport".
HIDInputReportEventInit
A dictionary including the following fields:
data
The a {{jsxref("DataView")}} object containing the input report.
device
The {{domxref("HIDDevice")}} instance representing the device sending the input report.
reportId>
An 8-bit reportId. If the HID device does not use report IDs, will be 0.
bubbles {{optional_inline}}
A {{jsxref("Boolean")}} indicating whether the event bubbles. The default is false.
cancelable {{optional_inline}}
A {{jsxref("Boolean")}} indicating whether the event can be cancelled. The default is false.
composed {{optional_inline}}
A {{jsxref("Boolean")}} indicating whether the event will trigger listeners outside of a shadow root (see {{domxref("Event.composed")}} for more details). The default is false.
