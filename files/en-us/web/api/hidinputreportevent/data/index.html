The data read-only property of the {{domxref("HIDInputReportEvent")}} interface contains the data sent by device as the input report.

Syntax

let data = HIDInputReportEvent.data;

Value

A {{domxref("DataView")}} object containing the input report.

Examples

In the following example the returned data is logged to the console.

device.addEventListener("inputreport", event => {
  const { data, device, reportId } = event;
  console.log(data);
});
