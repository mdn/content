{{ApiRef("DOM")}}{{SeeCompatTable}}

The Comment() constructor returns a newly created {{domxref("Comment")}} object with the optional {{domxref("DOMString")}} given in parameter as its textual content.

Syntax

comment1 = new Comment(); // Create an empty comment
comment2 = new Comment("This is a comment");

Example

var comment = new Comment("Test");
