{{APIRef("File System API")}}{{SeeCompatTable}}

The read-only name property of the {{domxref("FileSystem")}} interface indicates the file system's name. This {{domxref("USVString")}} is unique among all file systems currently exposed by the File and Directory Entries API.

Syntax

var fsName = FileSystem.name;

Value

A {{domxref("USVString")}} representing the file system's name.
