{{ APIRef("Touch Events") }}

Summary

A {{jsxref("Boolean")}} value indicating whether the control (Control) key is enabled when the touch event is created. If this key is enabled, the attribute's value is true. Otherwise, it is false.

This property is {{readonlyInline}}.

Syntax

var ctrlEnabled = touchEvent.ctrlKey;

Return value

ctrlEnabled
true if the control key is enabled for this event; and false if the control is not enabled.

Example

The TouchEvent.altKey example includes an example of this property's usage.
