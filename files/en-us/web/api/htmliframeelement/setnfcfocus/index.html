{{APIRef("HTML DOM")}}{{non-standard_header}}

The setNfcFocus() method of the {{domxref("HTMLIFrameElement")}} interface sets whether an {{htmlelement("iframe")}} can receive an NFC event.

Syntax

myIFrame.setNfcFocus(true);

Returns

Void.

Parameters

A {{domxref("Boolean")}} indicating whether the {{htmlelement("iframe")}} can receive an NFC event. Default is false; set to true to allow it to receive NFC events.

Specifications

Not part of any specification.
