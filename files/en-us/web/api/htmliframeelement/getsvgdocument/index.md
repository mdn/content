{{APIRef("HTML DOM")}}

The **`getSVGDocument()`** method of the {{domxref("HTMLIframeElement")}} interface returns the {{domxref("Document")}} object of the embedded SVG.

## Value

A {{domxref("Document")}}.

## Examples

```js
const svgDoc = document.getElementById("el").getSVGDocument();
```
