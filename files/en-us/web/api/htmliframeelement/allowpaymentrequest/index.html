The allowPaymentRequest property of the {{domxref("HTMLIFrameElement")}} interface returns a {{domxref("Boolean")}} indicating whether the Payment Request API may be invoked on a cross-origin iframe.

Syntax

var allow = htmlIFrameElement.allowPaymentRequest

Value

A {{jsxref("Boolean")}}.
