{{APIRef("HTML DOM")}}

The **`height`** property of the {{domxref("HTMLIFrameElement")}} interface returns a string that reflects the [`height`](/en-US/docs/Web/HTML/Element/iframe#height) HTML attribute, indicating the height of the frame.

## Value

A string.
