The port property of the {{domxref("URLPattern")}} interface is a {{domxref("USVString")}} containing the pattern used to match the port part of a URL.

{{AvailableInWorkers}}

Syntax

const string = urlpattern.port

Value

A {{domxref("USVString")}}.

Examples

const pattern = new URLPattern({ port: "(80|443|8080)"});
console.log(pattern.port); // '(80|443|8080)'
