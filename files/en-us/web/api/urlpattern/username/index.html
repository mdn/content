The username property of the {{domxref("URLPattern")}} interface is a {{domxref("USVString")}} containing the pattern used to match the username part of a URL.

{{AvailableInWorkers}}

Syntax

const string = urlpattern.username

Value

A {{domxref("USVString")}}.

Examples

const pattern = new URLPattern({ username: 'admin' });
console.log(pattern.username); // 'admin'
