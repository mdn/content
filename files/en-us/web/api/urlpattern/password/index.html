The password property of the {{domxref("URLPattern")}} interface is a {{domxref("USVString")}} containing the pattern used to match the password part of a URL. This value may be differ from the input to the constructor due to normalization.

{{AvailableInWorkers}}

Syntax

const string = urlpattern.password

Value

A {{domxref("USVString")}}.

Examples

const pattern = new URLPattern({ password: 'correct-horse-battery{-staple}?' });
console.log(pattern.password); // 'correct-horse-battery{-staple}?'
