The hash property of the {{domxref("URLPattern")}} interface is a {{domxref("USVString")}} containing the pattern used to match the fragment part of a URL.

{{AvailableInWorkers}}

Syntax

const string = urlpattern.hash

Value

A {{domxref("USVString")}}.

Examples

const pattern = new URLPattern('https://example.org#books/:id');
console.log(pattern.hash); // 'books/:id'
