The hostname property of the {{domxref("URLPattern")}} interface is a {{domxref("USVString")}} containing the pattern used to match the hostname part of a URL. This value may be differ from the input to the constructor due to normalization.

{{AvailableInWorkers}}

Syntax

const string = urlpattern.hostname

Value

A {{domxref("USVString")}}.

Examples

const pattern = new URLPattern('https://*.example.org');
console.log(pattern.hostname); // '*.example.org'
