The pathname property of the {{domxref("URLPattern")}} interface is a {{domxref("USVString")}} containing the pattern used to match the pathname part of a URL.

{{AvailableInWorkers}}

Syntax

const string = urlpattern.pathname

Value

A {{domxref("USVString")}}.

Examples

const pattern = new URLPattern({ pathname: '/books/:id(\\d)' });
console.log(pattern.pathname); // '/books/:id(\\d)'
