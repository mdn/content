The search property of the {{domxref("URLPattern")}} interface is a {{domxref("USVString")}} containing the pattern used to match the search part of a URL.

{{AvailableInWorkers}}

Syntax

const string = urlpattern.search

Value

A {{domxref("USVString")}}.

Examples

const pattern = new URLPattern("https://example.com?*");
console.log(pattern.search); // '*'
