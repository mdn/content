{{ apiref("CSSOM") }} {{SeeCompatTable}}

The TransitionEvent.pseudoElement read-only property is a {{domxref("DOMString")}}, starting with '::', containing the name of the pseudo-element the animation runs on. If the transition doesn't run on a pseudo-element but on the element, an empty string: ''.

Syntax

name = TransitionEvent.pseudoElement
