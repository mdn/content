{{ apiref("CSSOM") }} {{SeeCompatTable}}

The TransitionEvent.elapsedTime read-only property is a float giving the amount of time the animation has been running, in seconds, when this event fired. This value is not affected by the {{cssxref("transition-delay")}} property.

Syntax

name = TransitionEvent.elapsedTime
