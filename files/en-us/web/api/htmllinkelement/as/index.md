{{APIRef("HTML DOM")}}

The **`as`** property of the {{domxref("HTMLLinkElement")}}
interface returns a string representing the type of content being
loaded by the HTML link, one of `"script"`, `"style"`,
`"image"`, `"video"`, `"audio"`, `"track"`,
`"font"`, `"fetch"`.

## Value

A string.
