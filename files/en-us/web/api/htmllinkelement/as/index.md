<p>The <strong><code>as</code></strong> property of the {{domxref("HTMLLinkElement")}}
  interface returns a {{domxref("DOMString")}} representing the type of content being
  loaded by the HTML link, one of <code>"script"</code>, <code>"style"</code>,
  <code>"image"</code>, <code>"video"</code>, <code>"audio"</code>, <code>"track"</code>,
  <code>"font"</code>, <code>"fetch"</code>.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js">var <em>as</em> = HTMLLinkElement.<em>as</em>
HTMLLinkElement.<em>as</em> = <em>as</em></pre>

<h3 id="Value">Value</h3>

<p>A {{domxref("DOMString")}}.</p>
