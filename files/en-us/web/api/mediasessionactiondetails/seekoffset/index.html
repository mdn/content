{{APIRef("Media Session API")}}

The {{domxref("MediaSessionActionDetails")}} dictionary's seekOffset property is an optional value passed into the action handler callback to provide the number of seconds the seekforward and seekbackward actions should move the playback time by.

Syntax

let mediaSessionActionDetails = { seekOffset: deltaTimeInSeconds };

let deltaTime = mediaSessionActionDetails.seekOffset;

Value

A floating-point value indicating the time delta in seconds by which to move the playback position relative to its current timestamp. If the offset isn't specified, the {{Glossary("user agent")}} will choose an appropriate offset automatically. This is typically in the range of five to ten seconds.
