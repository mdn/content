{{APIRef("Media Session API")}}

The action property is the only required property {{domxref("MediaSessionActionDetails")}} dictionary, specifying the type of media session action which the action handler callback is being executed for.

Syntax

let mediaSessionActionDetails = { action: actionType };

let actionType = mediaSessionActionDetails.action;

Value

A {{domxref("DOMString")}} specifying which of the action types the callback is being invoked for:
