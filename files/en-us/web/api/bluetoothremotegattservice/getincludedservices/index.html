The BluetoothGATTService.getIncludedServices() method returns a {{jsxref("Promise")}} to an {{domxref("Array")}} of {{domxref("BluetoothRemoteGATTService")}} instances for an optional universally unique identifier (UUID).

Syntax

bluetoothGATTServiceInstance.getIncludedService(service).then(function(BluetoothGATTService) { ... } )

Returns

A {{jsxref("Promise")}} to an instance of {{domxref("BluetoothRemoteGATTService")}}.

Properties

service
The UUID of a Bluetooth service.
