The BluetoothGATTService.isPrimary read-only property returns a {{jsxref("Boolean")}} that indicates whether this is a primary service. If it is not a primary service, it is a secondary service.

Syntax

var isPrimary = BluetoothGATTService.isPrimary

Returns

A {{jsxref("Boolean")}}.
