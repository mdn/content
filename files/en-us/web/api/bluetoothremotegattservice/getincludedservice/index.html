The BluetoothGATTService.getIncludedService() method returns a {{jsxref("Promise")}} to an instance of {{domxref("BluetoothRemoteGATTService")}} for a given universally unique identifier (UUID).

Syntax

bluetoothGATTServiceInstance.getIncludedService(service).then(function(BluetoothGATTService) { ... } )

Returns

A {{jsxref("Promise")}} to an instance of {{domxref("BluetoothRemoteGATTService")}}.

Properties

service
The UUID of a Bluetooth service.
