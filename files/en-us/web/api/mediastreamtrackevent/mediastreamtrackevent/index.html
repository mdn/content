{{APIRef("Media Streams API")}}

The MediaStreamTrackEvent() constructor returns a newly created {{domxref("MediaStreamTrackEvent")}} object, which represents an event announcing that a {{domxref("MediaStreamTrack")}} has been added to or removed from a {{domxref("MediaStream")}}.

Syntax

var trackEvent = new MediaStreamTrackEvent(type, {track: aMediaStreamTrack});

Parameters

The MediaStreamTrackEvent() constructor also inherits arguments from {{domxref("Event.Event", "Event()")}}.

type
A {{domxref("DOMString")}} representing the name of the type of the MediaStreamTrackEvent. It is case-sensitive and can be "{{event("addtrack")}}" or "{{event("removetrack")}}".
track
A {{domxref("MediaStreamTrack")}} object representing the track which was added to or removed from the stream.

Return value

A new {{domxref("MediaStreamTrackEvent")}}, initialized based on the provided options.
