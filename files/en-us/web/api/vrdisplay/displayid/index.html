The displayId read-only property of the {{domxref("VRDisplay")}} interface returns an identifier for this particular VRDisplay, which is also used as an association point in the Gamepad API (see {{domxref("Gamepad.displayId")}}).

Syntax

var myDisplayID = vrDisplayInstance.displayId;

Value

A number representing the ID of the specific VRDisplay.
