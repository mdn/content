<p>The <strong><code>brands</code></strong> read-only property of the {{domxref("NavigatorUAData")}} interface returns an array of brand information.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js">let brands = NavigatorUAData.brands;</pre>

<h3>Value</h3>
<p>An array containing the following information for each brand:</p>

<dl>
  <dt>brand</dt>
  <dd>A {{domxref("DOMString","string")}} containing the brand. For example, <code>"Google Chrome"</code>.</dd>
  <dt>version</dt>
  <dd>A {{domxref("DOMString","string")}} containing the version. For example, <code>"91"</code>.</dd>
</dl>

<h2 id="Examples">Examples</h2>

<p>The following example prints the value of {{domxref("NavigatorUAData.brands")}} to the console.</p>

<pre class="brush:js">console.log(navigator.userAgentData.brands);</pre>
