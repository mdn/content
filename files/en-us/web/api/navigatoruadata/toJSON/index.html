The toJSON() method of the {{domxref("NavigatorUAData")}} interface is a serializer that returns a JSON representation of the low entropy properties of the NavigatorUAData object.

Note

The terms high entropy and low entropy refer to the amount of information these values reveal about the browser. The values available as JSON, are the same as those that can be returned using the properties of the {{domxref("NavigatorUAData")}} interface.

Syntax

NavigatorUAData.toJSON();

Parameters

None.

Return value

A JSON object.

Examples

The following example prints the JSON object to the console.

console.log(navigator.userAgentData.toJSON());
