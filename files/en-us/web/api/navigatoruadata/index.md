<div>{{DefaultAPISidebar("User-Agent Client Hints API")}}</div>

<p>The <strong><code>NavigatorUAData</code></strong> interface of the {{domxref('User-Agent Client Hints API')}} returns information about the browser and operating system of a user.</p>

<p>An instance of this object is returned by calling {{domxref("Navigator.userAgentData")}}. Therefore, this interface has no constructor.</p>

<div class="notecard note">
  <p><strong>Note:</strong> The terms <em>high entropy</em> and <em>low entropy</em> refer to the amount of information these values reveal about the browser. The values returned as properties are deemed low entropy, and unlikely to identify a user. The values returned by {{domxref("NavigatorUAData.getHighEntropyValues()")}} could potentially reveal more information. These values are therefore retrieved via a {{jsxref("Promise")}}, allowing time for the browser to request user permission, or make other checks.</p>
</div>

<h2 id="Properties">Properties</h2>

<dl>
  <dt>{{domxref("NavigatorUAData.brands")}}{{ReadOnlyInline}}</dt>
  <dd>Returns an array of brand information containing the browser name and version.</dd>
  <dt>{{domxref("NavigatorUAData.mobile")}}{{ReadOnlyInline}}</dt>
  <dd>Returns <code>true</code> if the user-agent is running on a mobile device.</dd>
  <dt>{{domxref("NavigatorUAData.platform")}}{{ReadOnlyInline}}</dt>
  <dd>Returns the platform brand the user-agent is running on.</dd>
</dl>

<h2 id="Methods">Methods</h2>

<dl>
  <dt>{{domxref("NavigatorUAData.getHighEntropyValues()")}}</dt>
  <dd>Returns a {{jsxref("Promise")}} that resolves with a dictionary object containing the <em>high entropy</em> values the user-agent returns.</dd>
  <dt>{{domxref("NavigatorUAData.toJSON()")}}</dt>
  <dd>A <em>serializer</em> that returns a JSON representation of the <em>low entropy</em> properties of the <code>NavigatorUAData</code> object.</dd>
</dl>

<h2 id="Examples">Examples</h2>

<h3>Getting the brands</h3>

<p>The following example prints the value of {{domxref("NavigatorUAData.brands")}} to the console.</p>

<pre class="brush:js">console.log(navigator.userAgentData.brands);</pre>

<h3>Returning high entropy values</h3>

<p>In the following value a number of hints are requested using the {{domxref("NavigatorUAData.getHighEntropyValues()")}} method. When the promise resolves, this information is printed to the console.</p>

<pre class="brush:js">navigator.userAgentData.getHighEntropyValues(
  ["architecture",
  "model",
  "platform",
  "platformVersion",
  "uaFullVersion"])
  .then(ua => { console.log(ua) });</pre>
