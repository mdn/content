The SVGSymbolElement interface corresponds to the {{SVGElement("symbol")}} element.

{{InheritanceDiagram(600, 140)}}

Properties

This interface doesn't implement any specific properties, but inherits properties from its parent interface, {{domxref("SVGGraphicsElement")}}, and implements properties from {{domxref("SVGFitToViewBox")}}.

Methods

This interface doesn't implement any specific methods, but inherits methods from its parent interface, {{domxref("SVGGraphicsElement")}}, and implements methods from {{domxref("SVGFitToViewBox")}}.
