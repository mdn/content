The **`close()`** method of the {{domxref("AudioDecoder")}} interface ends all pending work and releases system resources.

## Syntax

```js
AudioDecoder.close()
```

### Parameters

None.

### Return Value

{{jsxref('undefined')}}.

## Examples

The following example closes the `AudioDecoder`.

```js
AudioDecoder.close();
```
