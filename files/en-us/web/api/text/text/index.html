The Text() constructor returns a newly created {{domxref("Text")}} object with the optional {{domxref("DOMString")}} given in parameter as its textual content.

Syntax

text1 = new Text(); // Create an empty text node
text2 = new Text("This is a text node");

Example

let text = new Text("Test");
