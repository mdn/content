<p>The <code><strong>Text()</strong></code> constructor returns a newly created
  {{domxref("Text")}} object with the optional {{domxref("DOMString")}} given in parameter
  as its textual content.</p>

<h2 id="Syntax">Syntax</h2>

<pre class="brush: js"><em>text1</em> = new Text(); // Create an empty text node
<em>text2</em> = new Text("This is a text node");
</pre>

<h2 id="Example">Example</h2>

<pre
  class="brush: js">let text = new Text(&quot;Test&quot;);</pre>
