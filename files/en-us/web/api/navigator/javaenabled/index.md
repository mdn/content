{{ APIRef("HTML DOM") }}

This method returns true if Java is enabled in the browser, otherwise it returns false.

## Syntax

```js
result = window.navigator.javaEnabled()
```

## Example

    if (window.navigator.javaEnabled()) {
       // code will only be executed if java is enabled
    }
