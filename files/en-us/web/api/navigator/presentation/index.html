The presentation property of {{DOMxRef("Navigator")}} serves as the entry point for the Presentation API and returns a reference to {{DOMxRef("Presentation")}} object.

Syntax

const presentation = navigator.presentation

Value

A reference to {{DOMxRef("Presentation")}} object.
