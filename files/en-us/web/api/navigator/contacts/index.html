The contacts read-only property of the {{domxref("Navigator")}} interface returns a {{domxref('ContactsManager')}} interface which allows users to select entries from their contact list and share limited details of the selected entries with a website or application.

Syntax

var contactsManager = navigator.contacts;

Return Value

{{domxref('ContactsManager')}}

Examples

The following code checks whether the Contact Picker API is supported.

const supported = ('contacts' in navigator && 'ContactsManager' in window);
