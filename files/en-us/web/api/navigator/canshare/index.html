The Navigator.canShare() method of the Web Share API returns true if a call to {{domxref("navigator.share()")}} would succeed.

The Web Share API is gated by the web-share permission (and corresponding web-share permission policy). The canShare() method will return false if the permission is supported on the platform but has not been granted.

Syntax

navigator.canShare()
navigator.canShare(data)

Parameters

data {{optional_inline}}
An object containing data to share that matches what you would pass to {{domxref("navigator.share()")}}.

Return value

The value true if the specified data can be shared with {{domxref("Navigator.share()")}}, otherwise false.

Examples

The example checks whether navigator.canShare() is defined, and if so uses it to check whether navigator.share() can share the specified data.

Note:The separate check for whether navigator.canShare() is defined is needed because "if (navigator.canShare)" evaluates to false in this case, even if the navigator.share() method might be supported and otherwise work.

HTML

The HTML just creates a paragraph in which to display the result of the test.

<p class="result"></p>

JavaScript

let shareData = {
  title: 'MDN',
  text: 'Learn web development on MDN!',
  url: 'https://developer.mozilla.org',
}

const resultPara = document.querySelector('.result');

if (typeof navigator.canShare === 'undefined') {
  resultPara.textContent = 'navigator.canShare() not supported.';
}
else if (navigator.canShare && navigator.canShare(shareData)) {
  resultPara.textContent = 'navigator.canShare() supported. We can use navigator.share() to send the data.';
} else {
  resultPara.textContent = 'navigator.canShare() supported. Specified data cannot be shared.';
}
