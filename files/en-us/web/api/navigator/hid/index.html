{{APIRef("WebHID API")}}{{SeeCompatTable}}

The Navigator.hid read-only property returns an {{domxref("HID")}} object providing methods for connecting to HID devices, listing attached HID devices, and event handlers for connected HID devices. This could be used to connect to HID devices and get information about them and their events.

Syntax

var hidVar = navigator.hid

Value

An {{domxref("HID")}} object.
