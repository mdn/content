{{APIRef("Intersection Observer API")}}

The **`disconnect()`** method of the {{domxref("IntersectionObserver")}} interface stops the observer watching all of its target elements for visibility changes.

## Syntax

```js-nolint
disconnect()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).
