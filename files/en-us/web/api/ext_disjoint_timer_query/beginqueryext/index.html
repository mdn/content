{{APIRef("WebGL")}}

The EXT_disjoint_timer_query.beginQueryEXT() method of the WebGL API starts a timer query.

Syntax

void ext.beginQueryEXT(target, query);

Parameters

target
A {{domxref("GLenum")}} specifying the target of the time query. Must be ext.TIME_ELAPSED_EXT.
query
A {{domxref("WebGLTimerQueryEXT")}} object for which to start the time querying.

Return value

None.

Examples

var ext = gl.getExtension('EXT_disjoint_timer_query');
var query = ext.createQueryEXT();
ext.beginQueryEXT(ext.TIME_ELAPSED_EXT, query);

// ...
