{{APIRef("WebGL")}}

The EXT_disjoint_timer_query.createQueryEXT() method of the WebGL API creates and initializes {{domxref("WebGLTimerQueryEXT")}} objects, which track the time needed to fully complete a set of GL commands.

Syntax

WebGLTimerQueryEXT ext.createQueryEXT();

Parameters

None.

Return value

A {{domxref("WebGLTimerQueryEXT")}} object.

Examples

var ext = gl.getExtension('EXT_disjoint_timer_query');
var query = ext.createQueryExt();
