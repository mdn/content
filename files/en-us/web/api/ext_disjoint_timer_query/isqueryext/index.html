{{APIRef("WebGL")}}

The EXT_disjoint_timer_query.isQueryEXT() method of the WebGL API returns true if the passed object is a {{domxref("WebGLTimerQueryEXT")}} object.

Syntax

GLBoolean ext.isQueryEXT(query);

Parameters

query
A {{domxref("WebGLTimerQueryEXT")}} object to test.

Return value

A {{domxref("GLboolean")}} indicating whether the given object is a {{domxref("WebGLTimerQueryEXT")}} object (true) or not (false).

Examples

var ext = gl.getExtension('EXT_disjoint_timer_query');
var query = ext.createQueryEXT();

// ...

ext.isQueryEXT(query);
