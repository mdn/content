The AnimationEvent.elapsedTime read-only property is a float giving the amount of time the animation has been running, in seconds, when this event fired, excluding any time the animation was paused. For an {{domxref("HTMLElement/animationstart_event", "animationstart")}} event, elapsedTime is 0.0 unless there was a negative value for animation-delay, in which case the event will be fired with elapsedTime containing (-1 * delay).

Syntax

time = AnimationEvent.elapsedTime
