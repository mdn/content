{{APIRef("Touch Events")}}{{deprecated_header}}

The identifiedTouch() method returns the first {{ domxref("Touch") }} item in the {{ domxref("TouchList") }} that matches the specified identifier.

It is recommended that you use {{domxref("TouchList.item()")}} instead.

Syntax

var touchItem = touchList.identifiedTouch(id);

Parameters

id
An integer value identifying the {{ domxref("Touch") }} object to retrieve from the list.

Return value

touchItem
A {{ domxref("Touch") }} object matching the specified id.

Specifications

Not part of any specification. This API was removed from the Touch Events v2 draft specification.
