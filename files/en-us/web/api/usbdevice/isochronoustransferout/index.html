{{APIRef("WebUSB API")}}{{SeeCompatTable}}

The isochronousTransferOut() method of the {{domxref("USBDevice")}} interface returns a {{jsxref("promise")}} that resolves with a {{domxref("USBIsochronousOutTransferResult")}} when time sensitive information has been transmitted to the USB device.

Syntax

var promise = USBDevice.isochronousTransferOut(endpointNumber, data, packetLengths)

Parameters

endpointNumber
The number of a device-specific endpoint (buffer).
data
A {{domxref("TypedArray")}} containing the data to send to the device.
packetLengths
An array of lengths for the packets being transfered.

Return value

A {{jsxref("promise")}} that resolves with a {{domxref("USBIsochronousOutTransferResult")}}.
