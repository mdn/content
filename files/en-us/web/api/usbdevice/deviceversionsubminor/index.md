The **`deviceVersionSubminor`** read only property of the
{{domxref("USBDevice")}} interface the patch version number of the device in a semantic
versioning scheme.

## Syntax

```js
var serialNumber = USBDevice.deviceVersionSubminor
```

### Value

A number.
