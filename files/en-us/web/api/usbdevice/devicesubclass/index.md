<p>{{SeeCompatTable}}{{APIRef("WebUSB API")}}</p>

<p>The <strong><code>deviceSubclass</code></strong> read only property of the
  {{domxref("USBDevice")}} interface one of three properties that identify USB devices for
  the purpose of loading a USB driver that will work with that device. The other two
  properties are USBDevice.deviceClass and USBDevice.deviceProtocol.</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js">var serialNumber = USBDevice.deviceSubclass</pre>

<h3 id="Value">Value</h3>

<p>A number.</p>
