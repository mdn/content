The **`open()`** method of the {{domxref("USBDevice")}}
interface returns a {{jsxref("promise")}} that resolves when a device session has
started.

## Syntax

```js-nolint
open()
```

### Parameters

None.

### Return value

A {{jsxref("promise")}}.
