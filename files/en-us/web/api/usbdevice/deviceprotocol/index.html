{{SeeCompatTable}}{{APIRef("WebUSB API")}}

The deviceProtocol read only property of the {{domxref("USBDevice")}} interface one of three properties that identify USB devices for the purpose of loading a USB driver that will work with that device. The other two properties are USBDevice.deviceClass and USBDevice.deviceSubclass.

Syntax

var number = USBDevice.deviceProtocol

Value

A number.
