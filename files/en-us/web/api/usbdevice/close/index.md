The **`close()`** method of the {{domxref("USBDevice")}}
interface returns a {{jsxref("promise")}} that resolves when all open interfaces are
released and the device session has ended.

## Syntax

```js
var promise = USBDevice.close()
```

### Parameters

None.

### Return value

A {{jsxref("promise")}}.
