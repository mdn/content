The releaseInterface() method of the {{domxref("USBDevice")}} interface returns a {{jsxref("promise")}} that resolves when a cliamed interface is released from exclusive access.

Syntax

var promise = USBDevice.releaseInterface(interfaceNumber)

Parameters

interfaceNumber
The device-specific index of the currently-claimed interface.

Return value

A {{jsxref("promise")}}.
