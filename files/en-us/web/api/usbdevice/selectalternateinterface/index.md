<p>The <strong><code>selectAlternateInterface()</code></strong> method of the
  {{domxref("USBDevice")}} interface returns a {{jsxref("promise")}} that resolves when
  the specified alternative endpoint is selected.</p>

<h2 id="Syntax">Syntax</h2>

<pre
  class="brush: js">var promise = USBDevice.selectAlternateInterface(<em>interfaceNumber</em>, <em>alternateSetting</em>)</pre>

<h3 id="Parameters">Parameters</h3>

<dl>
  <dt>interfaceNumber</dt>
  <dd>The index of one of the interfaces supported by the device. Interfaces are
    device-specific.</dd>
  <dt>alternateSetting</dt>
  <dd>The configuration of the selected interface.</dd>
</dl>

<h3 id="Return_value">Return value</h3>

<p>A {{jsxref("promise")}}.</p>
