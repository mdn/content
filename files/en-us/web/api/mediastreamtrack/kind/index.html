{{APIRef("Media Capture and Streams")}}

The MediaStreamTrack.kind read-only property returns a {{domxref("DOMString")}} set to "audio" if the track is an audio track and to "video", if it is a video track. It doesn't change if the track is deassociated from its source.

Syntax

const type = track.kind

Value

The possible values are a {{domxref("DOMString")}} with on of the following values:

  • "audio": the track is an audio track.
  • "video": the track is a video track.
