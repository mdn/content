The MediaStreamTrack.remote read-only property allows Javascript to know whether a WebRTC MediaStreamTrack is from a remote source or a local one. It returns a {{domxref("Boolean")}} with a value of true if the track is sourced remotely (that is, sourced by an {{domxref("RTCPeerConnection")}}), or false if it is sourced locally.

Syntax

var bool = track.remote;
