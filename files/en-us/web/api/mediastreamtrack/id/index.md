{{APIRef("Media Capture and Streams")}}

The **`MediaStreamTrack.id`** read-only property returns a
string containing a unique identifier (GUID) for the track, which is
generated by the {{glossary("user agent")}}.

## Syntax

```js
const id = track.id
```
