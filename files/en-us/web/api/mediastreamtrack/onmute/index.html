{{domxref("MediaStreamTrack")}}'s onmute event handler is called when the {{event("mute")}} event is received. Such an event is sent when the track is temporarily not able to send data.

Syntax

track.onmute = muteHandler;

Value

A function to serve as an {{event("Event_handlers", "event handler")}} for the {{event("mute")}} event. The event handler function receives a single parameter: the event object, which is a simple {{domxref("Event")}} object.

Example

In this example, an onmute handler is established to set the content HTML of an element to display the "muted speaker" Emoji.

myTrack.onmute = function(evt) {
  playStateIcon.innerHTML = "&#1F507;";
};
