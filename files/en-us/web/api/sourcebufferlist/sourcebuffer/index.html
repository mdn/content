The indexed property getter of the {{domxref("SourceBufferList")}} interface allows the SourceBuffer objects in the list to be accessed with an array operator (i.e. [].)

Syntax

var mySourceBuffer = sourceBufferList[index];

Parameters

index
The index position of the {{domxref("SourceBuffer")}} object you want to return.

Return value

A {{domxref("SourceBuffer")}} object.

Exceptions

No specific exceptions are thrown, but if the supplied index is great than or equal to {{domxref("SourceBufferList.length")}}, the operation will return undefined.
