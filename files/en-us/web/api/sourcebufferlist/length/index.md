The **`length`** read-only property of the
{{domxref("SourceBufferList")}} interface returns the number of
{{domxref("SourceBuffer")}} objects in the list.

## Syntax

```js
var myListLength = sourceBufferList.length;
```

### Value

An unsigned long number.
