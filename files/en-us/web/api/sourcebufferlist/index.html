The SourceBufferList interface represents a simple container list for multiple {{domxref("SourceBuffer")}} objects.

The source buffer list containing the SourceBuffers appended to a particular MediaSource can be retrieved using the {{domxref("MediaSource.sourceBuffers")}} property.Â The individual source buffers can be accessed using array operator (i.e. []) or functions such as {{jsxref("Array.forEach", "forEach()")}} for example.

Methods

Inherits methods from its parent interface, {{domxref("EventTarget")}}.

SourceBufferList: indexed property getter
This getter allows the SourceBuffer objects in the list to be accessed with an array operator (i.e. [].)
