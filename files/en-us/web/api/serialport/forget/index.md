The **`SerialPort.forget()`** method of the {{domxref("SerialPort")}} interface returns a {{jsxref("Promise")}} that resolves when the port is closed and is forgotten.

## Syntax

```js
SerialPort.forget()
```

### Parameters

None.

### Return value

A {{jsxref("Promise")}}.
