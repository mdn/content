{{APIRef("EncryptedMediaExtensions API")}}

The **`values`** property of the
{{domxref("MediaKeyStatusMap")}} interface returns a new Iterator object, containing
values for each element in the status map, in insertion order.

## Syntax

```js-nolint
values()
```

### Parameters

None.

### Return value

A new iterator.
