The keys property of the {{domxref("MediaKeyStatusMap")}} interface returns a new Iterator object, containing keys for each element in the status map, in insertion order.

Syntax

var iterator = mediaKeyStatusMap.keys()

Parameters

None.

Returns

A new iterator.
