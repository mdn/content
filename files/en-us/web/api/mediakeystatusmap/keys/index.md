{{APIRef("EncryptedMediaExtensions API")}}

The **`keys`** property of the
{{domxref("MediaKeyStatusMap")}} interface returns a new Iterator object, containing
keys for each element in the status map, in insertion order.

## Syntax

```js
keys()
```

### Parameters

None.

### Return value

A new iterator.
