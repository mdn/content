The read-only media property of the {{domxref("CSSMediaRule")}} interface {{domxref("MediaList")}} represents the intended destination medium for style information.

Syntax

var media = CSSMediaRule.media;

Value

a {{domxref("MediaList")}}

Examples

The CSS includes a media query with one style rule. This will be the first {{domxref("CSSRule")}} returned by document.styleSheets[0].cssRules. Calling myRules[0].media therefore returns a {{domxref("MediaList")}} object representing the media query. 

@media (min-width: 500px) {
  body {
    color: blue;
  }
}
let myRules = document.styleSheets[0].cssRules;
console.log(myRules[0].media); // a MediaList
