{{ APIRef("Pointer Events") }}

The tiltX read-only property of the {{domxref("PointerEvent")}} interface is the angle (in degrees) between the Y-Z plane of the pointer and the screen. This property is typically only useful for a pen/stylus pointer type.

For an illustration of this property see Figure 2 in the specification.

Syntax

var tiltX = pointerEvent.tiltX;

Return value

tiltX
The angle in degrees between the Y-Z plane of the pointer (stylus) and the screen. The range of values is -90 to 90, inclusive, where a positive value is a tilt to the right. For devices that do not support this property, the value is 0.

Example

This example illustrates simple accessing of the tiltX and {{domxref("PointerEvent.tiltY","tiltY")}} properties.

someElement.addEventListener("pointerdown", function(event) {
  process_tilt(event.tiltX, event.tiltY);
}, false);
