{{APIRef("File System API")}}{{SeeCompatTable}}

The exclusive property on the {{domxref("FileSystemFlags")}} dictionary is used in tandem with the create property to determine whether or not it's acceptable to require that the file not already exist when the reference to it is created by calling {{domxref("FileSystemDirectoryEntry.getFile()")}} or {{domxref("FileSystemDirectoryEntry.getDirectory()")}}.

Syntax

fileSystemFlags.exclusive = booleanValue
