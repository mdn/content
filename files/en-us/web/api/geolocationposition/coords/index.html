The GeolocationPosition.coords read-only property returns a {{domxref("GeolocationCoordinates")}} object representing a geographic position. It contains the location, that is longitude and latitude on the Earth, the altitude, and the speed of the object concerned, regrouped inside the returned value. It also contains accuracy information about these values.

Syntax

let coord = geolocationPositionInstance.coords

Value

A {{domxref("GeolocationCoordinates")}} object instance.
