{{APIRef("Service Workers API")}}

The waiting property of the {{domxref("ServiceWorkerRegistration")}} interface returns a service worker whose {{domxref("ServiceWorker.state")}} is installed. This property is initially set to null.

Note: This feature is available in Web Workers.

Syntax

var serviceWorker = serviceWorkerRegistration.waiting;

Value

A {{domxref("ServiceWorker")}} object, if it is currently in an installed state.
