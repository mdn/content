{{APIRef("Service Workers API")}}

The installing property of the {{domxref("ServiceWorkerRegistration")}} interface returns a service worker whose {{domxref("ServiceWorker.state")}} is installing. This property is initially set to null.

Note: This feature is available in Web Workers.

Syntax

var serviceWorker = serviceWorkerRegistration.installing;

Value

A {{domxref("ServiceWorker")}} object, if it is currently in an installing state.
