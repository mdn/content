{{APIRef("WebRTC")}}

The remoteId property of the {{domxref("RTCOutboundRtpStreamStats")}} dictionary specifies the {{domxref("RTCStats.id", "id")}} of the {{domxref("RTCRemoteInboundRtpStreamStats")}} object representing the remote peer's {{domxref("RTCRtpReceiver")}} which is sending the media to the local peer for this SSRC.

Syntax

var remoteStatsId = RTCOutboundRtpStreamStats.remoteId;

Value

A {{domxref("DOMString")}} containing the ID of the {{domxref("RTCRemoteInboundRtpStreamStats")}} object that represents the remote peer's {{domxref("RTCRtpReceiver")}} for the synchronization source represented by this stats object.
