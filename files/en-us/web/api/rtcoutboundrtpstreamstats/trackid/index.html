{{APIRef("WebRTC")}}

The trackId property of the {{domxref("RTCOutboundRtpStreamStats")}} dictionary indicates the {{domxref("RTCStats.id", "id")}} of the {{domxref("RTCSenderAudioTrackAttachmentStats")}} or {{domxref("RTCSenderVideoTrackAttachmentStats")}} object representing the {{domxref("MediaStreamTrack")}} which is being sent on this stream.

Syntax

var trackStatsId = RTCOutboundRtpStreamStats.trackId;

Value

A {{domxref("DOMString")}} containing the ID of the {{domxref("RTCSenderAudioTrackAttachmentStats")}} or {{domxref("RTCSenderVideoTrackAttachmentStats")}} object representing the track which is the source of the media being sent on this stream.
