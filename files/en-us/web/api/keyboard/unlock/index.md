{{APIRef("Keyboard API")}}{{SeeCompatTable}}

The **`unlock()`** method of the
{{domxref("Keyboard")}} interface unlocks all keys captured by the
{{domxref('Keyboard.lock()')}} method and returns synchronously.

## Syntax

```js
Keyboard.unlock()
```

### Parameters

None.

### Return value

{{jsxref('undefined')}}
