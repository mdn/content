The HTMLMenuElement interface provides special properties (beyond those defined on the regular {{DOMxRef("HTMLElement")}} interface it also has available to it by inheritance) for manipulating {{HTMLElement("menu")}} elements.

{{InheritanceDiagram(600, 120)}}

Constructor

{{DOMxRef("HTMLMenuElement.HTMLMenuElement", "HTMLMenuElement()")}}
Returns a newly constructed HTMLMenuElement.

Properties

Inherits properties from its parent, {{domxref("HTMLElement")}}, and its ancestors.

{{DOMxRef("HTMLMenuElement.compact", "compact")}} {{deprecated_inline}}
Is a Boolean value determining if the menu displays in a compact way.
{{DOMxRef("HTMLMenuElement.type", "type")}} {{Non-Standard_Inline}}
Returns context if the menu is a context menu. This use of the {{HTMLElement("menu")}} element has never been implemented widely and is now obsolete.
{{DOMxRef("HTMLMenuElement.label", "label")}} {{Non-Standard_Inline}}
A string associating the menu with a name, displayed when the menu is used as a context menu. This use of the {{HTMLElement("menu")}} element has never been implemented widely and is now obsolete.

Methods

Inherits methods from its parent, {{domxref("HTMLElement")}}, and its ancestors.

HTMLMenuElement doesn't implement specific methods.
