The inlineVerticalFieldOfView read-only property of the {{domxref("XRRenderState")}} interface defines the angle of the field of view in radians used when computing projection matrices for "inline" {{domxref("XRSession")}} objects. This option must be null for immersive sessions.

Syntax

var aDouble = XRRenderState.inlineVerticalFieldOfView;

Value

A {{jsxref("Number")}}.
