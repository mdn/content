The **`values`** read-only property of
the {{domxref("KeyboardLayoutMap")}} interface returns a new Array
Iterator object that contains the values for each index in the
map.

## Syntax

```js
var iterator = KeyboardLayoutMap.values
```

### Value

An iterator.
