The has() method of the {{domxref('KeyboardLayoutMap')}} interface returns a boolean indicating whether the object has an element with the specified key. A list of valid keys is found in the UI Events KeyboardEvent code Values spec.

Syntax

var aBoolean = KeyboardLayoutMap.has(key)

Parameters

key
The key of an element to search for in the map.

Return value

A {{jsxref('Boolean')}} indicating whether the specified key was found.
