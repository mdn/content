The write read-only property of the {{domxref("BluetoothCharacteristicProperties")}} interface returns a boolean that is true if the writing to the characteristic with response is permitted.

Syntax

var aBoolean = BluetoothCharacteristicProperties.write;

Value

A {{jsxref("Boolean")}}.
