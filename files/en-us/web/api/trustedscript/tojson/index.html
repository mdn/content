The toJSON() method of the {{domxref("TrustedScript")}} interface returns a string which may safely inserted into an injection sink.

Syntax

var json = TrustedScript.toJSON();

Return value

A {{domxref("DOMString")}} containing the value of the [[Data]] internal slot.

Examples

const sanitized = scriptPolicy.createScript("eval('2 + 2')");
  console.log(sanitized.toJSON());
