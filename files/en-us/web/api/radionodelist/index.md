<div>{{APIRef("HTML DOM")}}</div>

<p>The <strong><code>RadioNodeList</code></strong> interface represents a collection of radio elements in a {{HTMLElement("form")}} or a {{HTMLElement("fieldset")}} element.</p>

<p>{{InheritanceDiagram}}</p>

<h2 id="Properties">Properties</h2>

<p><em>The <code>RadioNodeList</code> interface inherits the properties of </em>{{domxref("NodeList")}}.</p>

<dl>
 <dt>{{domxref("RadioNodeList.value")}}</dt>
 <dd>If the underlying element collection contains radio buttons, the <code>value</code> property represents the checked radio button. On retrieving the <code>value</code> property, the <code>value</code> of the currently <code>checked</code> radio button is returned as a string. If the collection does not contain any radio buttons or none of the radio buttons in the collection is in <code>checked</code> state, the empty string is returned. On setting the <code>value</code> property, the first radio button input element whose <code>value</code> property is equal to the new value will be set to <code>checked</code>.</dd>
</dl>

<h2 id="Methods">Methods</h2>

<p><em>The <code>RadioNodeList</code> interface inherits the methods of </em>{{domxref("NodeList")}}.</p>
