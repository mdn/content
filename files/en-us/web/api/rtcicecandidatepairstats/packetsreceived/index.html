{{APIRef("WebRTC")}}

The {{domxref("RTCIceCandidatePairStats")}} dictionary's packetsReceived property indicates the total number of packets of any kind that have been received on the connection described by the pair of candidates.

The number of packets sent to date on the connection can be obtained using {{domxref("RTCIceCandidatePairStats.packetsSent", "packetsSent")}}.

Syntax

packetsReceived = rtcIceCandidatePairStats.packetsReceived;

Value

An integer value indicating the total number of packets, of any kind, which have been received on the connection described by the two candidates comprising this pair.
