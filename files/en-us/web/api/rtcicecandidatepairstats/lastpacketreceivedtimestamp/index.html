{{APIRef("WebRTC")}}

The {{domxref("RTCIceCandidatePairStats")}} property lastPacketReceivedTimestamp indicates the time at which the connection described by the candidate pair last received a packet. {{Glossary("STUN")}} packets are not included.

Syntax

lastPacketReceivedTimestamp = rtcIceCandidatePairStats.lastPacketReceivedTimestamp;

Value

A {{domxref("DOMHighResTimeStamp")}} object indicating the timestamp at which the connection described by pair of candidates last received a packet, STUN packets excluded.
