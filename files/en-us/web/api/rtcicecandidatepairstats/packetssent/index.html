{{APIRef("WebRTC")}}

The {{domxref("RTCIceCandidatePairStats")}} dictionary's packetsSent property indicates the total number of packets which have been sent on the connection described by the pair of candidates.

The number of packets received to date on the connection can be obtained using {{domxref("RTCIceCandidatePairStats.packetsReceived", "packetsReceived")}}.

Syntax

packetsSent = rtcIceCandidatePairStats.packetsSent;

Value

An integer value indicating the total number of packets, of any kind, which have been sent on the connection described by the two candidates comprising this pair.
