{{APIRef("WebRTC")}}

The {{domxref("RTCIceCandidatePairStats")}} property lastPacketSentTimestamp indicates the time at which the connection described by the candidate pair last sent a packet, not including {{Glossary("STUN")}} packets.

Syntax

lastPacketSentTimestamp = rtcIceCandidatePairStats.lastPacketSentTimestamp;

Value

A {{domxref("DOMHighResTimeStamp")}} object indicating the timestamp at which the connection described by pair of candidates last sent a packet, STUN packets excluded.
