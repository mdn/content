{{APIRef("WebRTC")}}

The {{domxref("RTCIceCandidatePairStats")}} property localCandidateId is a string that uniquely identifies the local {{Glossary("ICE")}} candidate which was analyzed to generate the {{domxref("RTCIceCandidateStats")}} used to compute the statistics for this pair of candidates.

Syntax

localCandidateId = rtcIceCandidatePairStats.localCandidateId;

Value

A {{domxref("DOMString")}} giving a unique identifier for the local {{domxref("RTCIceCandidate")}} for the connection described by this RTCIceCandidatePairStats object.

This candidate is the source of one of the two {{domxref("RTCIceCandidateStats")}} objects that were used to compute the contents of this {{domxref("RTCIceCandidatePairStats")}} object.
