The **`close()`** method of the {{domxref("VideoEncoder")}} interface ends all pending work and releases system resources.

## Syntax

```js-nolint
close()
```

### Parameters

None.

### Return value

None ({{jsxref("undefined")}}).

## Examples

The following example closes the `VideoEncoder`.

```js
VideoEncoder.close();
```
