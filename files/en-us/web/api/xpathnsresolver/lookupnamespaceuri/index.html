{{APIRef("DOM XPath")}}

The lookupNamespaceURI method looks up the namespace URI associated to the given namespace prefix within an {{Glossary("XPath")}} expression evaluated by the {{domxref("XPathEvaluator")}} interface.

Syntax

DOMString XPathNSResolver.lookupNamespaceURI(prefix);

Parameters

prefix
A {{domxref("DOMString")}} representing the prefix to look for.

Return value

A {{domxref("DOMString")}} representing the associated namespace URI or null if none is found.
