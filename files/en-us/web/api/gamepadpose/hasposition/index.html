The hasPosition read-only property of the {{domxref("GamepadPose")}} interface returns a {{domxref("Boolean")}} stating whether the {{domxref("Gamepad")}} can track and return position information.

Syntax

var hasItGotPosition = gamepadPoseInstance.hasPosition;

Value

A {{domxref("Boolean")}}.
