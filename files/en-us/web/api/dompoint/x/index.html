{{APIRef("DOM")}}

The DOMPoint interface's x property holds the horizontal coordinate, x, for a point in space. In general, positive values x mean to the right, and negative values of x means to the left, barring any transforms that may have altered the orientation of the axes.

Syntax

var xPos = DOMPoint.x;

Value

A double-precision floating-point value indicating the x coordinate's value for the point. This value is unrestricted, meaning that it is allowed to be infinite or invalid (that is, its value may be {{jsxref("NaN")}} or {{jsxref("Infinity", "Â±Infinity")}}).
