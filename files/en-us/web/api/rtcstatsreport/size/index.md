{{APIRef("WebRTC")}}

The **`size`** read-only property of the {{domxref("RTCStatsReport")}} interface returns the number of `id`/`statistics-object` (key/value) pairs in the current report.

## Value

An integer indicating the number of statistics objects in this report.
The value is zero if the report is empty.
