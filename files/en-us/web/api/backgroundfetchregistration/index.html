The BackgroundFetchRegistration interface of the {{domxref('null','','',' ')}} null

A BackgroundFetchRegistration instance is returned by the {{domxref("BackgroundFetchManager.fetch()")}} or {{domxref("BackgroundFetchManager.get()")}} methods, therefore there is no constructor.

Properties

The following properties are copies of those in the BackgroundFetchRegistration instance, in order that they may be available synchronously.

{{domxref("BackgroundFetchRegistration.id")}}{{ReadOnlyInline}}
A {{domxref("DOMString","string")}} containing a copy of the background fetch's id.
{{domxref("BackgroundFetchRegistration.uploadTotal")}}{{ReadOnlyInline}}
A {{jsxref("number")}} which initially contains a copy of the background fetch's stored body bytes total.
{{domxref("BackgroundFetchRegistration.uploaded")}}{{ReadOnlyInline}}
A {{jsxref("number")}} which contains the size in bytes that has been uploaded, initially 0.
