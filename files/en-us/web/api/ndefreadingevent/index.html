The NDEFReadingEvent interface represents events dispatched on new NFC readings obtained by {{DOMxRef("NDEFReader")}}.

Constructor

{{DOMxRef("NDEFReadingEvent.NDEFReadingEvent", "NDEFReadingEvent.NDEFReadingEvent()")}}
Creates an NDEFReadingEvent event with the given parameters.

Properties

Also inherits properties from its parent {{DOMxRef("Event")}}.

{{DOMxRef("NDEFReadingEvent.serialNumber")}} {{ReadOnlyInline}}
Represents the serial number of the device used for anti-collision and identification, or empty string in case none is available.
{{DOMxRef("NDEFReadingEvent.message")}} {{ReadOnlyInline}}
Represents the received message as an {{DOMxRef("NDEFMessage")}} object.

Inherits methods from its parent {{DOMxRef("Event")}}.
