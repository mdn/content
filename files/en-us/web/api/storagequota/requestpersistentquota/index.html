{{APIRef("Quota Management API")}}{{SeeCompatTable}}

The requestPersistentQuota() property of the {{domxref("StorageQuota")}} interface requests persistent storage for the requesting application and returns a {{jsxref('Promise')}} to an instance of {{jsxref('StorageInfo')}}. There is no guarantee that the requested space will be granted by the user agent.

Syntax

StorageQuota.requestPersistentQuota().then(function(storageInfo) { ... })

Parameters

None.

Returns

A reference to a {{jsxref('StorageInfo')}} object.
