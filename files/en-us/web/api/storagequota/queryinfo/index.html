The queryInfo() property of the {{domxref("StorageQuota")}} interface returns a {{domxref("StorageInfo")}} object containting the current data usage and available quota information for the application.

Syntax

StorageQuota.queryInfo().then(function(storageInfo) { ... })

Parameters

None.

Returns

A reference to a {{jsxref('StorageInfo')}} object.
