Warning: This is a non-standard deprecated feature which was implemented only in Firefox under a different name. Use {{DOMxRef("ScreenOrientation.onchange")}} instead.

An event handler for the {{domxref("Window/orientationchange_event", "orientationchange")}} events sent to the screen object.

Syntax

screen.onorientationchange = funcRef;

Where funcRef is a reference to a function.
