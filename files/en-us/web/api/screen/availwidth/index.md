{{APIRef("CSSOM View")}}

The **`Screen.availWidth`** property returns the amount of
horizontal space (in pixels) available to the window.

## Value

A number.

## Examples

```js
const screenAvailWidth = window.screen.availWidth;

console.log(screenAvailWidth);
```
