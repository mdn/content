The **`BluetoothRemoteGATTCharacteristic.stopNotifications()`** method
returns a {{jsxref("Promise")}} to the BluetoothRemoteGATTCharacteristic instance when
there is no longer an active notification on it.

## Syntax

```js-nolint
stopNotifications()
```

### Parameters

None.

### Return value

A {{jsxref("Promise")}}.
