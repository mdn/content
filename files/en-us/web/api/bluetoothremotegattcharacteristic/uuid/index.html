The BluetoothRemoteGATTCharacteristic.uuid read-only property returns {{domxref("DOMString")}} containing the UUID of the characteristic, for example '00002a37-0000-1000-8000-00805f9b34fb' for the Heart Rate Measurement characteristic.

Syntax

var uuid = BluetoothRemoteGATTCharacteristic.uuid

Returns

A {{domxref("DOMString")}}.
