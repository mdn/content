The reserve() property of the {{domxref("BudgetService")}} interface returns a {{domxref("Promise")}} that resolves to a boolean indicating whether the requested budget operation can be reserved.

Syntax

var aPromise = BudgetService.reserve(operation);
aPromise.then(function(boolean){ ... });

Parameters

operation
desc

Returns

A {{domxref("Promise")}} that resolves to a boolean.
