{{SVGRef}}{{deprecated_header}}

This element has been deprecated in SVG 1.1 Second Edition and may be removed in a future version of SVG. It provides no features not already available by using the {{SVGElement("animate")}} element. So, authors should use the <animate> element instead.

The <animateColor> SVG element specifies a color transformation over time.
