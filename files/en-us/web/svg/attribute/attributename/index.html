The attributeName attribute indicates the name of the CSS property or attribute of the target element that is going to be changed during an animation.

Four elements are using this attribute: {{SVGElement("animate")}}, {{SVGElement("animateColor")}}, {{SVGElement("animateTransform")}}, and {{SVGElement("set")}}

html, body, svg {
  height: 100%;
}
