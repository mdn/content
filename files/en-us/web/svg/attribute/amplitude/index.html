{{SVGRef}}

The amplitude attribute controls the amplitude of the gamma function of a component transfer element when its {{SVGAttr("type")}} attribute is gamma.

You can use this attribute with the following SVG elements:

  • {{SVGElement("feFuncA")}}
  • {{SVGElement("feFuncB")}}
  • {{SVGElement("feFuncG")}}
  • {{SVGElement("feFuncR")}}

Usage notes

Value <number>
Default value 1
Animatable Yes
