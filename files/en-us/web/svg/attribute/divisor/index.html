{{SVGRef}}

The divisor attribute specifies the value by which the resulting number of applying the {{SVGAttr("kernelMatrix")}} of a {{SVGElement("feConvolveMatrix")}} element to the input image color value is divided to yield the destination color value. A divisor that is the sum of all the matrix values tends to have an evening effect on the overall color intensity of the result.

Only one element is using this attribute: {{SVGElement("feConvolveMatrix")}}

Usage notes

Value {{cssxref("number")}}
Default value Sum of all values in {{SVGAttr("kernelMatrix")}} or 1 if sum is 0
Animatable Yes
<number>
This value defines the divisor. If the specified divisor is 0 then the default value will be used instead.
