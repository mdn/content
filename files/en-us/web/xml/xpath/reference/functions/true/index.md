The `true` function returns a boolean value of true.

## Syntax

```plain
true()
```

### Return value

Boolean `true`.

## Specifications

[XPath 1.0 4.3](https://www.w3.org/TR/1999/REC-xpath-19991116/#function-true)

## Gecko support

Supported.
