The `<xsl:call-template>` element invokes a named template.

## Syntax

```xml
<xsl:call-template name=NAME>
  <xsl:with-param> [optional]
</xsl:call-template>
```

### Required Attribute

- `name`
  - : Specifies the name of the template you wish to invoke.

### Optional Attributes

None.

### Type

Instruction, appears within a template.

## Specifications

XSLT, section 6.

## Gecko support

Supported.
