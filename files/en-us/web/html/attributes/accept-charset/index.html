{{HTMLSidebar}}

The accept-charset attribute is an allowed attribute for {{htmlelement("form")}}. When used on a <form> element it specifies the character encodings that are to be used for the form submission.

Usage

Syntax

<form accept-charset="character_set">

Attribute values

A space-separated list of one or more character encodings that are to be used for the form submission.

Common values:

  • UTF-8 - Character encoding for Unicode
  • ISO-8859-1 - Character encoding for the Latin alphabet

In theory, any character encoding can be used, but no browser understands all of them. The more widely a character encoding is used, the better the chance that a browser will understand it.

Examples

<form accept-charset="ISO-8859-1">
  <label for="fname">First name:</label>
  <input type="text" id="fname" name="fname" value="10">
  <label for="lname">Last name:</label>
  <input type="text" id="lname" name="lname" value="10">
</form>

See more examples of usage on the element pages for {{htmlelement("form")}.
