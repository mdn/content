{{HTMLSidebar}}

The defer attribute is a boolean type of attribute.

The defer attribute is an allowed attribute for {{htmlelement("script")}}. When used on a <script> element, the script will be executed automatically without clicking on any button, after loading of page.

Note: Use the defer attribute only when script is external file and linked to web page by using src attribute.

Note: In several ways, we can execute an external script:

  • If async is present, then script will be executed automatically with loading of the web page(Execution of script and loading of web page occur at the same time).
  • If async is not present and defer is present, then script is executed after completion of loading (First web page loads, then script executes).
  • If neither async nor defer is present, then web page will be loaded after execution of script (First script executes, then page loads). Until script execution completes, parsing will be blocked by browser.

Usage

The defer can be used on <script> element.

Syntax

<script src="sample_defer.js" defer></script>

Examples

See examples of usage on the element pages for {{htmlelement("script")}}.
