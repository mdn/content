{{HTMLSidebar}}

The action attribute is an allowed attribute for {{htmlelement("form")}}. When used on a <form> element it specifies where to send the form-data when a form is submitted.

Usage

Syntax

<form action="URL" method="get">

On submit, send the form-data to URL specified in action, to process the input.

Examples

<form action="/action_page.php" method="get">
  <label for="fname">First name:</label>
  <input type="text" id="fname" name="fname" value="10">
  <label for="lname">Last name:</label>
  <input type="text" id="lname" name="lname" value="10">
</form>

See more examples of usage on the element pages for {{htmlelement("form")}}.
