{{APIRef("DOM")}}

{{Non-standard_header()}}

The x-ms-aria-flowfrom property specifies the id of the previous element in an alternative reading order, allowing assistive technology to override the general default of reading in document source order. The property serves to define element relationships utilizing ARIA relationship attributes and the aria-flowto property.

This proprietary property is specific to Internet Explorer and Microsoft Edge.

Syntax

x-ms-aria-flowfrom="elementID";

Value

The x-ms-aria-flowfrom property value uses an ID selector to define which previous element the reading order will flow from.

Example

<div tabindex="0" class="foo" id="element2" role="option" aria-posinset="1" aria-setsize="15" aria-flowto="element8" x-ms-aria-flowfrom="element5">
