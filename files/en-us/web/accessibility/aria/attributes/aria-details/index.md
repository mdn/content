The  global `aria-details` attribute  identifies the element (or elements) that provide additional information related to the object

## Description

identifies the element (or elements) that provide additional information related to the object

## Values

- 
  - : 

## ARIAMixin API 

- {{domxref("Element.aria-details")}}
  - : The  [`aria-details`](/en-US/docs/Web/API/Element/aria-details) property, part of the {{domxref("ARIAMixin")}} interface, reflects the value of the `aria-details` attribute, which identifies the element (or elements) that provide additional information related to the object.

## Associated roles

Used in **ALL** roles.
