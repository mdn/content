{{CSSRef}}

The resolved value of a CSS property is the value returned by {{domxref("Window.getComputedStyle", "getComputedStyle()")}}.

For most properties, it is the computed value, but for a few legacy properties (including {{cssxref("width")}} and {{cssxref("height")}}), it is instead the used value. See the specification link below for more per-property details.
