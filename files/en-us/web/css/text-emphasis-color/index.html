{{CSSRef}}

The text-emphasis-color CSS property sets the color of emphasis marks. This value can also be set using the {{cssxref("text-emphasis")}} shorthand.

/* Initial value */
text-emphasis-color: currentcolor;

/* <color> */
text-emphasis-color: #555;
text-emphasis-color: blue;
text-emphasis-color: rgba(90, 200, 160, 0.8);
text-emphasis-color: transparent;

/* Global values */
text-emphasis-color: inherit;
text-emphasis-color: initial;
text-emphasis-color: unset;

Syntax

Values

<color>
Defines the color of the emphasis marks. If no color is present, it defaults to currentcolor.
