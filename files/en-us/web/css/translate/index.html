{{CSSRef}}

The translate CSS property allows you to specify translation transforms individually and independently of the {{CSSxRef("transform")}} property. This maps better to typical user interface usage, and saves having to remember the exact order of transform functions to specify in the transform value.

Syntax

/* Keyword values */
translate: none;

/* Single values */
translate: 100px;
translate: 50%;

/* Two values */
translate: 100px 200px;
translate: 50% 105px;

/* Three values */
translate: 50% 105px 5rem;

/* Global values */
translate: inherit;
translate: initial;
translate: revert;
translate: unset;

Values

Single {{cssxref("<length-percentage>")}} value
A {{cssxref("<length>")}} or {{cssxref("<percentage>")}} that specifies a 2D translation, with the same translation along both the X and Y axes. Equivalent to a translate() (2D translation) function with a single value specified.
Two {{cssxref("<length-percentage>")}} values
Two {{cssxref("<length>")}} or {{cssxref("<percentage>")}} that specify the X and Y axis translation values (respectively) of a 2D translation. Equivalent to a translate() (2D translation) function with two values specified.
Three values
Two {{cssxref("<length-percentage>")}} and single {{cssxref("<length>")}} values that specify the X, Y, and Z axis translation values (respectively) of a 3D translation. Equivalent to a translate3d() (3D translation) function.
none
Specifies that no translation should be applied.
