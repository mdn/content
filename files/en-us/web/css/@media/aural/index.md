<p>The <code>aural</code> <a href="/en-US/docs/Web/CSS">CSS</a> <a href="/en-US/docs/Web/CSS/@media#media_types">media type</a> is used for devices that have speech output capabilities.</p>

<div class="note">
<p><strong>Note:</strong> This media type has been deprecated in favor of <code><a href="/en-US/docs/Web/CSS/@media#speech">speech</a></code>.</p>
</div>

<h2 id="Syntax">Syntax</h2>

<p>The <code>aural</code> CSS media type—which has been deprecated in favor of the <code><a href="/en-US/docs/Web/CSS/@media#speech">speech</a></code> media type—was used to specify a block of CSS that applied only when the content is being presented using a speech synthesis device.</p>

<pre class="brush: css">@media aural {
  /* speech-specific styles here */
}
</pre>

<p>Updating existing CSS to use the <code>speech</code> media type should be as simple as replacing <code>aural</code> with <code>speech</code>.</p>

<h2 id="Examples">Examples</h2>

<h3 id="Basic_example">Basic example</h3>

<pre class="brush: css">@media aural {
  body { voice-family: Paul }
}</pre>
