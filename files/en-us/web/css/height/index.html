The height CSS property specifies the height of an element. By default, the property defines the height of the content area. If {{cssxref("box-sizing")}} is set to border-box, however, it instead determines the height of the border area.

{{EmbedInteractiveExample("pages/css/height.html")}}

The {{cssxref("min-height")}} and {{cssxref("max-height")}} properties override height.

Syntax

/* <length> values */
height: 120px;
height: 10em;

/* <percentage> value */
height: 75%;

/* Keyword values */
height: max-content;
height: min-content;
height: fit-content(20em);
height: auto;

/* Global values */
height: inherit;
height: initial;
height: revert;
height: unset;

Values

{{cssxref("<length>")}}
Defines the height as an absolute value.
{{cssxref("<percentage>")}}
Defines the height as a percentage of the containing block's height.
auto
The browser will calculate and select a height for the specified element.
max-content
The intrinsic preferred height.
min-content
The intrinsic minimum height.
fit-content({{cssxref("<length-percentage>")}})
Uses the fit-content formula with the available space replaced by the specified argument, i.e. min(max-content, max(min-content, <length-percentage>)).

Accessibility concerns

Ensure that elements set with a height aren't truncated and/or don't obscure other content when the page is zoomed to increase text size.

Examples

Setting height using pixels and percentages

HTML

<div id="taller">I'm 50 pixels tall.</div>
<div id="shorter">I'm 25 pixels tall.</div>
<div id="parent">
  <div id="child">
    I'm half the height of my parent.
  </div>
</div>

CSS

div {
  width: 250px;
  margin-bottom: 5px;
  border: 2px solid blue;
}

#taller {
  height: 50px;
}

#shorter {
  height: 25px;
}

#parent {
  height: 100px;
}

#child {
  height: 50%;
  width: 75%;
}
