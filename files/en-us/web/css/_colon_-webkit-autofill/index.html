The :-webkit-autofill CSS pseudo-class matches when an {{HTMLElement("input")}} element has its value autofilled by the browser.

Note: The user agent style sheets of many browsers use !important in their :-webkit-autofill style declarations, making them non-overrideable by webpages without resorting to JavaScript hacks.

Syntax

:-webkit-autofill

Specifications

Not part of any standard.

Browser compatibility

{{Compat("css.selectors.-webkit-autofill")}}
