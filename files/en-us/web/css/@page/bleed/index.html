The bleed CSS at-rule descriptor, used with the {{cssxref("@page")}} at-rule, specifies the extent of the page bleed area outside the page box. This property only has effect if crop marks are enabled using the {{cssxref("@page/marks", "marks")}} property.

Syntax

/* Keyword values */
bleed: auto;

/* <length> values */
bleed: 8pt;
bleed: 1cm;

Values

auto
Computes to 6pt if the value of {{cssxref("@page/marks", "marks")}} is crop. Otherwise it computes to zero.
{{cssxref("<length>")}}
Specifies by how far outward, in each direction, the bleed area extends past the page box. Values may be negative, but there may be implementation-specific limits.
