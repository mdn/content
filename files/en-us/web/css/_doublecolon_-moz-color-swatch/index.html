The ::-moz-color-swatch CSS pseudo-element is a Mozilla extension that represents the color selected in an {{HTMLElement("input")}} of type="color".

Note: Using ::-moz-color-swatch with anything but an <input type="color"> doesn't match anything and has no effect.

Syntax

{{csssyntax}}

Examples

HTML

<input type="color" value="#de2020" />

CSS

input[type=color]::-moz-color-swatch {
  border-radius: 10px;
  border-style: none;
}
