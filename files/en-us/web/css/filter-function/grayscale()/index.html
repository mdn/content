The grayscale() CSS function converts the input image to grayscale. Its result is a {{cssxref("<filter-function>")}}.

{{EmbedInteractiveExample("pages/css/function-grayscale.html")}}

Syntax

grayscale(amount)

Parameters

amount
The amount of the conversion, specified as a {{cssxref("<number>")}} or a {{cssxref("<percentage>")}}. A value of 100% is completely grayscale, while a value of 0% leaves the input unchanged. Values between 0% and 100% are linear multipliers on the effect. Default value when omitted is 1. The initial value for interpolation is 0.

Examples

Examples of correct values for grayscale()

grayscale(0)     /* No effect */
grayscale(.7)    /* 70% grayscale */
grayscale(100%)  /* Completely grayscale */
