The brightness() CSS function applies a linear multiplier to the input image, making it appear brighter or darker. Its result is a {{cssxref("<filter-function>")}}.

{{EmbedInteractiveExample("pages/css/function-brightness.html")}}

Syntax

brightness(amount)

Parameters

amount
The brightness of the result, specified as a {{cssxref("<number>")}} or a {{cssxref("<percentage>")}}. A value under 100% darkens the image, while a value over 100% brightens it. A value of 0% will create an image that is completely black, while a value of 100% leaves the input unchanged. The lacuna value for interpolation is 1.

Examples

Setting brightness using numbers and percentages

brightness(0%)   /* Completely black */
brightness(0.4)  /* 40% brightness */
brightness(1)    /* No effect */
brightness(200%) /* Double brightness */
