The ::-webkit-slider-thumb CSS pseudo-element represents the "thumb" that the user can move within the "groove" of an {{HTMLElement("input")}} of type="range" to alter its numerical value.

Syntax

::-webkit-slider-thumb

Specifications

Not part of any standard.
