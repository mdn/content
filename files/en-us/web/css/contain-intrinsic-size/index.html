{{CSSRef}}

The contain-intrinsic-size CSS property controls the natural size of an element specified by content-visibility.

Syntax

/* <length> values */
contain-intrinsic-size: 1000px;
contain-intrinsic-size: 10rem;

/* <percentage> value */
contain-intrinsic-size: 10%;
