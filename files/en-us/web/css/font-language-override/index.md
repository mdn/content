<p>The <strong><code>font-language-override</code></strong> CSS property controls the use of language-specific glyphs in a typeface.</p>

<pre class="brush:css no-line-numbers">/* Keyword value */
font-language-override: normal;

/* &lt;string&gt; values */
font-language-override: "ENG";  /* Use English glyphs */
font-language-override: "TRK";  /* Use Turkish glyphs */

/* Global values */
font-language-override: initial;
font-language-override: inherit;
font-language-override: revert;
font-language-override: unset;
</pre>

<p>By default, HTML's <code>lang</code> attribute tells browsers to display glyphs designed specifically for that language. For example, a lot of fonts have a special character for the digraph <code>fi</code> that merge the dot on the "i" with the "f." However, if the language is set to Turkish the typeface will likely know not to use the merged glyph; Turkish has two versions of the "i," one with a dot (<code>i</code>) and one without (<code>ı</code>), and using the ligature would incorrectly transform a dotted "i" into a dotless "i."</p>

<p>The <code>font-language-override</code> property lets you override the typeface behavior for a specific language. This is useful, for example, when the typeface you're using lacks proper support for the language. For instance, if a typeface doesn't have proper rules for the Azeri language, you can force the font to use Turkish glyphs, which follow similar rules.</p>

<h2 id="Syntax">Syntax</h2>

<p>The <code>font-language-override</code> property is specified as the keyword <code>normal</code> or a <code>&lt;string&gt;</code>.</p>

<h3 id="Values">Values</h3>

<dl>
 <dt><code>normal</code></dt>
 <dd>Tells the browser to use font glyphs that are appropriate for the language specified by the <code>lang</code> attribute. This is the default value.</dd>
 <dt>{{cssxref("string")}}</dt>
 <dd>Tells the browser to use font glyphs that are appropriate for the language specified by the string. The string must match a language tag found in the <a href="https://www.microsoft.com/typography/otspec/languagetags.htm">OpenType language system</a>. For example, "ENG" is English, and "KOR" is Korean.</dd>
</dl>

<h2 id="Examples">Examples</h2>

<h3 id="Using_Danish_glyphs">Using Danish glyphs</h3>

<h4 id="HTML">HTML</h4>

<pre class="brush: html">&lt;p class=&quot;para1&quot;&gt;Default language setting.&lt;/p&gt;
&lt;p class=&quot;para2&quot;&gt;This is a string with the &lt;code&gt;font-language-override&lt;/code&gt; set to Danish.&lt;/p&gt;
</pre>

<h4 id="CSS">CSS</h4>

<pre class="brush: css">p.para1 {
  font-language-override: normal;
}

p.para2 {
  font-language-override: "DAN";
}</pre>
