{{CSSRef}}

The **`text-wrap-mode`** CSS property controls whether the text inside an element is wrapped. The different values provide alternate ways of wrapping the content of a block element.

{{EmbedInteractiveExample("pages/css/text-wrap-mode.html")}}

## Syntax

```css
/* Keyword values */
text-wrap-style: wrap;
text-wrap-style: nowrap;

/* Global values */
text-wrap-style: inherit;
text-wrap-style: initial;
text-wrap-style: revert;
text-wrap-style: revert-layer;
text-wrap-style: unset;
```
