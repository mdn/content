{{CSSRef}}

CSS Color is a CSS module that deals with colors, color types, color blending, opacity, and how you can apply these colors and effects to HTML content. Not all CSS properties that take a {{cssxref("<color>")}} as a value are part of this module, but they do depend upon it.
