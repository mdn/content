<p><strong>CSS Ruby Layout</strong> is a module of <a href="/en-US/docs/Web/CSS">CSS</a> that provides the rendering model and formatting controls related to the display of ruby annotation. Ruby annotation is a form of interlinear annotation, consisting of short runs of text alongside the base text. They are typically used in East Asian documents to indicate pronunciation or to provide a short annotation.</p>
