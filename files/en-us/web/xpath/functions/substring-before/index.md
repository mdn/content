<p><br>
 The <code>substring-before</code> function returns a string that is the part of a given string before a given substring.</p>

<h3 id="Syntax">Syntax</h3>

<pre class="eval">substring-before(<em>haystack</em> ,<em>needle</em> )
</pre>

<h3 id="Arguments">Arguments</h3>

<dl>
 <dt><em><code>haystack</code></em></dt>
 <dd>The string to be evaluated. Part of this string will be returned.</dd>
 <dt><em><code>needle</code></em></dt>
 <dd>The substring to search for. Everything before the first occurrence of<em><code>needle</code></em> in<em><code>haystack</code></em> will be returned.</dd>
</dl>

<h3 id="Returns">Returns</h3>

<p>A string.</p>
