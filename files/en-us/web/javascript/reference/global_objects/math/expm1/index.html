{{JSRef}}

The Math.expm1() function returns e^x - 1, where x is the argument, and {{jsxref("Math.E", "e", "", 1)}} the base of the natural logarithms.

{{EmbedInteractiveExample("pages/js/math-expm1.html")}}

Syntax

Math.expm1(x)

Parameters

x
A number.

Return value

A number representing e^x - 1, where e is {{jsxref("Math.E", "Euler's number", "", 1)}} and x is the argument.

Description

Because expm1() is a static method of Math, you always use it as Math.expm1(), rather than as a method of a Math object you created (Math is not a constructor).

Examples

Using Math.expm1()

Math.expm1(-1); // -0.6321205588285577
Math.expm1(0);  // 0
Math.expm1(1);  // 1.718281828459045
