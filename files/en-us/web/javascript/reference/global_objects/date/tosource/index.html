The toSource() method returns a string representing the source code of the object.

Syntax

dateObj.toSource()
Date.toSource()

Return value

A string representing the source code of the given {{jsxref("Global_Objects/Date", "Date")}} object.

Examples

Native function

For the built-in {{jsxref("Date")}} object, toSource() returns the following string indicating that the source code is not available:

function Date() {
    [native code]
}

Specifications

Not part of any standard.
