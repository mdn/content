{{JSRef}}

The **`year`** accessor property of {{jsxref("Temporal.PlainDate")}} instances returns a non-zero integer representing the number of years of this date relative to the start of a calendar-specific epoch year. Usually year 1 is either the first year of the latest era or the ISO 8601 year `0001`. If the epoch era starts in the middle of the year, the year will be the same value before and after the start date of the era.
