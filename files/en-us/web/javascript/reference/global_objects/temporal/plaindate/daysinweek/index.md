{{JSRef}}

The **`daysInWeek`** accessor property of {{jsxref("Temporal.PlainDate")}} instances returns a positive integer representing the number of days in the week of this date. For the ISO 8601 calendar, this is always 7, but in other calendar systems it may differ from week to week.
