The fixed() method creates a {{HTMLElement("tt")}} HTML element that causes a string to be displayed in fixed-pitch font.

Syntax

fixed()

Return value

A string representing a {{HTMLElement("tt")}} HTML element.

Description

The fixed() method embeds a string in a <tt> element: "<tt>str</tt>".

Examples

Using fixed()

The following example uses the fixed method to change the formatting of a string:

var worldString = 'Hello, world';
console.log(worldString.fixed()); // "<tt>Hello, world</tt>"
