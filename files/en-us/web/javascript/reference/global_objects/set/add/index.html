The add() method appends a new element with a specifiedÂ value to the end of a Set object.

{{EmbedInteractiveExample("pages/js/set-prototype-add.html")}}

Syntax

mySet.add(value);

Parameters

value
The value of the element to add to the Set object.

Return value

The Set object.

Examples

Using the add method

var mySet = new Set();

mySet.add(1);
mySet.add(5).add('some text'); // chainable

console.log(mySet);
// Set [1, 5, "some text"]
