The clear() method removes all elements from a Set object.

{{EmbedInteractiveExample("pages/js/set-prototype-clear.html")}}

Syntax

clear()

Return value

{{jsxref("undefined")}}.

Examples

Using the clear() method

var mySet = new Set();
mySet.add(1);
mySet.add('foo');

mySet.size;       // 2
mySet.has('foo'); // true

mySet.clear();

mySet.size;       // 0
mySet.has('bar')  // false
