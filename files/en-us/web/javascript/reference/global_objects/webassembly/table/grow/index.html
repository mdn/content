{{JSRef}}

The grow() prototype method of theÂ {{jsxref("WebAssembly.Table")}} object increases the size of the Table instance by a specified number of elements.

Syntax

table.grow(number);

Parameters

number
The number of elements you want to grow the table by.

Return value

The previous length of the table.

Exceptions

If the grow() operation fails for whatever reason, a {{jsxref("RangeError")}} is thrown.

Examples

Using grow

The following example creates a new WebAssembly Table instance with an initial size of 2 and a maximum size of 10.

var table = new WebAssembly.Table({ element: "anyfunc", initial: 2, maximum: 10 });

You can then grow the table by 1 with the following:

console.log(table.length);   // "2"
console.log(table.grow(1));  // "2"
console.log(table.length);   // "3"
