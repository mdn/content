The toSource() method returns a string representing the source code of the object.

Syntax

toSource()

Return value

A string representing the source code of the object.

Examples

Native function

For the built-in {{jsxref("Boolean")}} object, toSource returns the following string indicating that the source code is not available:

function Boolean() {
    [native code]
}

Specifications

Not part of any standard.
