{{JSRef}}

The valueOf() method returns the wrapped primitive value of a {{jsxref("BigInt")}} object.

{{EmbedInteractiveExample("pages/js/bigint-valueof.html","shorter")}}

Syntax

bigIntObj.valueOf()

Return value

A BigInt representing the primitive value of the specified {{jsxref("BigInt")}} object.

Examples

Using valueOf

typeof Object(1n); // object
typeof Object(1n).valueOf(); // bigint
