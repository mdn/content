The Number.MAX_VALUE property represents the maximum numeric value representable in JavaScript.

Description

The MAX_VALUE property has a value of approximately 1.79E+308, or 21024. Values larger than MAX_VALUE are represented as {{jsxref("Infinity")}}.

Because MAX_VALUE is a static property of {{jsxref("Number")}}, you always use it as Number.MAX_VALUE, rather than as a property of a {{jsxref("Number")}} object you created.

Examples

Using MAX_VALUE

The following code multiplies two numeric values. If the result is less than or equal to MAX_VALUE, the func1 function is called; otherwise, the func2 function is called.

if (num1 * num2 <= Number.MAX_VALUE) {
  func1();
} else {
  func2();
}
