{{JSRef}}


This method has the very same functionality as the global {{jsxref("parseInt()")}} function:

```js
Number.parseInt === parseInt; // true
```

Its purpose is modularization of globals. Please see
{{jsxref("parseInt()")}} for detail and examples.
