The size accessor property returns the number of elements in a {{jsxref("Map")}} object.

{{EmbedInteractiveExample("pages/js/map-prototype-size.html")}}

Description

The value of size is an integer representing how many entries the Map object has. A set accessor function for size is undefined; you can not change this property.

Examples

Using size

var myMap = new Map();
myMap.set('a', 'alpha');
myMap.set('b', 'beta');
myMap.set('g', 'gamma');

myMap.size // 3
