<p>The <strong>Exchangeable Image File Format (EXIF)</strong> is a standard that specifies how metadata about a multimedia file can be embedded within the file. For example, an image might contain EXIF data describing the: pixel width, height, and density, shutter speed, aperture, ISO settings, capture date, etc.</p>
