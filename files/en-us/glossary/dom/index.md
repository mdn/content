The **DOM** (Document Object Model) is a programming interface for {{Glossary("HTML")}} and {{Glossary("XML")}} documents. It represents the document as a tree of {{Glossary("node","nodes")}} that can be manipulated with scripts.

Using the DOM, developers can dynamically change document content, structure, and style. The DOM is {{Glossary("W3C")}}-standardized and language-independent.
