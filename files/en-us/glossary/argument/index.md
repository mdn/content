An **argument** is a value ({{Glossary("primitive")}} or {{Glossary("object")}}) passed as input to a {{Glossary("function")}}. In JavaScript, functions can access arguments through the {{jsxref("Functions/arguments", "arguments")}} object or through named {{Glossary("parameter")}} names.

Arguments are distinct from parameters: parameters are the names listed in the function definition, while arguments are the actual values passed when the function is called.
