<p><strong>Time to First Byte</strong> (TTFB) refers to the time between the browser requesting a page and when it receives the first byte of information from the server.Â  This time includes <a href="/en-US/docs/Glossary/DNS">DNS</a> lookup and establishing the connection using a <a href="/en-US/docs/Glossary/TCP">TCP</a> handshake and <a href="/en-US/docs/Glossary/SSL">SSL</a> handshake if the request is made over <a href="/en-US/docs/Glossary/https">https</a>.</p>

<p>TTFB is the time it takes between the start of the request and the start of the response, in milliseconds:</p>
