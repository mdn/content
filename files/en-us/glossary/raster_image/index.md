<p>A <strong><dfn>raster image</dfn></strong> is an image file defined as a grid of pixels. Theyâ€™re also referred to as <em>bitmaps</em>. Common raster image formats on the Web are <a href="/en-US/docs/Glossary/jpeg">JPEG</a>, <a href="/en-US/docs/Glossary/PNG">PNG</a>, <a href="/en-US/docs/Glossary/gif">GIF</a>, and <a href="https://en.wikipedia.org/wiki/ICO_(file_format)">ICO</a>.</p>

<p>Raster image files usually contain one set of dimensions, but the ICO and CUR formats, used for favicons and <a href="/en-US/docs/Web/CSS/cursor">CSS cursor images</a>, can contain multiple sizes.</p>
