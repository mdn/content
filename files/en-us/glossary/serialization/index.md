<p>The process whereby an object or data structure is translated into a format suitable for transferral over a network, or storage (e.g. in an array buffer or file format).</p>

<p>In {{Glossary("JavaScript")}}, for example, you can serialize an object to a {{Glossary("JSON")}} {{Glossary("string")}} by calling the {{Glossary("function")}} {{jsxref("JSON.stringify()")}}.</p>

<p>{{Glossary("CSS")}} values are serialized by calling the function {{domxref("CSSStyleDeclaration.getPropertyValue()")}}.</p>
