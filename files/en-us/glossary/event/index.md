An **event** is a signal that something has occurred, typically generated by the user (clicks, keypresses), the browser (page load, resizing), or JavaScript code.

In web development, event handling uses the {{Glossary("DOM")}} event model, where listeners are registered to respond to specific events. The `event` object contains details about the occurrence.
