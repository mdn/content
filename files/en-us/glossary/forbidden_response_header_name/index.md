<p>A <dfn>forbidden response header name</dfn> is an <a href="/en-US/docs/Web/HTTP/Headers">HTTP header</a> name (either `<code>Set-Cookie</code>` or `<code>Set-Cookie2</code>`) that cannot be modified programmatically.</p>
