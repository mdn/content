--- title: Gonk slug: Glossary/Gonk tags: - B2G - Boot2Gecko - Firefox OS - Glossary - Gonk - Infrastructure - Intro ---

Gonk is basically an extremely simple Linux distribution and consists of a Linux kernel (based on the Android Open Source Project (AOSP)) and userspace hardware abstraction layer (HAL). See {{interwiki("wikipedia", "Firefox OS")}} for more details.
