A **Bézier curve** is a mathematically described curve used in computer graphics and animation. In CSS, cubic Bézier curves are used in the {{cssxref("animation-timing-function")}} and {{cssxref("transition-timing-function")}} properties to control the pacing of animations and transitions.

The curve is defined by control points: a cubic Bézier curve uses four points (P0, P1, P2, P3), while a quadratic Bézier curve uses three.
