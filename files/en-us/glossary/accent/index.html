--- title: accent slug: Glossary/accent tags: - Glossary - Input - accent ---

An accent is a typically bright color that contrasts markedly with the more utilitarian background and foreground colors within a color scheme. These are present on many platforms (though not all).

Learn more

CSS related to the accent

You can set the color of the accent for a given element by setting the element's CSS {{cssxref("accent-color")}} property to the appropriate {{cssxref("<color>")}} value.

HTML elements that may have an accent

  • <input type="checkbox">
  • <input type="radio">
  • <input type="range">
