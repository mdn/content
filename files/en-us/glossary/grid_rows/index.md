<p>A <strong>grid row</strong> is a horizontal track in a <a href="/en-US/docs/Web/CSS/CSS_Grid_Layout">CSS Grid Layout</a>, that is the space between two horizontal grid lines. It is defined by the {{cssxref("grid-template-rows")}} property or in the shorthand {{cssxref("grid")}} or {{cssxref("grid-template")}} properties.</p>

<p>In addition, rows may be created in the <em>implicit grid</em> when items are placed outside of rows created in the <em>explicit grid</em>. These rows will be auto sized by default, or can have a size specified with the {{cssxref("grid-auto-rows")}} property.</p>

<p>When working with alignment in <a href="/en-US/docs/Web/CSS/CSS_Grid_Layout">CSS Grid Layout</a>, the axis along which rows run is known as the <em>inline, or row, axis</em>.</p>
