**CSS** (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in {{Glossary("HTML")}} or {{Glossary("XML")}}.

CSS controls layout, colors, fonts, animations, and responsive design. The latest version is CSS3, which is modularized into specifications.
