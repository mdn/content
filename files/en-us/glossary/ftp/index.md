**FTP** (File Transfer Protocol) is a standard network protocol used to transfer files between a client and server over a {{Glossary("TCP")}}-based network.

FTP has been largely replaced by more secure alternatives like {{Glossary("SFTP")}} and FTPS, which add encryption on top of FTP.
