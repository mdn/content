A **function** is a reusable block of code that performs a specific task. Functions can accept {{Glossary("parameter","parameters")}} and return a value.

In {{Glossary("JavaScript")}}, functions are first-class objects, meaning they can be assigned to variables, passed as arguments, and returned from other functions.
