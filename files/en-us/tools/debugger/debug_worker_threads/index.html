Debugging a worker process is the same as debugging the main thread! The debugger shows the source files for all active worker threads in the sources list (web workers, shared workers, service workers, etc.). You can open the JavaScript file for a worker process and set breakpoints and logpoints just as you can with JavaScript code running on the main thread.
