This article lists different resources for developers to check web documents.

  • The W3C HTML Validator validates HTML documents.
  • The W3C CSS Validator validates CSS stylesheets.
  • The W3C Link Checker checks for broken links in HTML documents.
  • The HTML Tidy tool finds errors in HTML documents, and can automatically fix some errors.
