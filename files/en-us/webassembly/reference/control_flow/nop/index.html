nop stands for no-operation. It literally does nothing.

Syntax

nop

Full working example

Wasm file

(module
  (func (export "do_nothing")
    nop
  )
)

JavaScript file

WebAssembly.instantiateStreaming(
  fetch("../out/main.wasm")
).then(result => {
  result.instance.exports.do_nothing();
});
Opcodes
nop 0x01
