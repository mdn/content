**`call`** calls a function. `call_indirect` calls a function in a table.

{{EmbedInteractiveExample("pages/wat/call.html")}}

## Syntax

```wasm
call $greet
```

| Instruction     | Binary opcode |
| --------------- | ------------- |
| `call`          | `0x10`        |
| `call_indirect` | `0x11`        |
