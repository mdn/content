This is the part you’ve been waiting for: creating the peer connection and call logic. First, create a script.js file. This is where all your logic will live.

Next, we need to create a peer object with an ID. The ID will be used to connect two peers together and if you don’t create one, one will be assigned to the peer.

const peer = new Peer(''+Math.floor(Math.random()*2**18).toString(36).padStart(4,0), {
    host: location.hostname,
    debug: 1,
    path: '/myapp'
});

Then attach the peer to the window so that it’s accessible.

window.peer = peer;

In another tab in your terminal, start the peer server.

peerjs --port 443 --key peerjs --path /myapp
